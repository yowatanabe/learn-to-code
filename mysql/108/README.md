# No.108

セールが開始されると、対象商品の価格が一括で更新されるとともに、価格の変更履歴が保存されます。大量の商品が更新される際のトランザクションを最適化するために、バルク処理とインデックスを活用します。

| 3         | Headphones  | 150   |
