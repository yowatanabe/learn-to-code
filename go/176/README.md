### 問題: 配列の要素の積を計算する

整数の配列 `nums` が与えられます。
この配列に含まれるすべての要素の **積**（掛け算の結果）を返す関数 `productOfArray` を実装してください。
配列には必ず1つ以上の要素が含まれているものとします。

### 実行例

#### 例 1

```
入力: nums = [1, 2, 3, 4]
出力: 24
説明: 1 × 2 × 3 × 4 = 24
```

#### 例 2

```
入力: nums = [2, 5, 6]
出力: 60
```

#### 例 3

```
入力: nums = [10]
出力: 10
```

### 実行結果

```sh
24
60
10
```
