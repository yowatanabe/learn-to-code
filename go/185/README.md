# No.185

### 問題: 配列の中に指定された数がいくつ含まれているか数える

整数の配列 `nums` と整数 `target` が与えられます。
`nums` の中に `target` が **何回出現するか** を数えて返す関数 `countOccurrences` を実装してください。

### 実行例

#### 例 1:

```
入力: nums = [1, 2, 3, 2, 4, 2], target = 2
出力: 3
```

#### 例 2:

```
入力: nums = [5, 5, 5, 5, 5], target = 5
出力: 5
```

#### 例 3:

```
入力: nums = [1, 3, 5], target = 2
出力: 0
```

### 実行結果

```bash
3
5
0
```
