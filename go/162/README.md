# No.162

### 問題: 整数が3の倍数かつ5の倍数かを判定する

整数 `n` が与えられます。
`n` が **3の倍数かつ5の倍数** のとき `"FizzBuzz"` を返し、
**3の倍数** のときは `"Fizz"`、**5の倍数** のときは `"Buzz"`、
いずれでもない場合は整数を文字列に変換して返してください。

### 🔹 実行例

#### 例 1:

```
入力: n = 15
出力: "FizzBuzz"
```

#### 例 2:

```
入力: n = 9
出力: "Fizz"
```

#### 例 3:

```
入力: n = 10
出力: "Buzz"
```

#### 例 4:

```
入力: n = 7
出力: "7"
```

### 実行結果

```sh
FizzBuzz
Fizz
Buzz
7
```
