# No.158

### 問題: 配列の中に特定の要素が存在するか判定する

整数の配列 `nums` と整数 `target` が与えられます。
`nums` の中に `target` が**少なくとも1回でも存在する場合は `true`** を、**存在しない場合は `false`** を返す関数 `containsTarget` を実装してください。

### 実行例

#### 例 1:

```
入力: nums = [1, 2, 3, 4, 5], target = 3
出力: true
```

#### 例 2:

```
入力: nums = [10, 20, 30], target = 25
出力: false
```

#### 例 3:

```
入力: nums = [], target = 1
出力: false
```

### 実行結果

```sh
true
false
false
```
