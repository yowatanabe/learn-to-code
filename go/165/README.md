# No.165

### 問題: 2の累乗かどうか判定する

整数 `n` が与えられます。
`n` が **2 の累乗**（1, 2, 4, 8, 16, …）であれば `true` を、そうでなければ `false` を返す関数 `isPowerOfTwo` を実装してください。
なお、`n` が 1 のときは `2^0` とみなして `true` を返します。

### 実行例

#### 例 1

```
入力: n = 1
出力: true
説明: 1 = 2^0
```

#### 例 2

```
入力: n = 16
出力: true
説明: 16 = 2^4
```

#### 例 3

```
入力: n = 3
出力: false
```

#### 例 4

```
入力: n = 0
出力: false
```

### 実行結果

```sh
true
true
false
false
```
