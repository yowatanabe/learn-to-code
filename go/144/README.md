# No.144

### 問題: 文字列内の小文字をカウントする

英字からなる文字列 `s` が与えられます。
その中に含まれる **小文字のアルファベット** の数を返す関数 `countLowercase` を実装してください。
大文字はカウントに含めないものとします。

### 🔹 実行例

#### 例 1:

```
入力: s = "HelloWorld"
出力: 8
説明: 小文字は e, l, l, o, o, r, l, d の8文字
```

#### 例 2:

```
入力: s = "ABCdefGHI"
出力: 3
```

#### 例 3:

```
入力: s = "GO"
出力: 0
```

### 実行結果

```sh
8
3
0
```
