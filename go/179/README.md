# No.179

### 問題: 整数の桁の合計を求める

整数 `n` が与えられます。
その整数に含まれる **各桁の数字の合計** を返す関数 `sumOfDigits` を実装してください。
`n` は正の整数とします。

### 実行例

#### 例 1:

```
入力: n = 123
出力: 6
説明: 1 + 2 + 3 = 6
```

#### 例 2:

```
入力: n = 409
出力: 13
説明: 4 + 0 + 9 = 13
```

#### 例 3:

```
入力: n = 5
出力: 5
```

### 実行結果

```sh
6
13
5
```
