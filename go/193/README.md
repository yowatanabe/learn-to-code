# No.193

### 問題: 配列の要素をすべて2倍にする

整数の配列 `nums` が与えられます。
各要素をすべて **2倍にした新しい配列** を返す関数 `doubleElements` を実装してください。

### 🔹 実行例

#### 例 1:

```
入力: nums = [1, 2, 3]
出力: [2, 4, 6]
```

#### 例 2:

```
入力: nums = [-1, 0, 5]
出力: [-2, 0, 10]
```

#### 例 3:

```
入力: nums = []
出力: []
```

### 実行結果

```sh
[2 4 6]
[-2 0 10]
[]
```
