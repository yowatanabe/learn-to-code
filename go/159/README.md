# No.159

## 問題文

### 配列内のゼロを削除せよ

整数のスライス `nums` が与えられます。
このスライスから **値が `0` の要素をすべて削除し、残りの要素のみを保持した新しいスライスを返す関数**を実装してください。
元の順序は保つものとします。

### 実行例

```go
入力: nums = []int{1, 0, 2, 0, 3, 4}
出力: []int{1, 2, 3, 4}

入力: nums = []int{0, 0, 0}
出力: []int{}

入力: nums = []int{5, 6, 0, 7}
出力: []int{5, 6, 7}
```

### 制約

- `1 <= len(nums) <= 1000`
- `-10^4 <= nums[i] <= 10^4`

### 実行結果

```sh
[1 2 3 4]
[]
[5 6 7]
```
