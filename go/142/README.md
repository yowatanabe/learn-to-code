# No.142

### **問題: 2つの配列の交差**

2つの整数配列 `nums1` と `nums2` が与えられます。
両方の配列に共通して存在する**重複しない要素の集合**を配列として返す関数 `intersection` を実装してください。
結果の配列の順序は問いません。

### 🔹 **実行例**

#### 例 1:

```
入力: nums1 = [1, 2, 2, 1], nums2 = [2, 2]
出力: [2]
```

#### 例 2:

```
入力: nums1 = [4, 9, 5], nums2 = [9, 4, 9, 8, 4]
出力: [9, 4]
```

### **実行結果**

```sh
[2]
[4 9]
```
