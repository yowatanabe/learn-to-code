# No.182

### 問題: 重複する要素があるかを判定する

整数の配列 `nums` が与えられます。
配列内に **重複する要素が存在するかどうか** を判定し、存在すれば `true`、存在しなければ `false` を返す関数 `containsDuplicate` を実装してください。

### 実行例

#### 例 1:

```
入力: nums = [1, 2, 3, 4]
出力: false
```

#### 例 2:

```
入力: nums = [1, 2, 3, 1]
出力: true
```

#### 例 3:

```
入力: nums = []
出力: false
```

### 実行結果

```sh
false
true
false
```
