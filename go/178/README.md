### 問題: 配列の中に0が含まれているか判定する

整数の配列 `nums` が与えられます。
配列 `nums` に **少なくとも1つの `0` が含まれている場合** は `true` を、含まれていない場合は `false` を返す関数 `containsZero` を実装してください。

### 実行例

#### 例 1:

```
入力: nums = [1, 2, 3, 0, 4]
出力: true
```

#### 例 2:

```
入力: nums = [5, 6, 7]
出力: false
```

#### 例 3:

```
入力: nums = [0, 0, 0]
出力: true
```

### 実行結果

```sh
true
false
true
```
