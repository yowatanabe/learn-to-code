### 問題: 文字列に含まれる数字の合計

英数字からなる文字列 `s` が与えられます。
この文字列に含まれる **数字 (0-9)** の合計を求める関数 `sumOfDigits` を実装してください。

### 実行例

#### 例 1:

```
入力: s = "a1b2c3"
出力: 6
説明: 1 + 2 + 3 = 6
```

#### 例 2:

```
入力: s = "abc"
出力: 0
```

#### 例 3:

```
入力: s = "5a9b"
出力: 14
説明: 5 + 9 = 14
```

### 実行結果

```sh
6
0
14
```
