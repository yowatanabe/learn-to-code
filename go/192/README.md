# No.192

### 問題: 配列の中の最大値を返す

整数の配列 `nums` が与えられます。
この配列の中で最も大きい値を返す関数 `findMax` を実装してください。
配列には少なくとも1つ以上の要素が含まれていると仮定します。

### 実行例

#### 例 1:

```
入力: nums = [1, 3, 2, 5, 4]
出力: 5
```

#### 例 2:

```
入力: nums = [-10, -20, -5]
出力: -5
```

#### 例 3:

```
入力: nums = [7]
出力: 7
```

### 実行結果

```sh
5
-5
7
```
