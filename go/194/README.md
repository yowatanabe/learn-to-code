# No.194

### 問題: 配列内のゼロを末尾に移動させる

整数の配列 `nums` が与えられます。
すべての **0 を配列の末尾に移動**させ、非ゼロ要素の順序は変更しないようにしてください。
この操作を行った新しい配列を返す関数 `moveZeroes` を実装してください。

### 実行例

#### 例 1:

```
入力: nums = [0, 1, 0, 3, 12]
出力: [1, 3, 12, 0, 0]
```

#### 例 2:

```
入力: nums = [0, 0, 0]
出力: [0, 0, 0]
```

#### 例 3:

```
入力: nums = [1, 2, 3]
出力: [1, 2, 3]
```

### 実行結果

```sh
[1 3 12 0 0]
[0 0 0]
[1 2 3]
```
