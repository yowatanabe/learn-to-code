# No.191

### 問題: すべての文字が一意かを判定する

文字列 `s` が与えられます。
`s` に含まれるすべての文字が **一意（重複なし）** であれば `true` を、
同じ文字が複数回現れる場合は `false` を返す関数 `isUnique` を実装してください。
すべての文字は英小文字のみで構成されるとします。

### 実行例

#### 例 1:

```
入力: s = "abcdef"
出力: true
```

#### 例 2:

```
入力: s = "hello"
出力: false
```

#### 例 3:

```
入力: s = "world"
出力: true
```

### 実行結果

```sh
true
false
true
```
