# No.139

### **問題: 配列内の偶数カウント**

整数配列 `nums` が与えられます。配列内に含まれる **偶数の個数** を返す関数 `countEvens` を実装してください。

#### **制約:**

- `1 <= nums.length <= 10^4`
- `-10^5 <= nums[i] <= 10^5`

### **🔹 実行例**

#### **例 1**

```
入力: nums = [1, 2, 3, 4, 5, 6]
出力: 3
説明: 偶数は 2, 4, 6 の 3 個
```

#### **例 2**

```
入力: nums = [7, 11, 13]
出力: 0
```

#### **例 3**

```
入力: nums = [-2, 0, 3, 5, 8]
出力: 3
説明: 偶数は -2, 0, 8 の 3 個
```

### **実行結果**

```sh
3
0
3
```
