# No.150

### 問題: 配列内で特定の数が出現する回数

整数の配列 `nums` と整数 `target` が与えられます。
`nums` の中で `target` が **出現する回数** を返す関数 `countOccurrences` を実装してください。

### 実行例

#### 例 1:

```
入力: nums = [1, 2, 3, 2, 4, 2], target = 2
出力: 3
```

#### 例 2:

```
入力: nums = [5, 5, 5, 5, 5], target = 5
出力: 5
```

#### 例 3:

```
入力: nums = [1, 3, 5, 7], target = 2
出力: 0
```

### 実行結果

```sh
3
5
0
```
