# No.190

### 問題: 2つの文字列がアナグラムか判定する

2つの文字列 `s` と `t` が与えられます。
`t` が `s` の **アナグラム（文字を並べ替えてできる文字列）** であるなら `true` を、そうでないなら `false` を返す関数 `isAnagram` を実装してください。
すべての文字は英小文字のみで構成されるとします。

### 実行例

#### 例 1:

```
入力: s = "listen", t = "silent"
出力: true
```

#### 例 2:

```
入力: s = "hello", t = "world"
出力: false
```

#### 例 3:

```
入力: s = "anagram", t = "nagaram"
出力: true
```

### 実行結果

```sh
true
false
true
```
