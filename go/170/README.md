# No.170

### 問題: 文字列がすべて小文字か判定する

文字列 `s` が与えられます。
`s` がすべて **英小文字（a〜z）** で構成されていれば `true` を、そうでなければ `false` を返す関数 `isAllLowercase` を実装してください。

### 実行例

#### 例 1

```
入力: s = "leetcode"
出力: true
```

#### 例 2

```
入力: s = "LeetCode"
出力: false
```

#### 例 3

```
入力: s = "hello123"
出力: false
```

#### 例 4

```
入力: s = "abcxyz"
出力: true
```

### 実行結果

```sh
true
false
false
true
```
