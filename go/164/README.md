# No.164

### 問題: 配列の平均値を計算する

整数の配列 `nums` が与えられます。
この配列に含まれる数値の **平均値（小数含む）** を `float64` 型で返す関数 `average` を実装してください。

### 🔹 実行例

#### 例 1:

```
入力: nums = [1, 2, 3, 4, 5]
出力: 3.0
```

#### 例 2:

```
入力: nums = [10, 20, 30]
出力: 20.0
```

#### 例 3:

```
入力: nums = [42]
出力: 42.0
```

### 実行結果

```sh
3
20
42
```
