## 問題文

### 最小コストでログ保存ポリシーを満たす

あなたは `n` 日分のログ保存設定を決めています。

整数配列 `costs` が与えられ、`costs[i]` は `i` 日目のログを保存するコストを表します。

ただし、監査要件により、**連続する3日間のうち少なくとも1日はログを保存する必要があります。**

すべての期間でこの条件を満たすようにログを保存するとき、

必要な**最小コスト**を返してください。

### 制約

* `1 <= len(costs) <= 10^5`
* `1 <= costs[i] <= 10^4`

## 実行例

### 例1

```python
costs = [5, 1, 2, 10, 1]
```

1日目と4日目を保存すると、すべての連続3日間に少なくとも1日保存日が含まれます。

保存コストは

```python
1 + 1 = 2
```

出力:

```python
2
```

### 例2

```python
costs = [3, 2, 4]
```

3日間の中で少なくとも1日保存すればよいので、最小は `2` です。

出力:

```python
2
```

### 例3

```python
costs = [1, 100, 100, 1]
```

0日目と3日目を保存すると条件を満たします。

出力:

```python
2
```

## 実行結果

```
2
2
2
```
