# No.404

## 問題：合計が `k` になる連続部分配列の個数

あなたは SRE として、時系列メトリクスの増減を表す整数配列 `nums` を分析しています。

ある連続区間の合計がちょうど `k` になるとき、その区間を「条件一致区間」と呼びます。

整数配列 `nums` と整数 `k` が与えられたとき、**合計がちょうど `k` になる連続部分配列の個数**を求めてください。

* 部分配列は **連続** している必要があります
* `1 <= len(nums) <= 2 * 10^5`
* `-10^9 <= nums[i] <= 10^9`
* `-10^14 <= k <= 10^14`

**求めるもの**：条件を満たす連続部分配列の個数（整数）

## 実行例

例1:

* 入力: `nums = [1, 1, 1]`, `k = 2`
* 説明:

  * 合計が 2 になる区間は `[1,1]` が 2 個ある
  * `nums[0:2] = [1,1]`
  * `nums[1:3] = [1,1]`
* 出力: `2`

例2:

* 入力: `nums = [1, 2, 3]`, `k = 3`
* 説明:

  * 合計が 3 になる区間は `[1,2]` と `[3]`
* 出力: `2`

例3:

* 入力: `nums = [3, 4, 7, 2, -3, 1, 4, 2]`, `k = 7`
* 説明:

  * 合計が 7 になる区間は次の 4 個
  * `[3,4]`
  * `[7]`
  * `[7,2,-3,1]`
  * `[1,4,2]`
* 出力: `4`

## 実行結果

```
2
2
4
```
