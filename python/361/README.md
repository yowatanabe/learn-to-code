## 問題: イベントの重複を除いた稼働時間（Merge Intervals）

監視イベントの発生区間が `intervals[i] = [start, end]` で与えられます（`start <= end`）。

区間が重なったり接したりしている場合、それらは同じ稼働区間としてまとめたいです。

> すべての区間を **重なり（または接触）** がないようにマージし、マージ後の区間リストを返してください。
> ここでは **[a,b] と [b,c] は接しているのでマージする** とします。

### 制約（目安）

* `1 ≤ len(intervals) ≤ 10^5`
* `-10^9 ≤ start, end ≤ 10^9`

時間計算量 **O(n log n)** を目指してください。

## 実行例

```python
# 例1
intervals = [[1,3],[2,6],[8,10],[15,18]]
# [1,3] と [2,6] をマージして [1,6]
# 出力: [[1,6],[8,10],[15,18]]

# 例2
intervals = [[1,4],[4,5]]
# 接しているので [1,5] にマージ
# 出力: [[1,5]]

# 例3
intervals = [[5,5],[1,2],[2,4]]
# [1,2] と [2,4] は接しているので [1,4]、[5,5] は別
# 出力: [[1,4],[5,5]]
```
