## 問題: 重複のない最長部分文字列の長さ（Longest Substring Without Repeating Characters）

文字列 `s` が与えられます。

> 文字が重複しない（同じ文字を2回以上含まない）
> **連続部分文字列**のうち、長さが最大のものを求めてください。

### 制約（目安）

* `0 ≤ len(s) ≤ 10^5`
* `s` は ASCII 文字（英数字や記号）を含んでもよい

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
s = "abcabcbb"
# "abc" の長さ 3 が最大
# 出力: 3

# 例2
s = "bbbbb"
# "b" の長さ 1
# 出力: 1

# 例3
s = "pwwkew"
# "wke" の長さ 3
# 出力: 3

# 例4
s = ""
# 出力: 0
```

## 実行結果

```
例1 入力: 'abcabcbb'
例1 出力: 3
----------------------------------------
例2 入力: 'bbbbb'
例2 出力: 1
----------------------------------------
例3 入力: 'pwwkew'
例3 出力: 3
----------------------------------------
例4 入力: ''
例4 出力: 0
----------------------------------------
```
