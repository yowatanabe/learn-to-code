# No.135

## 問題：1回だけ出現する数字

整数のリスト `nums` が与えられます。
この中にはすべての数字が**2回ずつ**出現していますが、ただ1つの数字だけが**1回だけ**出現しています。
その数字を返してください。

※ 空でないリストであり、必ずそのような数字が1つ存在すると仮定してよい。

### 実行例

#### 例 1:

```
入力: nums = [2, 3, 5, 4, 5, 3, 4]
出力: 2
```

#### 例 2:

```
入力: nums = [10, 1, 1, 2, 2]
出力: 10
```

### 実行結果

```text
2
10
```
