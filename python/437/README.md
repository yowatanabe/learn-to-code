## 問題文

### 障害チケットの重複範囲を検出する

あなたは障害チケットの対応時間を管理しています。各チケットには対応開始時刻と終了時刻があり、`tickets[i] = [start, end]` で表されます。

ここで、`start` 以上 `end` 未満の時間帯を対応時間とします。つまり `[start, end)` です。

**同時に対応中のチケット数の最大値**を返してください。

### 制約

* `1 <= len(tickets) <= 10^5`
* `0 <= start < end <= 10^9`

## 実行例

### 例1

```python
tickets = [[1, 5], [2, 6], [4, 8]]
```

時間 `4` から `5` の間は3件同時に対応中です。

出力:

```python
3
```

### 例2

```python
tickets = [[1, 3], [3, 5], [5, 7]]
```

`[1, 3)` と `[3, 5)` は端で接しているだけで、重なっていません。

出力:

```python
1
```

### 例3

```python
tickets = [[10, 20], [15, 25], [20, 30]]
```

`[10, 20)` と `[20, 30)` は重なりませんが、
`[15, 25)` は両方と一部重なります。

最大同時対応数は

```python
2
```

です。

## 実行結果

```
3
1
2
```
