## 問題文

### しきい値以上のメトリクスを含む最短区間

あなたは1分ごとのメトリクス値を監視しています。整数配列 `metrics` と整数 `target` が与えられます。

**合計値が `target` 以上になる連続部分配列のうち、最も短い長さ**を返してください。

そのような区間が存在しない場合は `0` を返してください。

### 制約

* `1 <= len(metrics) <= 10^5`
* `1 <= metrics[i] <= 10^4`
* `1 <= target <= 10^9`

## 実行例

### 例1

```python
metrics = [1, 2, 3, 4, 5]
target = 11
```

`[3, 4, 5]` の合計は `12` です。
長さは `3` なので、出力は

```python
3
```

### 例2

```python
metrics = [2, 2, 2]
target = 7
```

合計が `7` 以上になる区間はありません。

出力は

```python
0
```

### 例3

```python
metrics = [5, 1, 1, 1]
target = 5
```

`[5]` だけで条件を満たします。

出力は

```python
1
```

## 実行結果

```
3
0
1
```
