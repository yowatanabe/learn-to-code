# No.490

## 問題文

### 障害が全サーバーへ波及するまでの時間

2次元配列 `grid` が与えられます。

* `0`: 空きラック
* `1`: 正常サーバー
* `2`: 障害発生中サーバー

障害は1分ごとに、上下左右に隣接する正常サーバーへ波及します。

すべての正常サーバーに障害が波及するまでの最短時間を返してください。

不可能な場合は `-1` を返してください。

## 実行例

### 例1

```python
grid = [
    [2, 1, 1],
    [1, 1, 0],
    [0, 1, 1]
]
```

出力:

```python
4
```

### 例2

```python
grid = [
    [2, 1, 1],
    [0, 1, 1],
    [1, 0, 1]
]
```

左下の正常サーバーには障害が届きません。

出力:

```python
-1
```

### 例3

```python
grid = [
    [0, 2]
]
```

正常サーバーがないため、出力は

```python
0
```

## 実行結果

```
4
-1
0
```
