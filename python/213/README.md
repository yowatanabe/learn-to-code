# No.213

## 問題文

整数配列 `nums` と整数 `k`（`1 <= k <= len(nums)`）が与えられます。

`nums` の中で k 番目に大きい要素（重複を区別した順位）を返してください。

可能なら 平均 O(n) 時間（クイックセレクト）で実装してください。

* 例：`nums = [3,2,1,5,6,4], k = 2` → 2番目に大きい要素は `5`
* `1 <= len(nums) <= 10^5`
* `-10^9 <= nums[i] <= 10^9`

## 実行例

```
入力: nums = [3,2,1,5,6,4], k = 2
出力: 5

入力: nums = [3,2,3,1,2,4,5,5,6], k = 4
出力: 4

入力: nums = [1], k = 1
出力: 1

入力: nums = [7,7,7,7], k = 3
出力: 7
```

## 実行結果

```
5
4
1
7
```
