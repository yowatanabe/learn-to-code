# No.173

## 問題文

**整数の配列 `nums` が与えられます。
最大で1つの要素を削除することで、配列が単調増加（各要素が前の要素以上）になるかどうかを判定してください。
可能であれば `True` を、そうでなければ `False` を返してください。**

## 実行例

```
入力: nums = [4,2,3]
出力: True
説明: 4を2に変更するか、4を削除すれば単調増加になる。

入力: nums = [4,2,1]
出力: False
説明: 1つだけ削除しても単調増加にはできない。

入力: nums = [1,2,10,5,7]
出力: True
説明: 10を削除すれば [1,2,5,7] で単調増加になる。
```

### 実行結果

```
True
False
True
```
