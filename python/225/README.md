# No.225

## 問題文

文字列 `s` が与えられます。

`s` の中で 最長の回文部分文字列（前から読んでも後ろから読んでも同じになる連続部分文字列）を返してください。

最長が複数ある場合は、最も左に現れるものを返してください。

* `1 <= len(s) <= 2000`
* `s` は英数字や記号など任意のASCII文字を含んでもよい

## 実行例

```
入力: s = "babad"
出力: "bab"
説明: "aba" も最長だが、左にある "bab" を返す。

入力: s = "cbbd"
出力: "bb"

入力: s = "a"
出力: "a"

入力: s = "forgeeksskeegfor"
出力: "geeksskeeg"
```

## 実行結果

```
bab
bb
a
geeksskeeg
```
