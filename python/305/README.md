## 問題: 0 を含むときの積配列（Product of Array Except Self）

整数配列 `nums` が与えられます。
配列 `answer` を次の条件で作って返してください。

* `answer[i]` は **`nums[i]` 以外のすべての要素の積**

つまり `answer[i] = nums[0] * ... * nums[i-1] * nums[i+1] * ... * nums[n-1]`

制約として、**割り算（除算）を使ってはいけません**。

### 制約（目安）

* `2 ≤ len(nums) ≤ 10^5`
* `-30 ≤ nums[i] ≤ 30`
* 積は 32bit 整数に収まると仮定してよい

時間計算量 `O(n)`、追加メモリ `O(1)`（出力配列を除く）を目指してください。

## 実行例

```python
# 例1
nums = [1, 2, 3, 4]
# answer[0] = 2*3*4 = 24
# answer[1] = 1*3*4 = 12
# answer[2] = 1*2*4 = 8
# answer[3] = 1*2*3 = 6
# 出力: [24, 12, 8, 6]

# 例2（0 を含む）
nums = [-1, 1, 0, -3, 3]
# i=2 以外は積に 0 が入るので 0 になる
# answer[2] は 0 を除いた積: (-1)*1*(-3)*3 = 9
# 出力: [0, 0, 9, 0, 0]
```

## 実行結果

```
例1の入力: [1, 2, 3, 4]
例1の結果: [24, 12, 8, 6]
例2の入力: [-1, 1, 0, -3, 3]
例2の結果: [0, 0, 9, 0, 0]
```
