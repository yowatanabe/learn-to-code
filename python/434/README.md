## 問題文

### K回以内の設定変更で最長の正常稼働期間を作る

あなたはあるサービスの稼働状況を日ごとに記録しています。文字列配列 `status` が与えられ、各要素はその日の状態を表します。

* `"up"`: 正常稼働
* `"down"`: 停止

あなたは **最大で `k` 回まで** `"down"` を `"up"` に変更できるとします。

このとき、**連続する `"up"` の長さの最大値** を求めてください。

その最大長を返してください。

### 制約

* `1 <= len(status) <= 10^5`
* `0 <= k <= len(status)`
* `status[i]` は `"up"` または `"down"`

## 実行例

### 例1

```python
status = ["up", "down", "up", "up", "down", "up"]
k = 1
```

`"down"` を1回だけ `"up"` に変更できます。
例えば index 1 を変更すると、

```python
["up", "up", "up", "up", "down", "up"]
```

となり、連続する `"up"` の最大長は

```python
4
```

です。

したがって出力は

```python
4
```

### 例2

```python
status = ["down", "down", "up", "down", "up", "up"]
k = 2
```

2回まで変更できるので、例えば先頭2つの `"down"` を `"up"` に変更すると、

```python
["up", "up", "up", "down", "up", "up"]
```

となります。

さらに別の取り方として、index 1 と index 3 を変更すると

```python
["down", "up", "up", "up", "up", "up"]
```

となり、連続する `"up"` の最大長は

```python
5
```

です。

したがって出力は

```python
5
```

### 例3

```python
status = ["down", "down", "down"]
k = 3
```

すべて `"up"` に変更できるので、出力は

```python
3
```

## 実行結果

```
4
5
3
```
