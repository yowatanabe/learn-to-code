## 問題文

### 最小コストで全リージョンを接続する

あなたは複数リージョン間のネットワーク接続を設計しています。`n` 個のリージョンがあり、`0` から `n - 1` の番号が付いています。

接続候補は `connections` で与えられます。

```python
connections[i] = [regionA, regionB, cost]
```

これは `regionA` と `regionB` を接続するのに `cost` かかることを表します。

すべてのリージョンが直接または間接的につながるようにするとき、**必要な最小コスト**を返してください。

すべてのリージョンを接続できない場合は `-1` を返してください。

### 制約

* `1 <= n <= 10^5`
* `0 <= len(connections) <= 2 * 10^5`
* `0 <= regionA, regionB < n`
* `1 <= cost <= 10^9`

## 実行例

### 例1

```python
n = 4
connections = [
    [0, 1, 1],
    [1, 2, 2],
    [0, 2, 4],
    [2, 3, 1]
]
```

最小コストで接続するには、

```python
0 - 1 cost 1
1 - 2 cost 2
2 - 3 cost 1
```

を選びます。

合計は

```python
4
```

なので、出力は

```python
4
```

### 例2

```python
n = 3
connections = [
    [0, 1, 5]
]
```

リージョン `2` がどこにも接続できません。

出力は

```python
-1
```

### 例3

```python
n = 3
connections = [
    [0, 1, 10],
    [1, 2, 1],
    [0, 2, 3]
]
```

最小コストは、

```python
1 - 2 cost 1
0 - 2 cost 3
```

合計

```python
4
```

です。

## 実行結果

```
4
-1
4
```
