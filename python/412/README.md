# No.412

## 問題：障害ウィンドウのマージ後件数

あなたは SRE として、障害が発生した時間帯をまとめて管理しています。

各障害ウィンドウは `[start, end]` の形式で与えられ、`start <= end` を満たします。

2つのウィンドウが **重なっている** または **端で接している** 場合、それらは 1 つの障害としてまとめます。

* 例: `[1,3]` と `[2,5]` はまとめて `[1,5]`
* 例: `[4,6]` と `[6,8]` も接しているのでまとめて `[4,8]`

整数の二次元配列 `windows` が与えられたとき、

すべてのウィンドウをまとめた後の **区間数** を返してください。

* `1 <= len(windows) <= 2 * 10^5`
* `0 <= start <= end <= 10^9`

**求めるもの**：マージ後の区間数（整数）

## 実行例

例1:

* 入力: `windows = [[1,3],[2,5],[8,10],[10,12]]`
* 説明:

  * `[1,3]` と `[2,5]` は重なるので `[1,5]`
  * `[8,10]` と `[10,12]` は端で接しているので `[8,12]`
  * 最終的に `[[1,5],[8,12]]`
* 出力: `2`

例2:

* 入力: `windows = [[1,2],[4,5],[7,8]]`
* 説明:

  * どの区間も重ならず、端でも接していない
* 出力: `3`

例3:

* 入力: `windows = [[5,5],[1,4],[4,4]]`
* 説明:

  * `[1,4]` と `[4,4]` は接しているので `[1,4]`
  * `[1,4]` と `[5,5]` は接していない
  * 最終的に `[[1,4],[5,5]]`
* 出力: `2`

## 実行結果

```
2
3
2
```
