## 問題: 配列の一部を重複なしにする最小操作回数（Minimum Operations to Make Array Unique）

整数配列 `nums` が与えられます。

> 配列 `nums` から重複する要素を **最小回数で削除**して、すべての要素が **ユニーク（重複なし）** になるようにしてください。
> この操作を行うために必要な **最小の削除回数** を返してください。

### 制約（目安）

* `1 ≤ len(nums) ≤ 10^5`
* `0 ≤ nums[i] ≤ 10^9`

## 実行例

```python
# 例1
nums = [1, 2, 2, 3, 3, 3]
# 2,3 が重複しているので、削除最小は 3 回
# 出力: 3

# 例2
nums = [5, 5, 5, 5]
# すべて重複しているので、削除回数は 3 回
# 出力: 3

# 例3
nums = [1, 2, 3]
# すでにユニークな配列
# 出力: 0
```

## 実行結果

```
例1 入力: nums=[1, 2, 2, 3, 3, 3]
例1 出力: 3 (期待値: 2)
------------------------------------------------------------
例2 入力: nums=[5, 5, 5, 5]
例2 出力: 3 (期待値: 3)
------------------------------------------------------------
例3 入力: nums=[1, 2, 3]
例3 出力: 0 (期待値: 0)
------------------------------------------------------------
```
