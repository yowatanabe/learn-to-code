## 問題文

### K個のログストリームから最小範囲を見つける

あなたは複数のログストリームを監視しています。

各ログストリームには、エラーが発生した時刻が昇順で記録されています。

2次元配列 `streams` が与えられます。

```python
streams[i]
```

は `i` 番目のログストリームのエラー発生時刻一覧です。

**すべてのログストリームから少なくとも1つの時刻を含む、最小の時間範囲 `[start, end]`** を返してください。

範囲の長さは `end - start` とします。
長さが同じ場合は、`start` が小さい範囲を返してください。

### 制約

* `1 <= len(streams) <= 3500`
* `1 <= len(streams[i])`
* 全要素数の合計は `10^5` 以下
* `0 <= streams[i][j] <= 10^9`
* 各 `streams[i]` は昇順

## 実行例

### 例1

```python
streams = [
    [4, 10, 15, 24, 26],
    [0, 9, 12, 20],
    [5, 18, 22, 30]
]
```

各ストリームから少なくとも1つ含む最小範囲は、

```python
[20, 24]
```

です。

* 1番目のストリーム: `24`
* 2番目のストリーム: `20`
* 3番目のストリーム: `22`

を含みます。

出力:

```python
[20, 24]
```

### 例2

```python
streams = [
    [1, 2, 3],
    [1, 2, 3],
    [1, 2, 3]
]
```

すべてのストリームに `1` があるため、最小範囲は

```python
[1, 1]
```

です。

### 例3

```python
streams = [
    [1, 5],
    [10],
    [12, 15]
]
```

すべてのストリームから1つずつ含めるには、少なくとも

```python
[5, 12]
```

が必要です。

## 実行結果

```
[20, 24]
[1, 1]
[5, 12]
```
