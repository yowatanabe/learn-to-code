## 問題: 最長の回文部分文字列（Longest Palindromic Substring）

文字列 `s` が与えられます。

`s` の中で、最長の回文部分文字列を返してください。

**回文**とは、前から読んでも後ろから読んでも同じ文字列です。

例えば、`"racecar"` は回文です。

### 制約（目安）

* `1 ≤ len(s) ≤ 1000`

### 注意

* 回文部分文字列は連続している必要があります。
* 出力は最長の回文部分文字列を返すこと。

## 実行例

```python
# 例1
s = "babad"
# 最長の回文部分文字列は "bab" または "aba" です
# 出力: "bab" または "aba"

# 例2
s = "cbbd"
# 最長の回文部分文字列は "bb"
# 出力: "bb"

# 例3
s = "a"
# 最長の回文部分文字列は "a"
# 出力: "a"

# 例4
s = "ac"
# 最長の回文部分文字列は "a" または "c"
# 出力: "a" または "c"
```

## 実行結果

```

例1 出力: bab

----------------------------------------

----------------------------------------
例4 入力: s='ac'
例4 出力: a
----------------------------------------
```
