# No.102

### 問題: 2つのリストに共通する要素を返す

整数のリスト `list1` と `list2` が与えられます。
これらのリストに共通して含まれる要素を、重複なしでリストとして返す関数 `common_elements` を実装してください。
返却するリストの要素の順序は問わないものとします。

### 実行例

#### 例 1:

```
入力: list1 = [1, 2, 3], list2 = [2, 3, 4]
出力: [2, 3]
```

#### 例 2:

```
入力: list1 = [5, 6, 7], list2 = [7, 8, 9]
出力: [7]
```

#### 例 3:

```
入力: list1 = [1, 2], list2 = [3, 4]
出力: []
```

### 実行結果（例）

```text
[2, 3]
[7]
[]
```
