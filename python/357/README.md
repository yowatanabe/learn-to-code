# No.357

## 問題: “次に大きい値”までの日数（Daily Temperatures）

整数配列 `temps` が与えられます。`temps[i]` は i 日目の気温です。

> 各日 i について、**temps[i] より大きい気温**が出現する **最初の日**まで何日待つ必要があるかを求め、配列 `ans` を返してください。
> もし将来により大きい気温が存在しない場合は `ans[i] = 0` としてください。

### 制約（目安）

* `1 ≤ len(temps) ≤ 10^5`
* `-10^9 ≤ temps[i] ≤ 10^9`

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
temps = [73, 74, 75, 71, 69, 72, 76, 73]
# 出力: [1, 1, 4, 2, 1, 1, 0, 0]

# 例2
temps = [30, 40, 50, 60]
# 出力: [1, 1, 1, 0]

# 例3
temps = [60, 50, 40, 30]
# 出力: [0, 0, 0, 0]
```

## 実行結果

```

------------------------------------------------------------

例2 出力: [1, 1, 1, 0] (期待値: [1, 1, 1, 0])
------------------------------------------------------------
```
