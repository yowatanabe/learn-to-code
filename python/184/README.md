# No.184

## 問題文

整数配列 `nums` と整数 `k` が与えられます。

配列内で出現回数が多い要素を**上位 `k` 個**返してください。

* 返す順序は任意（どの順でも可）。
* もし同じ頻度の要素が複数ある場合、どれを返しても良い。
* 可能なら **O(n log k)** または **O(n)** で実装してください。
  （`n = len(nums)`）

## 実行例

```
入力: nums = [1,1,1,2,2,3], k = 2
出力: [2,1]
説明: 1の出現回数は3回、2は2回、3は1回。上位2個は[1,2]。

入力: nums = [4,4,4,6,6,7], k = 1
出力: [4]
説明: 4が最頻。

入力: nums = [1], k = 1
出力: [1]
```

## 実行結果

```
[2, 1]
[4]
[1]
```
