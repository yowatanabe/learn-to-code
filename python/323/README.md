## 問題: 2つの配列から最小の組を K 個求めよ（Find K Pairs with Smallest Sums）

昇順にソートされた 2 つの整数配列 `nums1` と `nums2`、そして整数 `k` が与えられます。

> `(u, v)`（`u` は `nums1` から、`v` は `nums2` から選ぶ）というペアのうち、
> **和 `u + v` が小さい順に k 個** のペアを返してください。

* 同じ値でも **インデックスが異なれば別の要素**として扱います。
* 返す順序は問いません（ただし「k 個の最小和ペア」であること）。
* `k` 個未満しかペアが存在しない場合は、存在するペアをすべて返してください。

### 制約（目安）

* `1 ≤ len(nums1), len(nums2) ≤ 10^5`（ただし `k` は比較的小さい想定）
* `1 ≤ k ≤ 10^4`
* `nums1`, `nums2` は昇順

時間計算量は **O(k log k)** 付近を目指してください（全列挙はNG）。

## 実行例

```python
# 例1
nums1 = [1,7,11]
nums2 = [2,4,6]
k = 3
# 和が小さい順:
# (1,2)=3, (1,4)=5, (1,6)=7, (7,2)=9, ...
# 出力例: [(1,2), (1,4), (1,6)]

# 例2
nums1 = [1,1,2]
nums2 = [1,2,3]
k = 2
# 最小:
# (1,1)=2, (1,1)=2（nums1の別インデックス）
# 出力例: [(1,1), (1,1)]

# 例3
nums1 = [1,2]
nums2 = [3]
k = 5
# ペアは2つしかない
# 出力: [(1,3), (2,3)]
```

## 実行結果

```

例1 出力: [(1, 2), (1, 4), (1, 6)]

例2 出力: [(1, 1), (1, 1)]
----------------------------------------
例3 入力: nums1=[1, 2], nums2=[3], k=5
例3 出力: [(1, 3), (2, 3)]
----------------------------------------
```
