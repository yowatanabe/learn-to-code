## 問題文

### 無効な監視クエリから括弧を削除する

文字列 `query` が与えられます。

`query` には英小文字と括弧 `"("`, `")"` が含まれます。

無効な括弧を最小数だけ削除して、括弧の対応が正しい文字列を返してください。

返す文字列が複数ある場合は、どれを返しても構いません。

## 実行例

### 例1

```python
query = "a)b(c)d"
```

出力:

```python
"ab(c)d"
```

### 例2

```python
query = "((log)"
```

出力:

```python
"(log)"
```

### 例3

```python
query = "api))("
```

出力:

```python
"api"
```

## 実行結果

```
ab(c)d
(log)
api
```
