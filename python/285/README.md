## 問題: 自分以外の要素の積

整数配列 `nums` が与えられます。

各インデックス `i` について、

> 「`nums[i]` 以外の **すべての要素** の積」

を要素にもつ配列を返してください。

* **割り算は使ってはいけません。**
* 時間計算量は O(n) を目指してください。
* 追加メモリ使用量は O(1)（戻り値の配列を除く）で解けるとなお良いです。

### 制約（目安）

* `2 ≤ len(nums) ≤ 10^5`
* `-30 ≤ nums[i] ≤ 30`
* 少なくとも 1 つは 0 でない要素が存在すると仮定してよい

## 実行例

```python
# 例1
nums = [1, 2, 3, 4]
# 各インデックスごとに自分以外の積は以下の通り:
# i = 0: 2 * 3 * 4 = 24
# i = 1: 1 * 3 * 4 = 12
# i = 2: 1 * 2 * 4 = 8
# i = 3: 1 * 2 * 3 = 6
# 出力: [24, 12, 8, 6]

# 例2
nums = [-1, 1, 0, -3, 3]
# i = 0: 1 * 0 * -3 * 3 = 0
# i = 1: -1 * 0 * -3 * 3 = 0
# i = 2: -1 * 1 * -3 * 3 = 9
# i = 3: -1 * 1 * 0 * 3 = 0
# i = 4: -1 * 1 * 0 * -3 = 0
# 出力: [0, 0, 9, 0, 0]
```

## 実行結果

```
例1の入力: [1, 2, 3, 4]
例1の結果: [24, 12, 8, 6]
例2の入力: [-1, 1, 0, -3, 3]
例2の結果: [0, 0, 9, 0, 0]
```
