## 問題文

### 最小ジャンプ回数で最後のサーバーへ到達する

あなたは複数台のサーバーを順番にたどる監視フローを設計しています。整数配列 `jumps` が与えられ、`jumps[i]` は **サーバー `i` から最大で何台先までジャンプできるか** を表します。

最初は `0` 番目のサーバーにいます。
**最後のサーバーに到達するために必要な最小ジャンプ回数** を返してください。

最後のサーバーには必ず到達できるものとします。

### 制約

* `1 <= len(jumps) <= 10^5`
* `0 <= jumps[i] <= 10^5`
* 最後のサーバーには必ず到達可能

## 実行例

### 例1

```python
jumps = [2, 3, 1, 1, 4]
```

* 最初に index `0` から index `1` へジャンプ
* 次に index `1` から index `4` へジャンプ

したがって、最小ジャンプ回数は

```python
2
```

です。

### 例2

```python
jumps = [2, 3, 0, 1, 4]
```

* index `0` から index `1`
* index `1` から index `4`

したがって、出力は

```python
2
```

です。

### 例3

```python
jumps = [1, 2, 1, 1, 1]
```

* index `0` から index `1`
* index `1` から index `3`
* index `3` から index `4`

したがって、出力は

```python
3
```

です。

## 実行結果

```
2
2
3
```
