# No.116

### 問題: 回文判定

文字列 `s` が与えられます。
この文字列が前から読んでも後ろから読んでも同じ（＝回文）であるかどうかを判定する関数を作成してください。
大文字と小文字は区別せず、英数字以外の文字は無視してください。

### 実行例

#### 例 1:

```
入力: s = "A man, a plan, a canal: Panama"
出力: True
説明: "amanaplanacanalpanama" は回文です。
```

#### 例 2:

```
入力: s = "race a car"
出力: False
```

#### 例 3:

```
入力: s = " "
出力: True
```

### 実行結果

```text
True
False
True
```
