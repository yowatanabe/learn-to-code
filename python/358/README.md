## 問題: 依存タスクの実行順を判定せよ（Course Schedule / Topological Sort）

SRE の運用タスク（デプロイ、マイグレーション、設定変更など）には依存関係があります。

`n` 個のタスクがあり、タスクは `0` から `n-1` のIDで表されます。

依存関係は `prerequisites` で与えられ、`prerequisites[i] = [a, b]` は

> タスク `a` を実行するには **先に** タスク `b` を完了している必要がある

ことを意味します。

> すべてのタスクを依存関係を守って完了できるなら `True`、不可能（依存が循環しているなど）なら `False` を返してください。

### 制約（目安）

* `1 ≤ n ≤ 10^5`
* `0 ≤ len(prerequisites) ≤ 2*10^5`
* `0 ≤ a, b < n`

時間計算量 **O(n + m)**（m = 依存関係数）を目指してください。

## 実行例

```python
# 例1
n = 2
prerequisites = [[1,0]]
# 0 -> 1 の順でできる
# 出力: True

# 例2
n = 2
prerequisites = [[1,0],[0,1]]
# 0 が 1 に依存し、1 が 0 に依存して循環
# 出力: False

# 例3
n = 4
prerequisites = [[1,0],[2,0],[3,1],[3,2]]
# 0 -> (1,2) -> 3 が可能
# 出力: True
```

## 実行結果

```
例1 入力: n=2, prerequisites=[[1, 0]]
例1 出力: True (期待値: True)
------------------------------------------------------------
例2 入力: n=2, prerequisites=[[1, 0], [0, 1]]
例2 出力: False (期待値: False)
------------------------------------------------------------
例3 入力: n=4, prerequisites=[[1, 0], [2, 0], [3, 1], [3, 2]]
例3 出力: True (期待値: True)
```
