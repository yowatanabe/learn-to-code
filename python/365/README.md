# No.365

## 問題: 監視対象を最小のコストで接続せよ（Minimum Cost to Connect All Points）

監視対象ノードが `n` 個あり、各ノードは 2 次元平面上の点 `points[i] = [xi, yi]` として与えられます。

ノード `i` と `j` を接続するコストは **マンハッタン距離**で定義されます。

- `cost(i, j) = |xi - xj| + |yi - yj|`

すべてのノードが **連結**（どのノードからでも他のノードへ到達可能）になるように接続したいです。

> すべてのノードを接続するための **最小コスト** を返してください。

### 制約（目安）

- `1 ≤ n ≤ 1000`
- `-10^6 ≤ xi, yi ≤ 10^6`

期待解法: 最小全域木（MST）

時間計算量は **O(n^2)** 程度を想定（Prim 法）。

## 実行例

```python
# 例1
points = [[0,0],[2,2],[3,10],[5,2],[7,0]]
# 出力: 20

# 例2
points = [[3,12],[-2,5],[-4,1]]
# 出力: 18

# 例3
points = [[0,0]]
# 出力: 0
```

## 実行結果
