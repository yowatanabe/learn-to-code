# No.309

## 問題: K 個のソート済みリストをマージせよ（Merge k Sorted Lists）

`k` 個の **昇順にソートされた**連結リスト `lists` が与えられます。
それらを 1 本の昇順連結リストにマージして返してください。

* `lists[i]` は連結リストの先頭ノード（空の場合は `None`）
* 連結リストのノードを新しく作ってもよいですが、できれば既存ノードをつなぎ替えてもOKです。
* 時間計算量は **O(N log k)**（`N` は全ノード数）を目指してください。

### 制約（目安）

* `0 ≤ k ≤ 10^4`
* `0 ≤ 各リストの長さ ≤ 10^4`
* `-10^4 ≤ Node.val ≤ 10^4`
* 全ノード数 `N` は `10^4` 程度までを想定

## 実行例

```python
# 例1
lists = [
  1 -> 4 -> 5,
  1 -> 3 -> 4,
  2 -> 6
]
# 出力:
# 1 -> 1 -> 2 -> 3 -> 4 -> 4 -> 5 -> 6

# 例2
lists = []
# 出力: (empty)

# 例3
lists = [ (empty) ]
# 出力: (empty)
```

## 実行結果

```
例1 出力: [1, 1, 2, 3, 4, 4, 5, 6]
例2 出力: []
例3 出力: []
```
