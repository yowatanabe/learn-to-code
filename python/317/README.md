## 問題: 目標和となる最短部分配列（Minimum Size Subarray Sum）

正の整数からなる配列 `nums` と正の整数 `target` が与えられます。

> 合計が **`target` 以上** となる **連続部分配列**のうち、
> 長さが最小のものの長さを返してください。
> 条件を満たす部分配列が存在しない場合は `0` を返してください。

### 制約（目安）

* `1 ≤ len(nums) ≤ 10^5`
* `1 ≤ nums[i] ≤ 10^4`
* `1 ≤ target ≤ 10^9`

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
target = 7
nums = [2,3,1,2,4,3]
# 合計 >= 7 の最短は [4,3] で長さ 2
# 出力: 2

# 例2
target = 4
nums = [1,4,4]
# [4] で長さ 1
# 出力: 1

# 例3
target = 11
nums = [1,1,1,1,1,1,1,1]
# どの部分配列でも 11 に届かない
# 出力: 0
```

## 実行結果

```

例1 出力: 2
----------------------------------------
例2 入力: target=4, nums=[1, 4, 4]
例2 出力: 1
----------------------------------------
例3 入力: target=11, nums=[1, 1, 1, 1, 1, 1, 1, 1]
例3 出力: 0
----------------------------------------
```
