## 問題文

### コンテナイメージタグの最長連続バージョン

あなたはコンテナイメージのタグ一覧を管理しています。

整数配列 `versions` が与えられ、`versions[i]` は存在するイメージバージョン番号を表します。

このとき、**連続する整数バージョンの最長長さ**を返してください。

順番は関係ありません。

### 制約

* `0 <= len(versions) <= 10^5`
* `-10^9 <= versions[i] <= 10^9`

## 実行例

### 例1

```python
versions = [100, 4, 200, 1, 3, 2]
```

連続するバージョンは

```python
[1, 2, 3, 4]
```

なので、出力は

```python
4
```

### 例2

```python
versions = [10, 5, 6, 7, 20]
```

連続するバージョンは

```python
[5, 6, 7]
```

なので、出力は

```python
3
```

### 例3

```python
versions = [1, 2, 2, 3]
```

重複は1回として扱います。
連続するバージョンは

```python
[1, 2, 3]
```

なので、出力は

```python
3
```

## 実行結果

```
4
3
3
```
