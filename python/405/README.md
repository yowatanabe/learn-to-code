# No.405

## 問題：ローテーションされた昇順配列からターゲットを探す

整数配列 `nums` は、もともと **昇順にソートされた配列**でしたが、ある位置を境にローテーションされています。

例えば、`[0,1,2,4,5,6,7]` は次のようにローテーションされる可能性があります。

* `[4,5,6,7,0,1,2]`
* `[0,1,2,4,5,6,7]` （ローテーションなし）

整数 `target` が与えられたとき、`target` が `nums` に存在するならその **インデックス** を返し、存在しないなら `-1` を返してください。

**計算量は O(log n) を目指してください。**

* `1 <= len(nums) <= 2 * 10^5`
* `-10^9 <= nums[i], target <= 10^9`
* `nums` の値はすべて異なる
* `nums` はローテーションされた昇順配列

**求めるもの**：`target` のインデックス。存在しなければ `-1`

## 実行例

例1:

* 入力: `nums = [4,5,6,7,0,1,2]`, `target = 0`
* 説明:

  * `0` はインデックス `4` にある
* 出力: `4`

例2:

* 入力: `nums = [4,5,6,7,0,1,2]`, `target = 3`
* 説明:

  * `3` は存在しない
* 出力: `-1`

例3:

* 入力: `nums = [1]`, `target = 0`
* 説明:

  * `0` は存在しない
* 出力: `-1`

## 実行結果

```
4
-1
-1
```
