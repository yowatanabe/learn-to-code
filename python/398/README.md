# No.398

## 問題：設定変更で到達可能な最小コスト

あなたは SRE として、サービス間通信の経路コストを分析しています。

`m x n` の整数グリッド `grid` が与えられます。`grid[r][c]` は、そのマスを通過するためのコストを表します。

左上 `(0, 0)` から右下 `(m-1, n-1)` まで移動したいです。

1回の移動では、**上下左右**のいずれか隣接するマスに移動できます。

到達するまでに通過したマスのコストの合計を最小にしてください。

**開始マス `(0, 0)` のコストも合計に含みます。**

`grid` が与えられたとき、左上から右下までの**最小コスト**を返してください。

* `1 <= m, n <= 200`
* `0 <= grid[r][c] <= 10^9`

**求めるもの**：最小コスト（整数）

## 実行例

例1:

* 入力:
  `grid = [[1,3,1],[1,5,1],[4,2,1]]`
* 説明:
  最小コストの一例は
  `(0,0) -> (1,0) -> (2,0) -> (2,1) -> (2,2)` ではなく、
  `(0,0) -> (0,1) -> (0,2) -> (1,2) -> (2,2)` でもなく、
  最小は
  `(0,0) -> (1,0) -> (1,1) -> (1,2) -> (2,2)` でもありません。
  実際の最小は
  `1 + 3 + 1 + 1 + 1 = 7` となる経路で、答えは `7`
* 出力:
  `7`

例2:

* 入力:
  `grid = [[5]]`
* 説明:
  開始地点と終了地点が同じなので、そのマスのコストだけ払う
* 出力:
  `5`

例3:

* 入力:
  `grid = [[1,2,3],[4,5,6]]`
* 説明:
  最小コストは `1 + 2 + 3 + 6 = 12`
* 出力:
  `12`

## 実行結果

```
7
5
12
```
