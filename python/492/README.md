## 問題文

### 監視ログから上位K種類のイベントを取得する

あなたは監視ログに出現するイベント種別を分析しています。

文字列配列 `events` が与えられ、`events[i]` は発生したイベント種別を表します。

整数 `k` が与えられたとき、**出現回数が多い順に上位 `k` 種類のイベント**を返してください。

出現回数が同じ場合は、イベント名の辞書順が小さいものを先にしてください。

## 実行例

### 例1

```python
events = ["error", "warn", "error", "info", "warn", "error"]
k = 2
```

出現回数は以下です。

```python
error: 3
warn: 2
info: 1
```

出力:

```python
["error", "warn"]
```

### 例2

```python
events = ["db", "api", "cache", "api", "db"]
k = 2
```

`"api"` と `"db"` はどちらも2回出現します。
辞書順では `"api"` が先です。

出力:

```python
["api", "db"]
```

### 例3

```python
events = ["a", "b", "c"]
k = 3
```

すべて1回ずつです。

出力:

```python
["a", "b", "c"]
```

## 実行結果

```
['error', 'warn']
['api', 'db']
['a', 'b', 'c']
```
