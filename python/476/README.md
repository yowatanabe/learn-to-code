## 問題文

### 1日分の監視データを削除した後の最長正常期間

あなたはサービスの稼働状況を日ごとに管理しています。

文字列 `status` が与えられます。

* `"1"` : 正常
* `"0"` : 異常

あなたは **必ず1日分のデータを削除しなければなりません。**

削除後に得られる文字列の中で、

**連続する `"1"` の最大長**

を返してください。

## 制約

* `1 <= len(status) <= 10^5`
* `status[i]` は `"0"` または `"1"`

## 実行例

### 例1

```python
status = "110111"
```

異常日の `"0"` を削除すると、

```python
11111
```

となります。

出力:

```python
5
```

### 例2

```python
status = "10110"
```

index 1 の `"0"` を削除すると、

```python
1110
```

となります。

最長の連続 `"1"` は

```python
3
```

です。

出力:

```python
3
```

### 例3

```python
status = "1111"
```

必ず1文字削除する必要があります。

例えば、

```python
111
```

となるため、

出力:

```python
3
```

## 実行結果

```
5
3
3
```
