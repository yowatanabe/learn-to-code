## 問題文

### 最小の作業者数でデプロイ作業を処理する

あなたは複数のデプロイ作業の開始・終了時刻を管理しています。

各作業は `[start, end]` で表され、`start` 時刻に開始し、`end` 時刻に終了します。

1人の作業者は、同時に1つの作業しか担当できません。ただし、ある作業が `end` 時刻に終わり、別の作業が同じ時刻 `start` に始まる場合は、同じ作業者が続けて担当できます。

すべての作業を処理するために必要な**最小の作業者数**を返してください。

### 制約

* `1 <= len(tasks) <= 10^5`
* `0 <= start < end <= 10^9`

## 実行例

### 例1

```python
tasks = [[1, 4], [2, 5], [7, 9]]
```

`[1, 4]` と `[2, 5]` は重なっているため、2人必要です。
`[7, 9]` はどちらかの作業者が担当できます。

出力:

```python
2
```

### 例2

```python
tasks = [[1, 3], [3, 5], [5, 7]]
```

終了時刻と開始時刻が同じ場合は同じ作業者が使えます。

出力:

```python
1
```

### 例3

```python
tasks = [[1, 10], [2, 3], [4, 5], [6, 7]]
```

長い作業 `[1, 10]` と、短い作業群が並行するため、最大2人必要です。

出力:

```python
2
```

## 実行結果

```
2
1
2
```
