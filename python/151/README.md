# No.151

## 問題：部分文字列の最長回文

文字列 `s` が与えられたとき、**回文（前から読んでも後ろから読んでも同じ）となる最長の部分文字列**を返してください。

* 部分文字列とは、文字列の中から連続した文字の並びを取り出したものです。
* 答えが複数ある場合は、最初に出現したものを返してください。

### 制約

* `1 <= len(s) <= 1000`
* `s` は英小文字および英大文字からなる

### 🔸 実行例

#### 例 1:

```
入力: s = "babad"
出力: "bab"
※ "aba" も正解だが、"bab" の方が先に出現するため
```

#### 例 2:

```
入力: s = "cbbd"
出力: "bb"
```

#### 例 3:

```
入力: s = "a"
出力: "a"
```

#### 例 4:

```
入力: s = "ac"
出力: "a"
```

### 実行結果

```text
bab
bb
a
a
```
