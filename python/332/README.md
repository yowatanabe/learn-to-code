## 問題: K 個ずつノードを反転せよ（Reverse Nodes in k-Group）

単方向連結リストの先頭ノード `head` と整数 `k` が与えられます。

> 連結リストを **k 個ずつ**のグループに分け、各グループを反転して返してください。
> ただし、最後のグループのノード数が `k` 未満の場合は **そのまま**にしてください。

### 制約（目安）

* ノード数 `n` は `0 ≤ n ≤ 10^5`
* `1 ≤ k ≤ 10^5`
* `-10^4 ≤ Node.val ≤ 10^4`

時間計算量 **O(n)**、追加メモリ **O(1)** を目指してください（再帰はスタックを使うので注意）。

## 実行例

```python
# 例1
head = 1 -> 2 -> 3 -> 4 -> 5
k = 2
# (1,2) を反転 => 2->1, (3,4) を反転 => 4->3, 5 は残る
# 出力: 2 -> 1 -> 4 -> 3 -> 5

# 例2
head = 1 -> 2 -> 3 -> 4 -> 5
k = 3
# (1,2,3) を反転 => 3->2->1, (4,5) はそのまま
# 出力: 3 -> 2 -> 1 -> 4 -> 5

# 例3
head = (empty)
k = 3
# 出力: (empty)
```

## 実行結果

```
例1 入力: vals=[1, 2, 3, 4, 5], k=2
例1 出力: [2, 1, 4, 3, 5]
--------------------------------------------------
例2 入力: vals=[1, 2, 3, 4, 5], k=3
例2 出力: [3, 2, 1, 4, 5]
--------------------------------------------------
例3 入力: vals=[], k=3
例3 出力: []
--------------------------------------------------
```
