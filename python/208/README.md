# No.208

## 問題文

二分木の根 `root` が与えられます。

木のノード値をジグザグ順（1段目は左→右、2段目は右→左、3段目は左→右…と交互）にレベル順で走査し、各レベルを配列として返してください。

* ノード数は 0 以上
* ノードの値は任意の整数

## 実行例

```
入力:
    3
   / \
  9  20
    /  \
   15   7

出力: [[3], [20, 9], [15, 7]]

入力:
root = [1,2,3,4,null,null,5]
   1
  / \
 2   3
/     \
4      5

出力: [[1],[3,2],[4,5]]

入力:
root = []
出力: []
```

## 実行結果

```
[[3], [20, 9], [15, 7]]
[[1], [3, 2], [4, 5]]
[]
```
