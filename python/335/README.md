## 問題: 最小コストで迷路を抜けろ（Minimum Cost Path in Grid）

`m x n` のグリッド `grid` が与えられます。各セルには `0` 以上の整数コストが入っています。

左上 `(0,0)` から右下 `(m-1,n-1)` まで移動するとき、

> 通過したセルのコスト合計（開始セルと終了セルを含む）が最小になる値を返してください。

移動は **上下左右** の4方向に可能です（グリッド外には出られません）。

### 制約（目安）

* `1 ≤ m, n ≤ 10^3`
* `0 ≤ grid[r][c] ≤ 10^4`

期待解法: ダイクストラ法（重み付き最短路） `O(mn log(mn))`

## 実行例

```python
# 例1
grid = [
  [1,3,1],
  [1,5,1],
  [4,2,1]
]
# 最小コスト経路: 1→1→4→2→1 ではなく
# 1→3→1→1→1 などを考えると
# 最小は 1 + 3 + 1 + 1 + 1 = 7
# 出力: 7

# 例2
grid = [
  [0,2,2],
  [1,3,1],
  [4,2,0]
]
# 最小: 0 + 1 + 3 + 1 + 0 = 5
# 出力: 5

# 例3
grid = [
  [5]
]
# 出力: 5
```
