## 問題: 文字列中の全アナグラム開始位置（Find All Anagrams in a String）

英小文字からなる文字列 `s` と `p` が与えられます。

> `s` の中で、`p` のアナグラム（並べ替えで一致する文字列）となる部分文字列が開始する **インデックス** をすべて返してください。

* 部分文字列は **連続**している必要があります。
* 返すインデックスは昇順で返してください。

### 制約（目安）

* `1 ≤ len(s), len(p) ≤ 3 * 10^4`
* `s` と `p` は `'a'`〜`'z'` のみ

時間計算量 **O(len(s))** を目指してください。

## 実行例

```python
# 例1
s = "cbaebabacd"
p = "abc"
# "cba" (0..2) は "abc" のアナグラム
# "bac" (6..8) もアナグラム
# 出力: [0, 6]

# 例2
s = "abab"
p = "ab"
# "ab" (0..1), "ba" (1..2), "ab" (2..3)
# 出力: [0, 1, 2]

# 例3
s = "aaaaa"
p = "b"
# 一致なし
# 出力: []
```

## 実行結果

```
例1 入力: s='cbaebabacd', p='abc'
例1 出力: [0, 6]
----------------------------------------
例2 入力: s='abab', p='ab'
例2 出力: [0, 1, 2]
----------------------------------------
例3 入力: s='aaaaa', p='b'
例3 出力: []
----------------------------------------
```
