## 問題: 負の数を含むときの最短部分配列（Shortest Subarray with Sum at Least K）

整数配列 `nums`（**負の数を含んでもよい**）と整数 `k` が与えられます。

> 合計が **`k` 以上**となる **連続部分配列**のうち、
> 長さが最小のものを求めてください。
> 条件を満たす部分配列が存在しない場合は `-1` を返してください。

### 制約（目安）

* `1 ≤ len(nums) ≤ 10^5`
* `-10^5 ≤ nums[i] ≤ 10^5`
* `-10^9 ≤ k ≤ 10^9`

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
nums = [2, -1, 2]
k = 3
# 全体 [2,-1,2] の合計は 3 で条件を満たす。長さ 3 が最短
# 出力: 3

# 例2
nums = [1, 2]
k = 4
# どの部分配列でも合計 4 に届かない
# 出力: -1

# 例3
nums = [84, -37, 32, 40, 95]
k = 167
# [32,40,95] の合計は 167 で長さ 3 が最短
# 出力: 3
```
