## 問題: 監視対象の依存関係を逆転せよ（Reverse Nodes in k-Group）

単方向連結リストの先頭ノード `head` と整数 `k` が与えられます。

> 連結リストを **k 個ずつのグループ** に分け、各グループを **その場で反転**してください。
> ただし、最後のグループのノード数が `k` 未満の場合、その部分は **反転せず** そのまま残します。
> 反転後の先頭ノードを返してください。

**追加の配列を使わず**（ノードの付け替えで）解くことを想定します。

### 制約（目安）

* `1 ≤ リスト長 ≤ 10^5`
* `1 ≤ k ≤ 10^5`

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
# head = 1 -> 2 -> 3 -> 4 -> 5, k = 2
# 2個ずつ反転: (1,2) (3,4) (5)
# 結果: 2 -> 1 -> 4 -> 3 -> 5
# 出力: [2,1,4,3,5]

# 例2
# head = 1 -> 2 -> 3 -> 4 -> 5, k = 3
# (1,2,3) は反転、(4,5) はそのまま
# 結果: 3 -> 2 -> 1 -> 4 -> 5
# 出力: [3,2,1,4,5]

# 例3
# head = 1 -> 2, k = 3
# k未満なので反転なし
# 出力: [1,2]
```

## 実行結果

```
例1 入力: values=[1, 2, 3, 4, 5], k=2
例1 出力: [2, 1, 4, 3, 5] (期待値: [2, 1, 4, 3, 5])
------------------------------------------------------------
例2 入力: values=[1, 2, 3, 4, 5], k=3
例2 出力: [3, 2, 1, 4, 5] (期待値: [3, 2, 1, 4, 5])
------------------------------------------------------------
例3 入力: values=[1, 2], k=3
例3 出力: [1, 2] (期待値: [1, 2])
------------------------------------------------------------
```
