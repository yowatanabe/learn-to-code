# No.301

## 問題: 最長増加部分列の長さ（Longest Increasing Subsequence）

整数配列 `nums` が与えられます。

> 要素が **厳密に増加する（`nums[i1] < nums[i2] < ...`）ような部分列** のうち、
> **最も長いものの長さ** を求めてください。

* **部分列** とは、元の配列からいくつかの要素を削除して得られる列であり、要素の順序は変えません（連続している必要はありません）。
* 要素が「以上（<=）」ではなく「より大きい（<）」になるような列のみを増加部分列とみなします。

### 制約（目安）

* `1 ≤ len(nums) ≤ 2500`
* `-10^4 ≤ nums[i] ≤ 10^4`

時間計算量 O(n²) の DP 解法を想定してください。

（O(n log n) の解法もありますが、まずは O(n²) をきちんと書けることを目標にします）

## 実行例

```python
# 例1
nums = [10, 9, 2, 5, 3, 7, 101, 18]
# 最長増加部分列の一例:
# [2, 3, 7, 101] → 長さ 4
# 他に [2, 5, 7, 101] なども OK
# 出力: 4

# 例2
nums = [0, 1, 0, 3, 2, 3]
# 最長増加部分列の一例:
# [0, 1, 2, 3] → 長さ 4
# 出力: 4

# 例3
nums = [7, 7, 7, 7, 7]
# すべて同じ値なので、増加する部分列はどれも長さ1
# 出力: 1

# 例4
nums = [4, 10, 4, 3, 8, 9]
# 最長増加部分列の一例:
# [4, 8, 9] → 長さ 3
# 出力: 3
```

## 実行結果

```
例1の入力: [10, 9, 2, 5, 3, 7, 101, 18]
例1の結果: 4
例2の入力: [0, 1, 0, 3, 2, 3]
例2の結果: 4
例3の入力: [7, 7, 7, 7, 7]
例3の結果: 1
例4の入力: [4, 10, 4, 3, 8, 9]
例4の結果: 3
```
