# No.428

## 問題文

### ジョブ依存関係を満たす実行順序を求める

あなたはバッチ処理基盤を運用しており、複数のジョブを依存関係に従って実行したいと考えています。ジョブは `0` から `numJobs - 1` までの整数で表されます。

`prerequisites` は `[[a, b], ...]` の形式で与えられ、これは **「ジョブ `a` を実行するには、先にジョブ `b` を実行しておく必要がある」** ことを意味します。

**すべてのジョブを実行できる順序を1つ返してください。**そのような順序が存在しない場合は、空の配列 `[]` を返してください。

### 制約

* `1 <= numJobs <= 10^5`
* `0 <= len(prerequisites) <= 10^5`
* `prerequisites[i].length == 2`
* `0 <= a, b < numJobs`

## 実行例

### 例1

```python
numJobs = 4
prerequisites = [[1, 0], [2, 0], [3, 1], [3, 2]]
```

* ジョブ `1` を実行するには `0` が必要
* ジョブ `2` を実行するには `0` が必要
* ジョブ `3` を実行するには `1` と `2` が必要

実行順序の一例は

```python
[0, 1, 2, 3]
```

です。
`[0, 2, 1, 3]` でも正解です。

### 例2

```python
numJobs = 2
prerequisites = [[0, 1], [1, 0]]
```

ジョブ `0` は `1` に依存し、
ジョブ `1` は `0` に依存しています。

循環依存があるため、すべてのジョブを実行することはできません。

出力は

```python
[]
```

### 例3

```python
numJobs = 3
prerequisites = [[1, 0]]
```

ジョブ `1` の前に `0` を実行する必要があります。
ジョブ `2` は独立しています。

実行順序の一例は

```python
[0, 2, 1]
```

です。
`[2, 0, 1]` でも正解です。

## 実行結果

```
[0, 1, 2, 3]
[]
[0, 2, 1]
```
