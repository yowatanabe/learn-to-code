## 問題: 最小のサブ配列長（Minimum Size Subarray Sum）

整数配列 `nums` と整数 `target` が与えられます。

> `nums` の連続する部分配列の中で、その合計が `target` 以上になる部分配列の **最小の長さ** を返してください。
> 条件を満たす部分配列が存在しない場合は `0` を返してください。

### 制約

* `1 ≤ len(nums) ≤ 10^5`
* `1 ≤ nums[i] ≤ 10^4`
* `1 ≤ target ≤ 10^9`

時間計算量 **O(n)** を目指してください。

### 実行例

```python
# 例1
nums = [2,3,1,2,4,3]
target = 7
# 部分配列 [4,3] の長さは 2 で、合計は 7 以上
# 出力: 2

# 例2
nums = [1,4,4]
target = 4
# 部分配列 [4] の長さは 1 で、合計は 4
# 出力: 1

# 例3
nums = [1,1,1,1,1,1]
target = 11
# 合計が 11 以上になる部分配列は存在しない
# 出力: 0
```

## 実行結果

```

例1 出力: 2 (期待値: 2)

例2 出力: 1 (期待値: 1)
------------------------------------------------------------
例3 入力: target=11, nums=[1, 1, 1, 1, 1, 1]
例3 出力: 0 (期待値: 0)
------------------------------------------------------------
```
