## 問題文

### 最長の安定メトリクス区間

あなたは1分ごとのCPU使用率を監視しています。

整数配列 `metrics` が与えられ、`metrics[i]` は `i` 分目のCPU使用率を表します。

連続部分配列の中で、

```python
最大値 - 最小値 <= limit
```

を満たすもののうち、**長さが最大のもの**を返してください。

### 制約

* `1 <= len(metrics) <= 10^5`
* `0 <= metrics[i] <= 10^9`
* `0 <= limit <= 10^9`

## 実行例

### 例1

```python
metrics = [8, 2, 4, 7]
limit = 4
```

`[2, 4]` は最大値 `4`、最小値 `2` なので差は `2` です。
`[4, 7]` も差は `3` です。

最長の長さは `2` なので、出力は

```python
2
```

### 例2

```python
metrics = [10, 1, 2, 4, 7, 2]
limit = 5
```

`[2, 4, 7, 2]` は最大値 `7`、最小値 `2` なので差は `5` です。

出力は

```python
4
```

### 例3

```python
metrics = [4, 4, 4, 4]
limit = 0
```

すべて同じ値なので、配列全体が条件を満たします。

出力は

```python
4
```

## 実行結果

```
2
4
4
```
