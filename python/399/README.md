# No.399

## 問題：最長連番ログ区間

あなたは SRE として、ログに含まれるイベント番号を分析しています。

整数配列 `nums` が与えられます。

`nums` の中から、**値が 1 ずつ連続する整数列**を作れるとします。

このとき作ることができる **最長の連続列の長さ** を求めてください。

ここで、連続列とは例えば次のようなものです。

* `[100, 101, 102, 103]` → 長さ 4
* `[5]` → 長さ 1
* `[1, 2, 4]` は連続ではない

配列中の要素の順番は関係ありません。

* `1 <= len(nums) <= 2 * 10^5`
* `-10^9 <= nums[i] <= 10^9`

**求めるもの**：最長の連続列の長さ（整数）

## 実行例

例1:

* 入力: `nums = [100, 4, 200, 1, 3, 2]`
* 説明:

  * `1, 2, 3, 4` が連続している
  * 長さは `4`
* 出力: `4`

例2:

* 入力: `nums = [0, 3, 7, 2, 5, 8, 4, 6, 0, 1]`
* 説明:

  * `0, 1, 2, 3, 4, 5, 6, 7, 8` が連続している
  * 長さは `9`
* 出力: `9`

例3:

* 入力: `nums = [10, 30, 20]`
* 説明:

  * 連続する値は存在しないので、各要素単体の長さ `1` が最大
* 出力: `1`

## 実行結果

```
4
9
1
```
