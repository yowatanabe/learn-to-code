## 問題文

### 最長の有効なデプロイ期間

あなたは日ごとのデプロイ結果を監視しています。

文字列 `s` が与えられ、各文字は以下を表します。

* `"S"`: デプロイ成功
* `"F"`: デプロイ失敗

連続する期間の中で、**成功数が失敗数より多い期間**を「有効なデプロイ期間」とします。

**有効なデプロイ期間のうち、最長の長さ**を返してください。

### 制約

* `1 <= len(s) <= 10^5`
* `s[i]` は `"S"` または `"F"`

## 実行例

### 例1

```python
s = "SFFSS"
```

全体では `S` が3個、`F` が2個なので有効です。

出力:

```python
5
```

### 例2

```python
s = "FFSS"
```

`"FSS"` は `S` が2個、`F` が1個なので有効です。

出力:

```python
3
```

### 例3

```python
s = "FFFF"
```

成功数が失敗数より多い期間はありません。

出力:

```python
0
```

## 実行結果

```
5
3
0
```
