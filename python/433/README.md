# No.433

## 問題文

### 監視ログ中の連続する重複イベントを圧縮する

あなたは監視システムのイベントログを圧縮したいと考えています。ログは文字列の配列 `events` で与えられ、同じイベントが連続して現れることがあります。

連続する同じイベントは、

* 1回だけ現れる場合はそのまま `"event"`
* 2回以上連続する場合は `"event:回数"`

という形式で圧縮します。

圧縮後の配列を返してください。

### 制約

* `1 <= len(events) <= 10^5`
* `events[i]` は長さ1以上20以下の文字列

## 実行例

### 例1

```python
events = ["db", "db", "db", "api", "api", "cache"]
```

`"db"` は 3 回連続、`"api"` は 2 回連続、`"cache"` は 1 回です。
したがって出力は

```python
["db:3", "api:2", "cache"]
```

### 例2

```python
events = ["error", "warn", "warn", "error"]
```

* 最初の `"error"` は 1 回
* `"warn"` は 2 回連続
* 最後の `"error"` は前と連続していないので 1 回

したがって出力は

```python
["error", "warn:2", "error"]
```

### 例3

```python
events = ["auth"]
```

1要素だけなので、そのまま

```python
["auth"]
```

を返します。

## 実行結果

```
['db:3', 'api:2', 'cache']
['error', 'warn:2', 'error']
['auth']
```
