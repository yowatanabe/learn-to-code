# No.277

## 問題文

**最大連続部分列の和**

整数の配列 `nums` が与えられるとき、その中で**連続する部分列**の和の**最大値**を返してください。

* 部分列とは、連続した要素を含む部分である（必ずしも全体の配列を使う必要はない）。
* 配列の長さは `1 ≤ len(nums) ≤ 10^5` で、要素の値は `-10^4 ≤ nums[i] ≤ 10^4` です。

**ヒント:**

* 動的計画法を使うと、前の部分列の和を記録していくことで、現在位置までの最大和を求めることができます。
* 直前の部分列の最大和を利用して次の部分列の和を計算していく。

## 実行例

**例1**

```text
入力:
nums = [-2,1,-3,4,-1,2,1,-5,4]

出力:
6

説明:
最大の部分列は [4,-1,2,1] で、その和は 6。
```

**例2**

```text
入力:
nums = [1]

出力:
1

説明:
配列に1つしか要素がない場合、その要素が最大となります。
```

**例3**

```text
入力:
nums = [5,4,-1,7,8]

出力:
23

説明:
最大の部分列は [5,4,-1,7,8] で、その和は 23。
```

**例4**

```text
入力:
nums = [-1,-2,-3,-4]

出力:
-1

説明:
最も大きい部分列は [-1] で、その和は -1 です。
```

## 実行結果

```
6
1
23
-1
```
