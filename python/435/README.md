## 問題文

### 2つのサーバーからのイベントを交互に実行できる最長列

あなたは2台のサーバー `A` と `B` から届くイベント列を処理しています。それぞれのイベント列は文字列 `a` と `b` で表されます。

あなたは次のルールで新しい文字列を1つ作ります。

* `a` からも `b` からも、文字を**左から順に**何文字か選べる
* 選んだ文字の相対順序は元の文字列内で保つ必要がある
* 作る文字列では、**同じ文字が連続してはいけない**

このとき、作ることができる文字列の**最大長**を返してください。

### 制約

* `1 <= len(a), len(b) <= 1000`
* `a`, `b` は英小文字のみからなる

## 実行例

### 例1

```python
a = "ab"
b = "bc"
```

例えば、

* `a` から `"ab"`
* `b` から `"c"`

を使って

```python
"abcb"
```

を作れます。

同じ文字は連続していないので有効です。
これが最大長なので、出力は

```python
4
```

### 例2

```python
a = "aa"
b = "bb"
```

例えば

```python
"abab"
```

を作れます。

出力は

```python
4
```

### 例3

```python
a = "aaa"
b = "a"
```

使える文字はすべて `"a"` なので、連続を避けると1文字しか使えません。

出力は

```python
1
```

## 実行結果

```
4
4
1
```
