## 問題: ストリームの中央値（Find Median from Data Stream）

整数が 1 つずつ順番に到着するストリームがあります。

あなたは到着するたびにデータを追加し、いつでも中央値を答えられるようにしたい。

以下の2つの操作を実装してください：

* `addNum(x)`: 整数 `x` をデータ構造に追加する
* `findMedian()`: 現在のデータの中央値を返す

  * 要素数が奇数なら中央の値
  * 要素数が偶数なら中央2つの平均

### 実装要件

* `addNum` は **O(log n)**
* `findMedian` は **O(1)** を目指してください

## 実行例

```python
# 例1
ops = ["add", "add", "find", "add", "find"]
vals = [1, 2, None, 3, None]

# 操作:
# add(1) -> データ [1]
# add(2) -> データ [1,2]
# find() -> (1+2)/2 = 1.5
# add(3) -> データ [1,2,3]
# find() -> 2

# 出力:
# 1.5
# 2
```

## 実行結果

```
add(1)
add(2)
median = 1.5
add(3)
median = 2.0
```
