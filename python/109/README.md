# No.109

### 問題: リスト内の重複を削除する

整数のリスト `nums` が与えられます。
`nums` から重複を削除し、要素の順序を保持したまま返す関数 `remove_duplicates` を実装してください。

### 実行例

#### 例 1:

```
入力: nums = [1, 2, 2, 3, 1]
出力: [1, 2, 3]
```

#### 例 2:

```
入力: nums = [4, 5, 4, 4, 6]
出力: [4, 5, 6]
```

#### 例 3:

```
入力: nums = []
出力: []
```

### 実行結果

```text
[1, 2, 3]
[4, 5, 6]
[]
```
