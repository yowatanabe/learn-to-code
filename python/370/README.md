## 問題: 監視メトリクスの“ピーク同時接続”を求めよ（Car Fleet）

目的地 `target`（整数）に向かって走る `n` 台の車があります。

車 `i` は初期位置 `position[i]` にいて、一定速度 `speed[i]` で進みます。すべての車は同じ方向に進み、追い越しはできません。

ある車が前の車に追いつくと、その後は前の車と同じ速度で走り続け、**1つの“車群（fleet）”**になります。

> 目的地に到達する車群（fleet）の数を返してください。

### 制約（目安）

* `1 ≤ n ≤ 10^5`
* `0 ≤ position[i] < target`
* `1 ≤ speed[i] ≤ 10^6`

時間計算量 **O(n log n)** を目指してください。

## 実行例

```python
# 例1
target = 12
position = [10, 8, 0, 5, 3]
speed    = [ 2, 4, 1, 1, 3]
# 到達時間:
# (10,2)->1, (8,4)->1, (5,1)->7, (3,3)->3, (0,1)->12
# 目的地へ到達する fleet は 3
# 出力: 3

# 例2
target = 10
position = [3]
speed    = [3]
# 出力: 1

# 例3
target = 100
position = [0, 2, 4]
speed    = [4, 2, 1]
# 追い越せないので全て別 fleet
# 出力: 3
```
