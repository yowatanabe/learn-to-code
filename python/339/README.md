## 問題: ログの冗長区間を1回だけ削って最大化せよ（Maximum Subarray Sum with One Deletion）

整数配列 `nums` が与えられます（負の数を含む）。

あなたは **高々1回**、配列から **要素を1つだけ削除**できます（削除しなくてもよい）。

> その後に得られる **連続部分配列**の合計の最大値を返してください。
> ただし、部分配列は **空であってはならない**（少なくとも1要素を含む）とします。

### 制約（目安）

* `1 ≤ len(nums) ≤ 10^5`
* `-10^4 ≤ nums[i] ≤ 10^4`

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
nums = [1, -2, 0, 3]
# -2 を削除すると [1,0,3] の合計 4 が最大
# 出力: 4

# 例2
nums = [1, -2, -2, 3]
# -2 を1つ削除して [1,-2,3] => 2 が最大
# 出力: 2

# 例3
nums = [-1, -1, -1, -1]
# 削除しても空はNGなので、最大は -1
# 出力: -1
```

## 実行結果

```

例1 出力: 4 (期待値: 4)
--------------------------------------------------
例2 入力: nums=[1, -2, -2, 3]
例2 出力: 3 (期待値: 2)
--------------------------------------------------
例3 入力: nums=[-1, -1, -1, -1]
例3 出力: -1 (期待値: -1)
--------------------------------------------------
```
