# No.360

## 問題: K個のグループに分けて“相性”を最小化（Divide Array Into K Groups With Minimum Incompatibility）

整数配列 `nums` と整数 `k` が与えられます。`len(nums)` は `k` で割り切れます。

`nums` を **ちょうど k 個のグループ**に分け、各グループのサイズはすべて `len(nums)/k` にしてください。

* 各要素は **ちょうど1つのグループ**に属します
* **同じグループ内に同じ値が2回以上現れてはいけません**（重複禁止）

グループの **相性（incompatibility）** を`max(group) - min(group)` と定義します。

> すべてのグループの相性の合計を最小化した値を返してください。
> 条件を満たす分割が不可能なら `-1` を返してください。

### 制約（目安）

* `1 ≤ len(nums) ≤ 16`（bit DP を想定）
* `1 ≤ k ≤ len(nums)`
* `len(nums) % k == 0`

## 実行例

```python
# 例1
nums = [1,2,1,4]
k = 2
# グループサイズは2
# 分け方の例: [1,2] と [1,4] （各グループ重複なし）
# 相性: (2-1) + (4-1) = 1 + 3 = 4
# 出力: 4

# 例2
nums = [6,3,8,1,3,1,2,2]
k = 4
# グループサイズは2
# 最小合計相性は 6
# 出力: 6

# 例3
nums = [1,1,1,1]
k = 2
# どのグループも重複禁止を満たせない
# 出力: -1
```

## 実行結果

```
例1 入力: nums=[1, 2, 1, 4], k=2
例1 出力: 4 (期待値: 4)
------------------------------------------------------------
例2 入力: nums=[6, 3, 8, 1, 3, 1, 2, 2], k=4
例2 出力: 6 (期待値: 6)
------------------------------------------------------------
例3 入力: nums=[1, 1, 1, 1], k=2
例3 出力: -1 (期待値: -1)
------------------------------------------------------------
```
