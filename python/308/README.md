## 問題: 次に気温が上がるまでの日数（Daily Temperatures）

整数配列 `temperatures` が与えられます。`temperatures[i]` は `i` 日目の気温です。

各 `i` について、**`i` 日目より後で気温がより高くなる最初の日**までに何日待つ必要があるかを求め、配列 `answer` を返してください。

* もし将来に **より高い気温の日が存在しない**場合、`answer[i] = 0` とします。

### 制約（目安）

* `1 ≤ len(temperatures) ≤ 10^5`
* `30 ≤ temperatures[i] ≤ 100`

時間計算量 **O(n)** を目指してください（全探索 O(n^2) は TLE 想定）。

## 実行例

```python
# 例1
temperatures = [73,74,75,71,69,72,76,73]
# 出力: [1,1,4,2,1,1,0,0]

# 例2
temperatures = [30,40,50,60]
# 出力: [1,1,1,0]

# 例3
temperatures = [30,60,90]
# 出力: [1,1,0]
```

## 実行結果

```

例1 入力: [73, 74, 75, 71, 69, 72, 76, 73]

----------------------------------------

例2 出力: [1, 1, 1, 0]
----------------------------------------
例3 入力: [30, 60, 90]
例3 出力: [1, 1, 0]
```
