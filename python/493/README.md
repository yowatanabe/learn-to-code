## 問題文

### レート制限を満たす最長リクエスト区間

あなたはAPI Gatewayのリクエストログを分析しています。

整数配列 `requests` が与えられ、`requests[i]` は `i` 秒目のリクエスト数を表します。

整数 `limit` が与えられたとき、**平均リクエスト数が `limit` 以下になる連続部分配列のうち、長さが最大のもの**を返してください。

平均は以下で定義します。

```python
区間の合計 / 区間の長さ
```

## 実行例

### 例1

```python
requests = [5, 1, 3, 2, 6]
limit = 3
```

`[1, 3, 2]` の平均は `2` です。
さらに `[1, 3, 2, 6]` の平均は `3` です。

長さ `4` が最大なので、出力は

```python
4
```

### 例2

```python
requests = [10, 10, 10]
limit = 5
```

どの区間も平均が `5` を超えるため、出力は

```python
0
```

### 例3

```python
requests = [1, 2, 3, 4]
limit = 3
```

`[1, 2, 3, 4]` の平均は `2.5` なので、出力は

```python
4
```

## 実行結果

```
4
0
4
```
