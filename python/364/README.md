## 問題: 重要イベントを含む最小区間（Minimum Window Substring）

文字列 `s` と `t` が与えられます。

> `t` に含まれる文字（重複回数も含む）をすべて含む `s` の連続部分文字列のうち、長さが最小のものを返してください。
> 条件を満たす部分文字列が存在しない場合は空文字列 `""` を返してください。

### 例

- `s = "ADOBECODEBANC"`, `t = "ABC"` のとき、答えは `"BANC"` です。

### 制約（目安）

- `1 ≤ len(s), len(t) ≤ 10^5`
- `s` と `t` は英大小文字（ASCII）からなるとしてよい

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
s = "ADOBECODEBANC"
t = "ABC"
# "BANC" が最短で A,B,C をすべて含む
# 出力: "BANC"

# 例2
s = "a"
t = "a"
# 出力: "a"

# 例3
s = "a"
t = "aa"
# 必要な a が2個ない
# 出力: ""

# 例4
s = "aaabdabcefaecbef"
t = "abc"
# 最短の1つは "abc"
# 出力: "abc"
````

## 実行結果

```
例1 入力: s='ADOBECODEBANC', t='ABC'
例1 出力: 'BANC' (期待値: 'BANC')
------------------------------------------------------------
例2 入力: s='a', t='a'
例2 出力: 'a' (期待値: 'a')
------------------------------------------------------------
例3 入力: s='a', t='aa'
例3 出力: '' (期待値: '')
------------------------------------------------------------
例4 入力: s='aaabdabcefaecbef', t='abc'
例4 出力: 'abc' (期待値: 'abc')
------------------------------------------------------------
```
