## 問題: 区間をまとめて削減せよ（Merge Intervals）

区間の配列 `intervals` が与えられます。

各区間は `intervals[i] = [start, end]`（`start <= end`）で表されます。

> 重なり合う区間をすべてマージ（統合）し、
> 重なりのない区間の配列を返してください。

* 返す区間は `start` の昇順で並んでいればよい
* 入力は順不同でもよい
* `end` が隣接する（例: `[1,3]` と `[4,5]`）場合は **重なりではない** とみなします
  （つまり `next_start <= current_end` ではなく、`next_start <= current_end` を重なり判定にしたい場合はそのように問題を変えてください。ここでは **重なり = next_start <= current_end** とします）

※この問題では **重なり判定は `next_start <= current_end`（端点が接していてもマージ）** とします。

例: `[1,3]` と `[3,4]` はマージして `[1,4]`。

### 制約（目安）

* `1 ≤ len(intervals) ≤ 10^4`
* `-10^4 ≤ start ≤ end ≤ 10^4`

時間計算量は **O(n log n)**（ソート）を目指してください。

## 実行例

```python
# 例1
intervals = [[1,3],[2,6],[8,10],[15,18]]
# [1,3] と [2,6] が重なるので [1,6]
# 残りは重ならない
# 出力: [[1,6],[8,10],[15,18]]

# 例2
intervals = [[1,4],[4,5]]
# 端点が接しているのでマージして [1,5]
# 出力: [[1,5]]

# 例3
intervals = [[5,5],[1,2],[2,3]]
# [1,2] と [2,3] は接しているので [1,3]
# [5,5] は単独
# 出力: [[1,3],[5,5]]
```

## 実行結果

```
例1 入力: [[1, 3], [2, 6], [8, 10], [15, 18]]
例1 出力: [[1, 6], [8, 10], [15, 18]]
----------------------------------------
例2 入力: [[1, 4], [4, 5]]
例2 出力: [[1, 5]]
----------------------------------------
例3 入力: [[5, 5], [1, 2], [2, 3]]
例3 出力: [[1, 3], [5, 5]]
----------------------------------------
```
