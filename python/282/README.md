## 問題: 異なる要素が高々 K 個の最長部分配列

整数配列 `nums` と整数 `k` (1 ≤ k) が与えられたとき、
「**異なる値の種類が高々 k 個だけ含まれる連続した部分配列**」のうち、
**最も長いものの長さ**を求めてください。

* 部分配列とは、配列の連続した要素の列のことです。
* `nums` が空配列の場合は、答えは 0 とします。

### 制約 (目安)

* `0 ≤ len(nums) ≤ 10^5`
* `-10^9 ≤ nums[i] ≤ 10^9`
* `1 ≤ k ≤ 10^5`

## 実行例

```python
# 例1
nums = [1, 2, 1, 2, 3]
k = 2
# 異なる値が高々2個の最長部分配列は [1,2,1,2] なので長さは 4
# 出力: 4

# 例2
nums = [4, 4, 4, 4]
k = 1
# すべて同じ値なので、配列全体が条件を満たす
# 出力: 4

# 例3
nums = [1, 2, 3, 4, 5]
k = 3
# 条件を満たす最長部分配列の一例は [1,2,3] や [2,3,4] などで長さは 3
# 出力: 3
```

## 実行結果

```
4
4
3
```
