## 問題: 最小の会議室数（Meeting Rooms II）

会議の開始・終了時刻を表す配列 `intervals` が与えられます。

`intervals[i] = [start, end]` は **start 時刻に開始し end 時刻に終了**する会議です（`start < end`）。

> すべての会議を重ならないように開催するために必要な **最小の会議室数** を返してください。

※ `end` と同じ時刻に次の会議が `start` する場合は **重なっていない** とみなします
（例: `[0,10]` と `[10,20]` は同じ部屋でOK）。

### 制約（目安）

* `1 ≤ len(intervals) ≤ 10^5`
* `0 ≤ start < end ≤ 10^9`

時間計算量 **O(n log n)** を目指してください。

## 実行例

```python
# 例1
intervals = [[0,30],[5,10],[15,20]]
# [0,30] と [5,10] が重なるので同時に2部屋必要
# 出力: 2

# 例2
intervals = [[7,10],[2,4]]
# 重ならないので1部屋でOK
# 出力: 1

# 例3
intervals = [[1,5],[5,10],[10,15]]
# 端が接するだけなので1部屋でOK
# 出力: 1
```
