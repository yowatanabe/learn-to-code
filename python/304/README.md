## 問題: K 個ずつ反転せよ（Reverse Nodes in k-Group）

単方向連結リストの先頭ノード `head` と整数 `k` が与えられます。

連結リストを **k 個ずつのグループ**に分け、各グループを **その場で反転**してください。

* 末尾に `k` 個未満のノードが残った場合、その部分は **反転しない**。
* ノードの値を入れ替えるのではなく、**ノード同士の next ポインタを付け替える**ことで反転すること。
* 反転後のリストの先頭を返してください。

### 制約（目安）

* `1 ≤ ノード数 ≤ 10^4`
* `1 ≤ Node.val ≤ 10^4`
* `1 ≤ k ≤ 10^4`

## 実行例

```python
# 例1
# 連結リスト: 1 -> 2 -> 3 -> 4 -> 5
# k = 2
# 2個ずつ反転:
# (1,2) -> 2 -> 1
# (3,4) -> 4 -> 3
# 5 は 2個未満なのでそのまま
# 出力: 2 -> 1 -> 4 -> 3 -> 5

# 例2
# 連結リスト: 1 -> 2 -> 3 -> 4 -> 5
# k = 3
# (1,2,3) -> 3 -> 2 -> 1
# (4,5) は 3個未満なのでそのまま
# 出力: 3 -> 2 -> 1 -> 4 -> 5
```

## 実行結果

```
例1 入力: [1, 2, 3, 4, 5] k = 2
例1 出力: [2, 1, 4, 3, 5]
例2 入力: [1, 2, 3, 4, 5] k = 3
例2 出力: [3, 2, 1, 4, 5]
```
