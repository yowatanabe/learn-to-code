# No.408

## 問題：最小会議室数（同時実行ジョブ数）

あなたは SRE として、複数のメンテナンスジョブをスケジューリングしています。

各ジョブは開始時刻と終了時刻を持ち、実行中は 1 つのワーカーを占有します。

整数の二次元配列 `intervals` が与えられます。

`intervals[i] = [start_i, end_i]` は、ジョブ `i` が **時刻 `start_i` 以上 `end_i` 未満** の間実行されることを表します。

同じ時刻に複数のジョブが重なって実行される場合、それぞれ別のワーカーが必要です。

すべてのジョブを実行するために必要な **最小ワーカー数** を返してください。

* `1 <= len(intervals) <= 2 * 10^5`
* `0 <= start_i < end_i <= 10^9`

**求めるもの**：必要な最小ワーカー数（整数）

## 実行例

例1:

* 入力: `intervals = [[0, 30], [5, 10], [15, 20]]`
* 説明:

  * `[0,30)` と `[5,10)` は重なるので 2 台必要
  * `[15,20)` の開始時点では `[5,10)` は終了済みなので再利用できる
* 出力: `2`

例2:

* 入力: `intervals = [[7, 10], [2, 4]]`
* 説明:

  * ジョブは重ならないので 1 台で足りる
* 出力: `1`

例3:

* 入力: `intervals = [[1, 5], [2, 6], [4, 8], [9, 10]]`
* 説明:

  * 時刻 `4` 以上 `5` 未満の間は 3 ジョブが同時に実行される
* 出力: `3`

## 実行結果

```
2
1
3
```
