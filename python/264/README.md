# No.264

## 問題文

**最小サーバ台数（同時実行数）**

ジョブの実行時間が `intervals` として与えられる。

`intervals[i] = [s, e]` はジョブ `i` が時刻 `s` に開始し、時刻 `e` に終了することを表す。

**区間は半開区間 `[s, e)`** とし、**終了時刻 `e` と同時刻に開始するジョブは重ならない** とみなす。

全ジョブを落とさず実行するために同時に必要な**サーバ（ワーカー）台数の最小値**を返せ。

* `0 ≤ len(intervals) ≤ 2 * 10^5`
* `0 ≤ s ≤ e ≤ 10^9`
* 期待計算量: **O(n log n)**、追加メモリ: **O(1)**（ソート除く）

## 実行例

**例1**

```
入力:
intervals = [[0, 30], [5, 10], [15, 20]]
出力:
2
説明:
[0,30) と [5,10) が重なる、[15,20) は [5,10) 終了後なので最大同時数は 2。
```

**例2**

```
入力:
intervals = [[7,10], [2,4]]
出力:
1
説明:
重ならないので 1 台で足りる。
```

**例3**

```
入力:
intervals = [[1,3], [3,5], [5,8]]
出力:
1
説明:
半開区間のため端点は重ならない。
```

**例4**

```
入力:
intervals = []
出力:
0
```

**例5**

```
入力:
intervals = [[1,1000000000], [2,3], [3,4], [4,5]]
出力:
2
説明:
巨大区間 [1,1e9) と各短区間の一部が重なるので最大同時数は 2。
```

## 実行結果

```
2
1
1
0
2
```
