# No.105

### 問題: 2つのリストが同じ要素を持つか判定する

2つの整数リスト `list1` と `list2` が与えられます。
順序や重複を無視し、両者が同じ要素を持つ場合 `True` を、そうでない場合 `False` を返す関数 `have_same_elements` を実装してください。

### 実行例

#### 例 1:

```
入力: list1 = [1, 2, 2, 3], list2 = [3, 2, 1]
出力: True
```

#### 例 2:

```
入力: list1 = [1, 2, 3], list2 = [1, 2, 4]
出力: False
```

#### 例 3:

```
入力: list1 = [], list2 = []
出力: True
```

### 実行結果

```text
True
False
True
```
