# No.134

## 問題：2つのリストの共通要素

2つの整数リスト `list1` と `list2` が与えられます。
両方のリストに共通して含まれている**重複しない整数のリスト**を昇順で返してください。

### 実行例

#### 例 1:

```
入力:
list1 = [1, 2, 2, 3]
list2 = [2, 3, 4]
出力:
[2, 3]
```

#### 例 2:

```
入力:
list1 = [4, 5, 6]
list2 = [7, 8, 9]
出力:
[]
```

#### 例 3:

```
入力:
list1 = []
list2 = [1, 2]
出力:
[]
```

### 実行結果

```text
[2, 3]
[]
[]
```
