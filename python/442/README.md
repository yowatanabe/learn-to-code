## 問題文

### K個のサーバーを再起動する最小コスト

あなたは `n` 台のサーバーを順番にメンテナンスしています。各サーバー `i` の再起動コストは `costs[i]` です。

整数 `k` が与えられたとき、**連続する `k` 台のサーバーを再起動する場合の最小合計コスト**を返してください。

### 制約

* `1 <= k <= len(costs) <= 10^5`
* `1 <= costs[i] <= 10^4`

## 実行例

### 例1

```python
costs = [5, 2, 4, 1, 3]
k = 2
```

連続する2台の合計は、

```python
[5, 2] = 7
[2, 4] = 6
[4, 1] = 5
[1, 3] = 4
```

最小は `4` なので、出力は

```python
4
```

### 例2

```python
costs = [10, 20, 30]
k = 3
```

全サーバーを選ぶしかないため、出力は

```python
60
```

### 例3

```python
costs = [3, 1, 2, 8, 1, 1]
k = 3
```

最小は `[1, 2, 8]` ではなく、`[8, 1, 1]` でもなく、
`[3, 1, 2]` の合計 `6` です。

出力は

```python
6
```

## 実行結果

```
4
60
6
```
