## 問題文

### 障害対応チケットの最小ジャンプ数

あなたは障害対応チケットを優先度順に確認しています。

整数配列 `tickets` が与えられ、`tickets[i]` は `i` 番目のチケットから最大で何個先のチケットまで進めるかを表します。

最初は index `0` にいます。

最後のチケットに到達するために必要な**最小ジャンプ回数**を返してください。

最後のチケットに到達できない場合は `-1` を返してください。

### 制約

* `1 <= len(tickets) <= 10^5`
* `0 <= tickets[i] <= 10^5`

## 実行例

### 例1

```python
tickets = [2, 3, 1, 1, 4]
```

`0 -> 1 -> 4` と進めます。

出力:

```python
2
```

### 例2

```python
tickets = [3, 2, 1, 0, 4]
```

index `3` から先へ進めないため、最後に到達できません。

出力:

```python
-1
```

### 例3

```python
tickets = [1, 1, 1, 1]
```

1つずつ進む必要があります。

出力:

```python
3
```

## 実行結果

```
2
-1
3
```
