# No.456

## 問題文

### 障害ログから最長のバランス区間を探す

あなたは障害ログを時系列で確認しています。

文字列配列 `logs` が与えられ、各要素は以下のどちらかです。

* `"error"`: 障害ログ
* `"ok"`: 正常ログ

**`"error"` と `"ok"` の数が同じになる連続部分配列のうち、長さが最大のもの**を求めてください。

その最大長を返してください。

### 制約

* `1 <= len(logs) <= 10^5`
* `logs[i]` は `"error"` または `"ok"`

## 実行例

### 例1

```python
logs = ["error", "ok", "ok", "error", "error", "ok"]
```

配列全体で `"error"` が3個、`"ok"` が3個なので、出力は

```python
6
```

### 例2

```python
logs = ["error", "error", "ok"]
```

`["error", "ok"]` の長さ `2` が最大です。

出力:

```python
2
```

### 例3

```python
logs = ["ok", "ok", "ok"]
```

`"error"` と `"ok"` の数が同じになる区間はありません。

出力:

```python
0
```

## 実行結果

```
6
2
0
```
