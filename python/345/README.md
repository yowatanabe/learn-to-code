## 問題: 0/1 マトリクスの最近傍距離（01 Matrix）

`m x n` の 2次元配列 `mat` が与えられます。`mat[i][j]` は `0` または `1` です。

> 各セル `(i, j)` について、**最も近い 0 までのマンハッタン距離**を求め、同じサイズの2次元配列 `dist` を返してください。

マンハッタン距離は上下左右の移動回数です（斜め移動は不可）。

### 制約（目安）

* `1 ≤ m, n ≤ 10^3`
* `mat[i][j] ∈ {0,1}`

時間計算量 **O(mn)** を目指してください。

## 実行例

```python
# 例1
mat = [
  [0,0,0],
  [0,1,0],
  [1,1,1]
]
# 出力:
# [
#   [0,0,0],
#   [0,1,0],
#   [1,2,1]
# ]

# 例2
mat = [
  [0,1,1],
  [1,1,1],
  [1,1,0]
]
# 出力:
# [
#   [0,1,2],
#   [1,2,1],
#   [2,1,0]
# ]
```
