# No.385

## 問題文（LeetCode風・日本語）

### 問題：メトリクスの「ゼロ区間」検知（合計が0になる最長区間）

あるメトリクスの差分（増減）を表す整数配列 `delta` が与えられます。

`delta[l..r]` の合計が `0` になるとき、その区間は「増減が相殺された＝実質変化なし」とみなせます。

`delta` の連続部分配列のうち、**合計が 0 になるものの最大長**を求めてください。

存在しない場合は `0` を返してください。

* `1 <= len(delta) <= 2 * 10^5`
* `-10^9 <= delta[i] <= 10^9`

**求めるもの**：合計0となる最長区間の長さ（整数）

## 実行例

例1:

* 入力: `delta = [1, -1, 2, -2, 3]`
* 説明: `[1, -1, 2, -2]` の合計は0、長さ4が最大
* 出力: `4`

例2:

* 入力: `delta = [1, 2, 3]`
* 説明: 合計0の区間は存在しない
* 出力: `0`

例3:

* 入力: `delta = [2, -2, 2, -2]`
* 説明: 全体の合計が0なので長さ4
* 出力: `4`

## 実行結果

4
0
4
