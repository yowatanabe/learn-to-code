## 問題: ログの“欠損区間”を最短で埋めよ（Find Minimum in Rotated Sorted Array II）

昇順にソートされた配列 `nums` がありましたが、ある地点で回転（rotate）されました。さらに **重複要素** を含みます。

例:

* ソート済み: `[0,1,2,4,4,5,6,7]`
* 回転後の例: `[4,5,6,7,0,1,2,4]`

整数配列 `nums` が与えられたとき、

> `nums` の最小値を返してください。

### 制約（目安）

* `1 ≤ len(nums) ≤ 10^5`
* `-10^9 ≤ nums[i] ≤ 10^9`
* `nums` はある昇順配列を回転させたもの（重複あり）

期待解法: 二分探索（重複があるので最悪 O(n) になりうるが、平均は良い）

## 実行例

```python
# 例1
nums = [2,2,2,0,1]
# 出力: 0

# 例2
nums = [1,3,5]
# 出力: 1

# 例3
nums = [10,1,10,10,10]
# 出力: 1
```
