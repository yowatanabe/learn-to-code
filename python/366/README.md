## 問題: リトライ回数を最小化して到達せよ（Minimum Jumps to Reach End）

整数配列 `nums` が与えられます。`nums[i]` は位置 `i` から最大でジャンプできる距離を表します。

あなたは位置 `0` から開始し、最後のインデックス `n-1` に到達したいです。

> 最後のインデックスに到達するために必要な **最小ジャンプ回数** を返してください。
> なお、必ず到達可能であるとします。

### 制約（目安）

- `1 ≤ len(nums) ≤ 10^5`
- `0 ≤ nums[i] ≤ 10^5`
- 必ずゴールに到達できる

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
nums = [2,3,1,1,4]
# 0 -> 1 -> 4 の2回
# 出力: 2

# 例2
nums = [2,3,0,1,4]
# 0 -> 1 -> 4 の2回
# 出力: 2

# 例3
nums = [1,1,1,1]
# 1つずつ進むので3回
# 出力: 3
```

## 実行結果

```

例1 出力: 2 (期待値: 2)
------------------------------------------------------------

例2 出力: 2 (期待値: 2)
------------------------------------------------------------

例3 入力: nums=[1, 1, 1, 1]
例3 出力: 3 (期待値: 3)
------------------------------------------------------------
```
