## 問題文

### 監視メトリクスの連続増加区間を数える

あなたは時系列の監視メトリクスを分析しています。

整数配列 `metrics` が与えられ、`metrics[i]` は `i` 分目のメトリクス値を表します。

**連続部分配列のうち、値が厳密に増加しているものの個数**を返してください。

長さ1の部分配列も、厳密に増加しているとみなします。

### 制約

* `1 <= len(metrics) <= 10^5`
* `0 <= metrics[i] <= 10^9`

## 実行例

### 例1

```python
metrics = [1, 2, 3]
```

条件を満たす連続部分配列は、

```python
[1], [2], [3], [1, 2], [2, 3], [1, 2, 3]
```

合計 `6` 個です。

出力:

```python
6
```

### 例2

```python
metrics = [3, 2, 1]
```

長さ1の部分配列だけが条件を満たします。

出力:

```python
3
```

### 例3

```python
metrics = [1, 3, 2, 4]
```

条件を満たす連続部分配列は、

```python
[1], [3], [2], [4], [1, 3], [2, 4]
```

合計 `6` 個です。

出力:

```python
6
```

## 実行結果

```
6
3
6
```
