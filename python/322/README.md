## 問題: 0/1 行列の最近接距離（01 Matrix）

`m x n` の 0/1 行列 `mat` が与えられます。

> 各セル `(r, c)` について、
> 値が **0 のセルまでの最短距離** を求めた行列 `dist` を返してください。

距離は上下左右に 1 マス移動するコストを 1 とする **マンハッタン距離**（4近傍）とします。

### 制約（目安）

* `1 ≤ m, n ≤ 10^3`
* `mat[r][c]` は 0 または 1
* 少なくとも 1 つは 0 が存在する

時間計算量 **O(mn)** を目指してください。

## 実行例

```python
# 例1
mat = [
  [0,0,0],
  [0,1,0],
  [1,1,1]
]
# 出力:
# [
#  [0,0,0],
#  [0,1,0],
#  [1,2,1]
# ]

# 例2
mat = [
  [0,1,1],
  [1,1,1],
  [1,1,0]
]
# 出力:
# [
#  [0,1,2],
#  [1,2,1],
#  [2,1,0]
# ]
```

## 実行結果

```
例1 入力:
[0, 0, 0]
[0, 1, 0]
[1, 1, 1]

[0, 0, 0]
[0, 1, 0]
[1, 2, 1]
----------------------------------------
例2 入力:
[0, 1, 1]
[1, 1, 1]
[1, 1, 0]
例2 出力:
[0, 1, 2]
[1, 2, 1]
[2, 1, 0]
----------------------------------------
```
