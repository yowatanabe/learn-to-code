# No.150

## 問題：和が0になる3つの数の組み合わせ

整数の配列 `nums` が与えられます。配列の中から、**和が 0 になるような異なる3つの数の組み合わせ**をすべて求めてください。

* 各組み合わせは昇順で返してください（例: `[-1, 0, 1]`）。
* 結果のリストの中に、同じ組み合わせが複数含まれないようにしてください。
* 結果のリスト自体の順序は問いません。

### 制約

* `3 <= len(nums) <= 3000`
* `-10^5 <= nums[i] <= 10^5`

### 実行例

#### 例 1:

```
入力: nums = [-1, 0, 1, 2, -1, -4]
出力: [[-1, -1, 2], [-1, 0, 1]]
```

#### 例 2:

```
入力: nums = [0, 0, 0]
出力: [[0, 0, 0]]
```

#### 例 3:

```
入力: nums = [0, 1, 1]
出力: []
```

### 実行結果

```text
[[-1, -1, 2], [-1, 0, 1]]
[[0, 0, 0]]
[]
```
