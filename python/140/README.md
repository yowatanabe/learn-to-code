# No.140

## 問題：2つのリストに共通する要素を返す

2つの整数のリスト `list1` と `list2` が与えられます。
両方のリストに含まれている**共通の整数**を**重複なしで**リストとして返してください。
結果の順序は問いません。

### 実行例

#### 例 1:

```
入力: list1 = [1, 2, 2, 3], list2 = [2, 3, 4]
出力: [2, 3]
```

#### 例 2:

```
入力: list1 = [5, 6, 7], list2 = [7, 8, 9]
出力: [7]
```

#### 例 3:

```
入力: list1 = [1, 2, 3], list2 = [4, 5, 6]
出力: []
```

### 実行結果

```text
[2, 3]
[7]
[]
```
