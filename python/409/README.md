# No.409

## 問題：エンコードされたログ文字列の展開

あなたは SRE として、圧縮されたログメッセージを復元したいです。

文字列 `s` は次のルールでエンコードされています。

* `k[encoded_string]` は、`encoded_string` を **ちょうど `k` 回繰り返す**ことを表す
* `k` は正の整数
* 入れ子になった表現も存在する
* 元の文字列には数字は含まれず、数字は繰り返し回数としてのみ使われる

エンコード文字列 `s` をデコードした結果を返してください。

* `1 <= len(s) <= 2 * 10^5`
* `s` は英小文字、数字、`[`、`]` から構成される
* 入力は常に有効であるとする

**求めるもの**：デコード後の文字列

## 実行例

例1:

* 入力: `s = "3[a]2[bc]"`
* 説明:

  * `"a"` を3回繰り返して `"aaa"`
  * `"bc"` を2回繰り返して `"bcbc"`
  * 結果は `"aaabcbc"`
* 出力: `"aaabcbc"`

例2:

* 入力: `s = "3[a2[c]]"`
* 説明:

  * 内側の `2[c]` は `"cc"`
  * `a2[c]` は `"acc"`
  * それを3回繰り返して `"accaccacc"`
* 出力: `"accaccacc"`

例3:

* 入力: `s = "2[abc]3[cd]ef"`
* 説明:

  * `2[abc]` は `"abcabc"`
  * `3[cd]` は `"cdcdcd"`
  * 最後に `"ef"` をつなげる
* 出力: `"abcabccdcdcdef"`

## 実行結果

```
aaabcbc
accaccacc
abcabccdcdcdef
```
