## 問題文

### 最短で全データセンターへ障害通知を届ける

あなたは複数のデータセンター間で障害通知を伝搬させます。データセンターは `0` から `n - 1` の番号で表されます。

配列 `times` が与えられます。

```python
times[i] = [from_node, to_node, delay]
```

これは `from_node` から `to_node` へ通知が届くまでに `delay` 秒かかることを表します。

開始地点 `start` から通知を送ったとき、**すべてのデータセンターに通知が届くまでに必要な最短時間**を返してください。

すべてのデータセンターに通知が届かない場合は `-1` を返してください。

### 制約

* `1 <= n <= 10^5`
* `0 <= len(times) <= 2 * 10^5`
* `0 <= from_node, to_node < n`
* `1 <= delay <= 10^9`
* `0 <= start < n`

## 実行例

### 例1

```python
n = 4
times = [
    [0, 1, 1],
    [0, 2, 4],
    [1, 2, 2],
    [2, 3, 1]
]
start = 0
```

最短到達時間は以下です。

```python
0 -> 0 = 0
0 -> 1 = 1
0 -> 2 = 3
0 -> 3 = 4
```

すべてに届くまでの時間は最大値の `4` です。

出力:

```python
4
```

### 例2

```python
n = 3
times = [
    [0, 1, 5]
]
start = 0
```

データセンター `2` には通知が届きません。

出力:

```python
-1
```

### 例3

```python
n = 3
times = [
    [0, 1, 2],
    [0, 2, 5],
    [1, 2, 1]
]
start = 0
```

`2` へは `0 -> 1 -> 2` の経路で `3` 秒が最短です。

出力:

```python
3
```

## 実行結果

```
4
-1
3
```
