# No.500

## 問題文

### 監視アラートの連続発生パターンを数える

整数配列 `alerts` が与えられます。

`alerts[i]` は `i` 分目に発生したアラート数です。

整数 `target` が与えられたとき、**合計が `target` になる連続部分配列の個数**を返してください。

## 実行例

### 例1

```python
alerts = [1, 1, 1]
target = 2
```

合計が `2` になる連続部分配列は以下の2つです。

```python
[1, 1]  # index 0-1
[1, 1]  # index 1-2
```

出力:

```python
2
```

### 例2

```python
alerts = [1, 2, 3]
target = 3
```

合計が `3` になる連続部分配列は以下の2つです。

```python
[1, 2]
[3]
```

出力:

```python
2
```

### 例3

```python
alerts = [3, 4, 7, 2, -3, 1, 4, 2]
target = 7
```

合計が `7` になる連続部分配列は以下の4つです。

```python
[3, 4]
[7]
[7, 2, -3, 1]
[1, 4, 2]
```

出力:

```python
4
```

## 実行結果

```
2
2
4
```
