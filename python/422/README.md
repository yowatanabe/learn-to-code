## 問題文

### 連続するリクエスト数が上限以内の最長期間

あなたはAPIサーバーのアクセス状況を監視しています。

配列 `requests` には、1分ごとのリクエスト数が記録されています。

整数配列 `requests` と整数 `limit` が与えられたとき、**連続部分配列の合計が `limit` 以下となるもののうち、長さが最大のもの** を求めてください。

その最大長を返してください。

### 制約

* `1 <= len(requests) <= 10^5`
* `1 <= requests[i] <= 10^4`
* `1 <= limit <= 10^9`

## 実行例

### 例1

```python
requests = [2, 1, 3, 2, 1]
limit = 5
```

合計が `5` 以下となる連続部分配列のうち、最長の一例は

```python
[2, 1]
```

または

```python
[2, 1]
```

や

```python
[3, 2]
```

ではなく、実際には

```python
[2, 1]
```

長さ `2` が最大です。

したがって出力は

```python
2
```

### 例2

```python
requests = [1, 1, 1, 1]
limit = 3
```

連続する3要素の合計は `3` なので、

```python
3
```

を返します。

### 例3

```python
requests = [10, 2, 3]
limit = 5
```

`10` は単体でも条件を満たしませんが、`[2, 3]` は合計 `5` です。

したがって出力は

```python
2
```

## 実行結果

```
2
3
2
```
