# No.164

## 問題文：配列内の2つの積の最大値

整数の配列 `nums` が与えられます。`nums` 内の**異なる2つの要素を選んで、それらの積を計算**し、その中で最大の値を返してください。

ただし、積を求めるときは、それぞれの数から1を引いた `(a - 1) * (b - 1)` の形で計算します。

### 制約

* `2 <= nums.length <= 500`
* `1 <= nums[i] <= 1000`

## 実行例

### 例 1:

```
入力: nums = [3, 4, 5, 2]
出力: 12
説明: (5 - 1) * (4 - 1) = 4 * 3 = 12
```

### 例 2:

```
入力: nums = [1, 5, 4, 5]
出力: 16
説明: (5 - 1) * (5 - 1) = 4 * 4 = 16
```

### 例 3:

```
入力: nums = [10, 2, 5, 2]
出力: 36
説明: (10 - 1) * (5 - 1) = 9 * 4 = 36
```

### 実行結果

```
12
16
36
```
