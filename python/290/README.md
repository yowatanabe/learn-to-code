# No.290

## 問題: 和が 0 になる 3 つ組 (3Sum)

整数配列 `nums` が与えられます。

> 「**合計が 0 になる 3 つの要素の組**」を、
> **重複なく** すべて列挙してください。

* 3つ組は「インデックス」ではなく「値の組」として返します。
* 3つ組内の順番は問いません（例: `[-1, 0, 1]` と `[0, -1, 1]` は同じとみなす）。
* 結果リスト内でも順番は問いません。
* 同じ値の組を **重複して返してはいけません**。

### 制約（目安）

* `3 ≤ len(nums) ≤ 10^3`
* `-10^5 ≤ nums[i] ≤ 10^5`

時間計算量は **O(n^2)** を目指してください（愚直な O(n^3) は TLE 想定）。

## 実行例

```python
# 例1
nums = [-1, 0, 1, 2, -1, -4]
# 合計が0になる3つ組は以下の2つ:
# [-1, -1, 2]
# [-1,  0, 1]
# 出力: [[-1, -1, 2], [-1, 0, 1]] など（順番は不問）

# 例2
nums = [0, 1, 1]
# 合計が0になる3つ組は存在しない
# 出力: []

# 例3
nums = [0, 0, 0, 0]
# 合計が0になる3つ組は [0, 0, 0] 1種類だけ
# 出力: [[0, 0, 0]]
```

## 実行結果

```
例1の入力: [-1, 0, 1, 2, -1, -4]
例1の結果: [[-1, -1, 2], [-1, 0, 1]]
例2の入力: [0, 1, 1]
例2の結果: []
例3の入力: [0, 0, 0, 0]
例3の結果: [[0, 0, 0]]
```
