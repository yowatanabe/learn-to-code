## 問題: 不安定なメトリクスを正規化せよ（Product of Array Except Self）

SRE のメトリクス前処理を想定します。

整数配列 `nums` が与えられます（0 を含む可能性あり）。

> `answer[i]` を「`nums[i]` 以外のすべての要素の積」として、配列 `answer` を返してください。
> ただし **割り算は使わない**でください。
> 時間計算量は **O(n)** を目指してください。

### 制約（目安）

* `2 ≤ len(nums) ≤ 10^5`
* `-30 ≤ nums[i] ≤ 30`

## 実行例

```python
# 例1
nums = [1, 2, 3, 4]
# answer = [24, 12, 8, 6]
# 出力: [24, 12, 8, 6]

# 例2
nums = [-1, 1, 0, -3, 3]
# 0 があるので、0以外の位置は 0 を含む積になり 0
# 0 の位置だけは 0 を除いた積 = (-1)*1*(-3)*3 = 9
# 出力: [0, 0, 9, 0, 0]
```

## 実行結果

```

例2 出力: [0, 0, 9, 0, 0] (期待値: [0, 0, 9, 0, 0])
------------------------------------------------------------
```
