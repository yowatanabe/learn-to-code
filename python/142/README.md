# No.142

## 問題：和が0になる3つの数の組み合わせ

整数の配列 `nums` が与えられます。合計が 0 になる **3つの整数の組み合わせ**をすべて求めてください（順不同）。

* 同じ組み合わせ（順序を変えただけのもの）は重複として数えず、**一度だけ返してください**。

### 実行例

#### 例 1:

```
入力: nums = [-1, 0, 1, 2, -1, -4]
出力: [[-1, -1, 2], [-1, 0, 1]]
```

#### 例 2:

```
入力: nums = [0, 1, 1]
出力: []
```

#### 例 3:

```
入力: nums = [0, 0, 0]
出力: [[0, 0, 0]]
```

### 実行結果

```text
[[-1, -1, 2], [-1, 0, 1]]
[]
[[0, 0, 0]]
```
