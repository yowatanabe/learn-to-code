## 問題文

### 障害チケットの処理順を決める

あなたは障害チケットを優先度順に処理します。

整数配列 `tickets` が与えられ、`tickets[i]` は `i` 番目のチケットの重要度を表します。

1回の操作で、以下を行います。

* まだ処理していないチケットの中から、重要度が最も高いものを1つ処理する
* 重要度が同じチケットが複数ある場合は、元の順番が早いものを先に処理する

整数 `target_index` が与えられたとき、`target_index` のチケットが何番目に処理されるかを返してください。

## 実行例

### 例1

```python
tickets = [2, 1, 3, 2]
target_index = 0
```

処理順は index `2 -> 0 -> 3 -> 1` です。
index `0` は2番目に処理されます。

出力:

```python
2
```

### 例2

```python
tickets = [1, 1, 1]
target_index = 2
```

重要度がすべて同じなので、元の順番通りです。

出力:

```python
3
```

### 例3

```python
tickets = [5, 3, 5, 4]
target_index = 2
```

index `0` と index `2` は重要度 `5` ですが、index `0` が先です。
index `2` は2番目です。

出力:

```python
2
```

## 実行結果

```
2
3
2
```
