# No.414

## 問題：最小帯域幅でのログ転送

あなたは SRE として、複数のログファイルを制限時間内に転送したいです。

整数配列 `files` が与えられ、`files[i]` は `i` 番目のログファイルのサイズを表します。

1 時間あたり `x` MB の帯域幅で転送するとき、1 つのファイル `files[i]` を転送するのに必要な時間は`ceil(files[i] / x)` 時間です。

すべてのファイルを **合計 `h` 時間以内** に転送したいです。

この条件を満たす **最小の整数帯域幅 `x`** を求めてください。

* `1 <= len(files) <= 2 * 10^5`
* `1 <= files[i] <= 10^9`
* `len(files) <= h <= 10^14`

**求めるもの**：条件を満たす最小の整数帯域幅（整数）

## 実行例

例1:

* 入力: `files = [3, 6, 7, 11]`, `h = 8`
* 説明:

  * 帯域幅 `x = 4` のとき

    * `ceil(3/4) = 1`
    * `ceil(6/4) = 2`
    * `ceil(7/4) = 2`
    * `ceil(11/4) = 3`
  * 合計 `1 + 2 + 2 + 3 = 8` 時間なので条件を満たす
  * `x = 3` だと合計 10 時間で間に合わない
* 出力: `4`

例2:

* 入力: `files = [30, 11, 23, 4, 20]`, `h = 5`
* 説明:

  * 5 時間以内に終えるには、ほぼ各ファイルを 1 時間以内で転送する必要がある
  * 最小帯域幅は `30`
* 出力: `30`

例3:

* 入力: `files = [30, 11, 23, 4, 20]`, `h = 6`
* 説明:

  * 帯域幅 `x = 23` のとき

    * `ceil(30/23) = 2`
    * `ceil(11/23) = 1`
    * `ceil(23/23) = 1`
    * `ceil(4/23) = 1`
    * `ceil(20/23) = 1`
  * 合計 `6` 時間
* 出力: `23`

## 実行結果

```
4
30
23
```
