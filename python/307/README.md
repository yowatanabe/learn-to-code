# No.307

暗号化された文字列 `s` が与えられます。`s` は `'0'`〜`'9'` と `'*'` からなります。

復号ルールは次の通りです。

* `'1'` → `'A'`, `'2'` → `'B'`, …, `'26'` → `'Z'`
* `'*'` は **`'1'`〜`'9'` のどれか 1 文字**を表す（`'0'` は表さない）

あなたのタスクは：

> `s` を有効にデコードできる通り数を求めてください。
> ただし答えが非常に大きくなる可能性があるので、`10^9 + 7` で割った余りを返してください。

### 注意

* `'0'` 単体は無効（どの文字にもならない）
* `'10'`, `'20'` のように 2 桁として解釈される場合は有効
* 2 桁として解釈できるのは `"10"`〜`"26"` の範囲のみ

### 制約（目安）

* `1 ≤ len(s) ≤ 10^5`

`O(n)` 時間、`O(1)` 追加メモリ（DPのローリング）を目指してください。

## 実行例

```python
# 例1
s = "*"
# '*' は '1'〜'9' のどれか → 9通り
# 出力: 9

# 例2
s = "1*"
# '1' + '*' で 9通り（11〜19のうち1桁+1桁として）
# さらに "1*" を 2桁として見ると 11〜19 の 9通り
# 合計 18通り
# 出力: 18

# 例3
s = "*0"
# "*0" は 10 or 20 になる必要がある
# '*' は 1 or 2 のときだけ有効 → 2通り
# 出力: 2

# 例4
s = "0*"
# 先頭 '0' は単体で無効 → 復元不能
# 出力: 0
```

## 実行結果

```
------------------------------
入力: *
出力: 9
------------------------------
入力: 1*
出力: 18
------------------------------
入力: *0
出力: 2
------------------------------
入力: 0*
出力: 0
```
