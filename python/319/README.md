## 問題: 文字列の乗算（Multiply Strings）

非負整数を表す 2 つの文字列 `num1` と `num2` が与えられます。

`num1 * num2` の結果を **文字列**として返してください。

制約として、次のことは禁止です：

* `int(num1)` や `int(num2)` のように **整数へ直接変換してはいけない**
* Python の `bigint`（任意精度整数）に頼った計算も禁止（つまり `int` を使うのはNG）

`num1` と `num2` の長さは最大 200 とします。

### 制約（目安）

* `1 ≤ len(num1), len(num2) ≤ 200`
* `num1` と `num2` は `'0'`〜`'9'` のみ
* 先頭に余計な 0 はない（ただし `"0"` はあり得る）

## 実行例

```python
# 例1
num1 = "2"
num2 = "3"
# 出力: "6"

# 例2
num1 = "123"
num2 = "456"
# 123 * 456 = 56088
# 出力: "56088"

# 例3
num1 = "0"
num2 = "999"
# 出力: "0"

# 例4
num1 = "999"
num2 = "999"
# 999 * 999 = 998001
# 出力: "998001"
```

## 実行結果

```

----------------------------------------

----------------------------------------
例3 入力: num1='0', num2='999'
例3 出力: '0'
----------------------------------------
例4 入力: num1='999', num2='999'
例4 出力: '998001'
----------------------------------------
```
