## 問題文（LeetCode風・日本語）

### 問題：再送制御の最小コスト

あなたは SRE として、失敗したジョブの再送設定を見直しています。

各ジョブには現在の再送間隔 `intervals[i]` が設定されています。

運用ルールとして、**最終的な再送間隔の列は非減少**（前から見て小さくならない）でなければなりません。

あなたは各要素 `intervals[i]` を **1回の操作で 1 だけ増やす** ことができます。要素を減らすことはできません。

配列 `intervals` が与えられたとき、配列全体を非減少にするために必要な**最小操作回数**を求めてください。

* `1 <= len(intervals) <= 2 * 10^5`
* `0 <= intervals[i] <= 10^9`

**求めるもの**：最小操作回数（整数）

## 実行例

例1:

* 入力: `intervals = [1, 1, 1]`
* 説明:

  * すでに非減少なので操作は不要
* 出力: `0`

例2:

* 入力: `intervals = [3, 1, 2, 1, 5]`
* 説明:

  * 2番目の `1` を `3` にする: 2回
  * 3番目の `2` を `3` にする: 1回
  * 4番目の `1` を `3` にする: 2回
  * 合計 `5` 回
  * 最終的に `[3, 3, 3, 3, 5]`
* 出力: `5`

例3:

* 入力: `intervals = [0, 5, 2, 2]`
* 説明:

  * 3番目の `2` を `5` にする: 3回
  * 4番目の `2` を `5` にする: 3回
  * 合計 `6` 回
  * 最終的に `[0, 5, 5, 5]`
* 出力: `6`
