## 問題: 文字列をデコードせよ（Decode String）

エンコード規則に従って作られた文字列 `s` が与えられます。

この規則は次の通りです：

* `k[encoded_string]` は、`encoded_string` を **k 回繰り返した文字列**を表す
* `k` は正の整数（1以上）
* ネスト（入れ子）があり得ます。例: `3[a2[c]]` → `accaccacc`

`s` をデコードした文字列を返してください。

### 制約（目安）

* `1 ≤ len(s) ≤ 10^5`
* `s` は英小文字、数字、`[`、`]` からなる
* 出力文字列長は `10^5` 以下と仮定してよい

## 実行例

```python
# 例1
s = "3[a]2[bc]"
# "aaa" + "bcbc" → "aaabcbc"
# 出力: "aaabcbc"

# 例2
s = "3[a2[c]]"
# "a2[c]" → "acc"
# それを3回 → "accaccacc"
# 出力: "accaccacc"

# 例3
s = "2[abc]3[cd]ef"
# "abcabc" + "cdcdcd" + "ef" → "abcabccdcdcdef"
# 出力: "abcabccdcdcdef"
```

## 実行結果

```

----------------------------------------
例2 入力: '3[a2[c]]'
例2 出力: 'accaccacc'
----------------------------------------
例3 入力: '2[abc]3[cd]ef'
例3 出力: 'abcabccdcdcdef'
----------------------------------------
```
