## 問題文

### サービスのデプロイ順を復元する

あなたはサービスのデプロイ順序を分析しています。

`n` 個のサービスがあり、サービス番号は `0` から `n - 1` です。

配列 `relations` が与えられます。

```python
relations[i] = [before, after]
```

これは、サービス `before` をデプロイした後でなければ、サービス `after` をデプロイできないことを意味します。

すべてのサービスをデプロイできる順序を1つ返してください。
循環依存があり不可能な場合は、空配列 `[]` を返してください。

## 実行例

### 例1

```python
n = 4
relations = [[0, 1], [0, 2], [1, 3], [2, 3]]
```

有効な順序の一例は、

```python
[0, 1, 2, 3]
```

です。

出力例:

```python
[0, 1, 2, 3]
```

### 例2

```python
n = 2
relations = [[0, 1], [1, 0]]
```

循環依存があります。

出力:

```python
[]
```

### 例3

```python
n = 3
relations = [[1, 2]]
```

`0` は独立しています。

出力例:

```python
[0, 1, 2]
```

## 実行結果

```
[0, 1, 2, 3]
[]
[0, 1, 2]
```
