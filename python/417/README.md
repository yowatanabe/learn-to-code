## 最短ログ区間

あなたは複数サービスのログを時系列順に受け取っています。

各ログには、そのログを出力したサービス名が文字列で記録されています。

配列 `logs` が与えられたとき、**配列内に登場するすべての異なるサービス名を少なくとも1回ずつ含む連続部分配列のうち、長さが最小のもの**を求めてください。

その最小長を返してください。

### 制約

* `1 <= len(logs) <= 10^5`
* `logs[i]` は英小文字・英大文字・数字・記号を含まない文字列
* `logs` には少なくとも1つの要素がある

## 実行例

### 例1

```python
logs = ["auth", "db", "cache", "auth", "queue", "db"]
```

配列全体に含まれる異なるサービス名は
`{"auth", "db", "cache", "queue"}` の4種類です。

これらすべてを含む最短の連続部分配列の1つは

```python
["cache", "auth", "queue", "db"]
```

なので、出力は

```python
4
```

### 例2

```python
logs = ["api", "api", "api"]
```

異なるサービス名は `{"api"}` の1種類だけです。
よって最短長は

```python
1
```

### 例3

```python
logs = ["web", "db", "web", "cache", "db", "queue", "cache"]
```

異なるサービス名は
`{"web", "db", "cache", "queue"}` の4種類です。

これらすべてを含む最短の連続部分配列の1つは

```python
["web", "cache", "db", "queue"]
```

なので、出力は

```python
4
```

## 実行結果

```
4
1
4
```
