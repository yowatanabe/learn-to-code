## 問題文

### IPアドレス範囲の不足区間を探す

整数配列 `used` が与えられます。

`used[i]` はすでに使用されているIPアドレス番号を表します。

IPアドレス番号は `0` から `max_ip` まで存在します。

使用されていないIPアドレス番号を、連続範囲としてまとめて返してください。

各範囲は以下の形式にします。

* 1つだけなら `"x"`
* 複数なら `"start->end"`

## 実行例

### 例1

```python
used = [0, 1, 3, 50, 75]
max_ip = 99
```

出力:

```python
["2", "4->49", "51->74", "76->99"]
```

### 例2

```python
used = [0, 1, 2, 3]
max_ip = 3
```

出力:

```python
[]
```

### 例3

```python
used = []
max_ip = 5
```

出力:

```python
["0->5"]
```

## 実行結果

```
['2', '4->49', '51->74', '76->99']
[]
['0->5']
```
