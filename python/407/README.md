# No.407

## 問題：全種類のイベントを含む最短ログ区間

あなたは SRE として、時系列ログに現れるイベントIDを分析しています。

整数配列 `events` が与えられます。

`events` 全体に登場する **すべての異なるイベントID** を少なくとも1回ずつ含む連続部分配列のうち、**長さが最短**のものを求めてください。

その最短長を返してください。

* `1 <= len(events) <= 2 * 10^5`
* `-10^9 <= events[i] <= 10^9`

**求めるもの**：条件を満たす最短の連続部分配列の長さ（整数）

## 実行例

例1:

* 入力: `events = [1, 2, 2, 3, 1]`
* 説明:

  * 配列全体に存在する異なる値は `{1, 2, 3}`
  * `[2, 3, 1]` はすべてを含むので長さ `3`
  * これより短い区間では全種類を含めない
* 出力: `3`

例2:

* 入力: `events = [5, 5, 5, 5]`
* 説明:

  * 異なる値は `{5}` だけ
  * `[5]` で条件を満たす
* 出力: `1`

例3:

* 入力: `events = [7, 1, 3, 7, 3, 1, 4, 1, 4]`
* 説明:

  * 異なる値は `{1, 3, 4, 7}`
  * `[3, 7, 3, 1, 4]` や `[7, 3, 1, 4]` が条件を満たす
  * 最短は `[7, 3, 1, 4]` で長さ `4`
* 出力: `4`

## 実行結果

```
3
1
4
```
