## 問題：2つのリストの共通要素

2つの整数のリスト `list1` と `list2` が与えられます。
両方のリストに共通して含まれる要素を**重複なしで**昇順にソートしてリストで返してください。

### 実行例

#### 例 1:

```
入力: list1 = [1, 2, 2, 3], list2 = [2, 3, 4]
出力: [2, 3]
```

#### 例 2:

```
入力: list1 = [5, 6, 7], list2 = [7, 8, 9]
出力: [7]
```

#### 例 3:

```
入力: list1 = [1, 2], list2 = [3, 4]
出力: []
```

### 実行結果

```text
[2, 3]
[7]
[]
```
