## 問題文

### 監視イベントから重複しない最長区間を求める

あなたは時系列の監視イベントを分析しています。

文字列 `events` が与えられ、`events[i]` は `i` 番目に発生したイベント種別を表します。

**同じイベント種別を2回以上含まない連続部分文字列のうち、長さが最大のもの**を求めてください。

その最大長を返してください。

### 制約

* `0 <= len(events) <= 10^5`
* `events` は英小文字のみからなる

## 実行例

### 例1

```python
events = "abcabcbb"
```

重複しない最長区間は `"abc"` です。

出力:

```python
3
```

### 例2

```python
events = "bbbbb"
```

重複しない最長区間は `"b"` です。

出力:

```python
1
```

### 例3

```python
events = "pwwkew"
```

重複しない最長区間は `"wke"` です。

出力:

```python
3
```

## 実行結果

```
3
1
3
```
