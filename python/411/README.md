# No.411

## 問題：ログ文字列内のアナグラム開始位置

あなたは SRE として、ログ文字列の中から特定パターンの並び替えを検出したいです。

文字列 `s` と文字列 `p` が与えられます。

`p` の **アナグラム** になっている部分文字列が `s` の中に存在する開始位置を、すべて求めてください。

アナグラムとは、文字の出現回数が同じで並び順だけが異なる文字列です。

* 返す開始位置は昇順で返してください
* `1 <= len(s), len(p) <= 2 * 10^5`
* `s` と `p` は英小文字のみからなる

**求めるもの**：`p` のアナグラムになっている部分文字列の開始インデックスの配列

## 実行例

例1:

* 入力: `s = "cbaebabacd"`, `p = "abc"`
* 説明:

  * `s[0:3] = "cba"` は `"abc"` のアナグラム
  * `s[6:9] = "bac"` も `"abc"` のアナグラム
* 出力: `[0, 6]`

例2:

* 入力: `s = "abab"`, `p = "ab"`
* 説明:

  * `s[0:2] = "ab"`
  * `s[1:3] = "ba"`
  * `s[2:4] = "ab"`
    の3つが条件を満たす
* 出力: `[0, 1, 2]`

例3:

* 入力: `s = "aaaaa"`, `p = "b"`
* 説明:

  * `"b"` のアナグラムに当たる部分文字列は存在しない
* 出力: `[]`

## 実行結果

```
[0, 6]
[0, 1, 2]
[]
```
