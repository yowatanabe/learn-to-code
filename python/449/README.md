## 問題文

### 最小削除でログレベル列を昇順にする

あなたは時系列ログの重要度を数値化して管理しています。

整数配列 `levels` が与えられ、`levels[i]` は `i` 番目のログの重要度を表します。

いくつかのログを削除して、残ったログの重要度が **厳密に昇順** になるようにしたいです。

このとき、**削除する必要があるログの最小数**を返してください。

### 制約

* `1 <= len(levels) <= 10^5`
* `0 <= levels[i] <= 10^9`

## 実行例

### 例1

```python
levels = [5, 1, 6, 2, 3, 4]
```

`[1, 2, 3, 4]` を残すと厳密に昇順になります。
残せる最大個数は `4` 個なので、削除数は

```python
6 - 4 = 2
```

出力:

```python
2
```

### 例2

```python
levels = [1, 2, 3, 4]
```

すでに厳密に昇順なので、削除は不要です。

出力:

```python
0
```

### 例3

```python
levels = [4, 3, 2, 1]
```

どれか1つだけ残せます。
削除数は

```python
3
```

## 実行結果

```
2
0
3
```
