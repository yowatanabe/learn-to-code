# No.104

### 問題: リスト内の最大値と最小値の差を求める

整数のリスト `nums` が与えられます。
このリスト内の最大値と最小値の差を返す関数 `max_min_difference` を実装してください。

### 実行例

#### 例 1:

```
入力: nums = [1, 2, 3, 4, 5]
出力: 4
説明: 5 - 1 = 4
```

#### 例 2:

```
入力: nums = [10, -2, 7, 4]
出力: 12
説明: 10 - (-2) = 12
```

#### 例 3:

```
入力: nums = [42]
出力: 0
説明: 最大値と最小値が同じなので差は0
```

### 実行結果

```text
4
12
0
```
