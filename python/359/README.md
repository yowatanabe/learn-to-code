## 問題: 文字列を分割して回文にせよ（Palindrome Partitioning II）

文字列 `s` が与えられます。

> `s` をいくつかの連続部分文字列に分割し、すべての部分文字列が **回文** になるようにしたい。
> 必要な **分割回数（cut回数）** の最小値を返してください。
> （cut回数 = 分割してできる部分文字列の個数 - 1）

### 例

* `s = "aab"` は `"aa" | "b"` にすると全部回文なので、cut回数は `1`。

### 制約（目安）

* `1 ≤ len(s) ≤ 2000`
* `s` は英小文字のみ（としてよい）

期待解法: DP（回文判定の前計算）で **O(n^2)**

## 実行例

```python
# 例1
s = "aab"
# "aa" | "b"
# 出力: 1

# 例2
s = "a"
# すでに回文
# 出力: 0

# 例3
s = "abccbc"
# 最小例: "a" | "bccb" | "c"
# 出力: 2
```

## 実行結果

```
例1 入力: s='aab'
例1 出力: 1 (期待値: 1)
------------------------------------------------------------
例2 入力: s='a'
例2 出力: 0 (期待値: 0)
------------------------------------------------------------
例3 入力: s='abccbc'
例3 出力: 2 (期待値: 2)
------------------------------------------------------------
```
