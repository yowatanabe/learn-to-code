# No.467

## 問題文

### 監視アラートの復旧に必要な最小反転回数

あなたは時系列の監視状態を文字列 `status` で管理しています。

* `"0"`: 正常
* `"1"`: 異常

1回の操作で、任意の位置から末尾までの状態をすべて反転できます。

つまり、ある index `i` を選ぶと、`status[i:]` の `"0"` は `"1"` に、`"1"` は `"0"` に変わります。

すべての状態を `"0"` にするために必要な**最小操作回数**を返してください。

### 制約

* `1 <= len(status) <= 10^5`
* `status[i]` は `"0"` または `"1"`

## 実行例

### 例1

```python
status = "10111"
```

最小操作の一例:

```python
10111
01000  # index 0 から反転
00011  # index 1 から反転
00000  # index 3 から反転
```

出力:

```python
2
```

### 例2

```python
status = "0000"
```

すでにすべて正常です。

出力:

```python
0
```

### 例3

```python
status = "010"
```

最小操作の一例:

```python
010
001  # index 1 から反転
000  # index 2 から反転
```

出力:

```python
2
```

## 実行結果

```
3
0
2
```
