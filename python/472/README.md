## 問題文

### ログ圧縮後の最短長

あなたは連続する同じログレベルを圧縮する仕組みを作っています。

文字列 `s` が与えられ、各文字はログレベルを表します。

同じ文字が連続している部分は、次の形式で圧縮されます。

* 1文字だけなら、その文字だけ
* 2文字以上なら、`文字 + 連続数`

例えば、

```python
"aaabbc"
```

は

```python
"a3b2c"
```

となり、圧縮後の長さは `5` です。

あなたは `s` から **最大で `k` 文字まで削除**できます。

削除後に圧縮したときの、圧縮文字列の最小長を返してください。

### 制約

* `1 <= len(s) <= 100`
* `0 <= k <= len(s)`
* `s` は英小文字のみ

## 実行例

### 例1

```python
s = "aaabcccd"
k = 2
```

`"b"` と `"d"` を削除すると、

```python
"aaaccc"
```

圧縮後は

```python
"a3c3"
```

長さは `4` です。

出力:

```python
4
```

### 例2

```python
s = "aabbaa"
k = 2
```

`"bb"` を削除すると、

```python
"aaaa"
```

圧縮後は

```python
"a4"
```

長さは `2` です。

出力:

```python
2
```

### 例3

```python
s = "abc"
k = 1
```

どれか1文字を削除すると、2文字残ります。

圧縮後の長さは

```python
2
```

## 実行結果

```
4
2
2
```
