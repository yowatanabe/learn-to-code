# No.415

## 問題：ログ文字列の最小分割

あなたは SRE として、1 本の長いログ文字列を複数の区間に分割して管理したいと考えています。

文字列 `s` が与えられます。

文字列をいくつかの連続した部分文字列に分割するとき、**各文字が含まれるのは高々 1 つの部分文字列だけ**になるようにしたいです。

つまり、ある文字が複数の部分文字列にまたがって現れてはいけません。

この条件を満たしつつ、**できるだけ多くの部分文字列に分割**してください。

各部分文字列の長さを、左から順に配列で返してください。

* `1 <= len(s) <= 2 * 10^5`
* `s` は英小文字のみからなる

**求めるもの**：分割後の各部分文字列の長さを並べた整数配列

## 実行例

例1:

* 入力: `s = "ababcbacadefegdehijhklij"`
* 説明:

  * `"ababcbaca"`, `"defegde"`, `"hijhklij"` に分割できる
  * 各文字は 1 つの区間にしか現れない
* 出力: `[9, 7, 8]`

例2:

* 入力: `s = "eccbbbbdec"`
* 説明:

  * `e` と `c` が文字列の前後にまたがって現れるため、全体を 1 区間にする必要がある
* 出力: `[10]`

例3:

* 入力: `s = "caedbdedda"`
* 説明:

  * `"c"`, `"aedbdedda"` に分割できる
* 出力: `[1, 9]`

## 実行結果

```
[9, 7, 8]
[10]
[1, 9]
```
