## 問題: ログの最小カバーレンジ（Minimum Window Substring）

文字列 `s` と `t` が与えられます。
`t` に含まれる文字（重複を含む）を **すべて含む** `s` の連続部分文字列のうち、**長さが最短**のものを返してください。

* 例えば `t = "AABC"` の場合、返す部分文字列には `'A'` が2回、`'B'` が1回、`'C'` が1回含まれている必要があります。
* 条件を満たす部分文字列が存在しない場合は空文字列 `""` を返してください。
* 文字は英数字など任意のASCIIとします（大小区別あり）。

時間計算量 **O(len(s) + len(t))** を目指してください。

## 実行例

```python
# 例1
s = "ADOBECODEBANC"
t = "ABC"
# "BANC" が最短で条件を満たす
# 出力: "BANC"

# 例2
s = "a"
t = "a"
# 出力: "a"

# 例3
s = "a"
t = "aa"
# 条件を満たす部分文字列が存在しない
# 出力: ""
```

## 実行結果

```
例1 入力: s='ADOBECODEBANC', t='ABC'
例1 出力: 'BANC'
----------------------------------------
例2 入力: s='a', t='a'
例2 出力: 'a'
----------------------------------------
例3 入力: s='a', t='aa'
例3 出力: ''
----------------------------------------
```
