## 問題文

### クールダウン付きタスク実行

あなたはサーバー上で複数のタスクを順番に実行します。

各タスクは英大文字1文字で表され、同じ文字は同じ種類のタスクを意味します。

配列 `tasks` と整数 `n` が与えられます。

同じ種類のタスクは、**実行してから次に同じタスクを実行するまでに少なくとも `n` 個の時間単位を空けなければなりません。**

1つの時間単位では、以下のどちらか1つを行えます。

* 1つのタスクを実行する
* 何もせず待機する

タスクはすべて必ず実行しなければなりません。

**すべてのタスクを実行し終えるまでに必要な最小の時間単位数** を返してください。

### 制約

* `1 <= len(tasks) <= 10^5`
* `tasks[i]` は `"A"` から `"Z"` のいずれか
* `0 <= n <= 100`

## 実行例

### 例1

```python
tasks = ["A", "A", "A", "B", "B", "B"]
n = 2
```

`A` と `B` を交互に実行しても、最後に待機が必要になります。

一例:

```python
A -> B -> idle -> A -> B -> idle -> A -> B
```

出力:

```python
8
```

### 例2

```python
tasks = ["A", "A", "A", "B", "B", "B"]
n = 0
```

クールダウンがないので、そのまま連続実行できます。

出力:

```python
6
```

### 例3

```python
tasks = ["A", "A", "A", "A", "B", "C", "D"]
n = 2
```

最も多い `A` を基準に考えると、待機が必要になります。

出力:

```python
10
```

## 実行結果

```
8
6
10
```
