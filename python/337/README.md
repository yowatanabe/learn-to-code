## 問題: 最長の連続シーケンス（Longest Consecutive Sequence）

整数配列 `nums` が与えられます。

> `nums` の中に含まれる整数を使って作れる **連続する整数列（例: 5,6,7,8）** のうち、
> **最長の長さ**を返してください。

* 連続は「差が 1」のこと（`x, x+1, x+2, ...`）
* 並び順は配列中の順序に関係ありません（集合として考える）
* 時間計算量は **O(n)** を目指してください（ソート `O(n log n)` は避けたい）

### 制約（目安）

* `0 ≤ len(nums) ≤ 10^5`
* `-10^9 ≤ nums[i] ≤ 10^9`

## 実行例

```python
# 例1
nums = [100, 4, 200, 1, 3, 2]
# 1,2,3,4 が連続で長さ 4
# 出力: 4

# 例2
nums = [0,3,7,2,5,8,4,6,0,1]
# 0,1,2,3,4,5,6,7,8 が連続で長さ 9
# 出力: 9

# 例3
nums = []
# 出力: 0
```

## 実行結果

```
例2 出力: 9 (期待値: 9)
--------------------------------------------------
例3 入力: nums=[]

--------------------------------------------------
```
