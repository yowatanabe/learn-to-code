## 問題文

### インシデント通知の削除

あなたは監視システムから送られてくるインシデント通知を処理しています。

通知は文字列の配列 `alerts` で与えられ、各要素は通知の種類を表します。

同じ種類の通知が **連続してちょうど `k` 個並んだら、そのグループを削除できる** とします。

削除は何度でも行えます。

あるグループを削除すると、左右の通知列がつながり、新たに削除可能なグループができることもあります。

最終的に削除しきれずに残る通知の数を返してください。

## 実行例

### 例1

```python
alerts = ["db", "db", "api", "api", "api", "db", "db"]
k = 2
```

* `["db", "db"]` を削除
* `["api", "api"]` を削除
* 残った `"api"` は1個
* 後ろの `["db", "db"]` を削除

最終的に残る要素数は

```python
1
```

### 例2

```python
alerts = ["a", "a", "b", "b", "b", "a"]
k = 3
```

* `["b", "b", "b"]` を削除
* 残りは `["a", "a", "a"]`
* これも削除

最終的に残る要素数は

```python
0
```

### 例3

```python
alerts = ["web", "web", "cache", "db"]
k = 2
```

* `["web", "web"]` を削除
* 残りは `["cache", "db"]`

最終的に残る要素数は

```python
2
```

## 実行結果

```
1
0
2
```
