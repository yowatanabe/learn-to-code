# No.480

## 問題文

### サービス名のグループ化

あなたは複数サービスの名前一覧を整理しています。

文字列配列 `names` が与えられます。

2つのサービス名が、文字を並べ替えることで同じ文字列になる場合、それらは同じグループとみなします。

すべてのサービス名をグループ化し、各グループを配列として返してください。グループの順番は問いません。

### 制約

* `1 <= len(names) <= 10^4`
* `1 <= len(names[i]) <= 100`
* `names[i]` は英小文字のみ

## 実行例

### 例1

```python
names = ["api", "pia", "db", "bd", "cache"]
```

`"api"` と `"pia"` は同じ文字から作れます。
`"db"` と `"bd"` も同じ文字から作れます。

出力例:

```python
[["api", "pia"], ["db", "bd"], ["cache"]]
```

### 例2

```python
names = ["abc", "bca", "cab", "xyz"]
```

出力例:

```python
[["abc", "bca", "cab"], ["xyz"]]
```

### 例3

```python
names = ["a"]
```

出力:

```python
[["a"]]
```

## 実行結果

```
[['api', 'pia'], ['db', 'bd'], ['cache']]
[['abc', 'bca', 'cab'], ['xyz']]
[['a']]
```
