# No.146

## 問題：ウィンドウ内の最大値

整数配列 `nums` と整数 `k` が与えられます。長さ `k` のスライディングウィンドウを配列の左端から右端まで移動させたとき、**各ウィンドウ内の最大値を順に出力する配列**を返してください。

### 実行例

#### 例 1:

```
入力: nums = [1,3,-1,-3,5,3,6,7], k = 3
出力: [3,3,5,5,6,7]
説明:
ウィンドウ位置                最大値
---------------              ------
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7
```

#### 例 2:

```
入力: nums = [1], k = 1
出力: [1]
```

### 実行結果

```text
[3, 3, 5, 5, 6, 7]
[1]
```
