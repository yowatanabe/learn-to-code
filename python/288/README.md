## 問題: 出現回数が多い上位 K 個の要素

整数配列 `nums` と整数 `k` が与えられます。

> 配列 `nums` の中で、**出現回数が多い要素を上位 `k` 個** 求めてください。

* 戻り値は、**要素そのもののリスト** とします（回数ではなく値）。
* 答えの順番は問いません（頻度順にソートされていなくてもよい）。
* `k` 個ちょうど返してください（`k` ≤ 異なる要素数）。

### 制約（目安）

* `1 ≤ len(nums) ≤ 10^5`
* `-10^4 ≤ nums[i] ≤ 10^4`
* `1 ≤ k ≤`（`nums` 内の異なる要素の個数）

時間計算量は **O(n log k)** 程度を目指してください。
（全要素をソートする O(n log n) ではなく、「必要な上位 k 個だけ」を取り出せるとベターです）

## 実行例

```python
# 例1
nums = [1, 1, 1, 2, 2, 3]
k = 2
# 出現回数:
# 1 -> 3回, 2 -> 2回, 3 -> 1回
# 上位2個は [1, 2] （順番は [2, 1] でも可）
# 出力: [1, 2]  または  [2, 1]

# 例2
nums = [4, 4, 4, 6, 6, 7, 7, 7, 7]
k = 1
# 出現回数:
# 4 -> 3回, 6 -> 2回, 7 -> 4回
# 最も多いのは 7
# 出力: [7]

# 例3
nums = [5, 3, 5, 2, 2, 3]
k = 3
# 出現回数:
# 5 -> 2回, 3 -> 2回, 2 -> 2回
# すべて同じ回数なので、[5, 3, 2] など任意の順で3つ返せばよい
# 出力例: [5, 3, 2]
```

## 実行結果

```
例1の入力: [1, 1, 1, 2, 2, 3] k = 2
例1の結果: [2, 1]
例2の入力: [4, 4, 4, 6, 6, 7, 7, 7, 7] k = 1
例2の結果: [7]
例3の入力: [5, 3, 5, 2, 2, 3] k = 3
例3の結果: [2, 5, 3]
```
