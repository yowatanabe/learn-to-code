## 問題：削除する区間の最小本数

あなたは SRE として、複数のメンテナンス作業時間帯を管理しています。

各作業は区間 `[start, end]` で表され、`start < end` を満たします。

同時に実行できない作業があるため、**互いに重なっている区間が存在しない状態**にしたいです。

そのために、いくつかの区間を削除します。

整数の二次元配列 `intervals` が与えられたとき、

**残った区間同士が重ならないようにするために削除すべき区間の最小本数**を返してください。

ここで、区間 `[a, b]` と `[c, d]` は **`c < b` のとき重なっている** とみなします。

つまり、`c == b` の場合は重なっていません。

* `1 <= len(intervals) <= 2 * 10^5`
* `-10^9 <= start < end <= 10^9`

**求めるもの**：削除する区間の最小本数（整数）

## 実行例

例1:

* 入力: `intervals = [[1,2],[2,3],[3,4],[1,3]]`
* 説明:

  * `[1,3]` を削除すれば、残りの区間は互いに重ならない
* 出力: `1`

例2:

* 入力: `intervals = [[1,2],[1,2],[1,2]]`
* 説明:

  * 3本のうち1本しか残せないので、2本削除する必要がある
* 出力: `2`

例3:

* 入力: `intervals = [[1,2],[2,3]]`
* 説明:

  * `[1,2]` と `[2,3]` は端で接しているだけで、重なっていない
* 出力: `0`

## 実行結果

```
1
2
0
```
