# No.144

## 問題：和が目標値に一致する部分配列の数

整数のリスト `nums` と整数 `k` が与えられます。連続する部分配列（空でない）のうち、**要素の合計が `k` に等しい部分配列の数**を返してください。

### 実行例

#### 例 1:

```
入力: nums = [1,1,1], k = 2
出力: 2
説明: 2つの部分配列 [1,1]（最初の2つと最後の2つ）が合計2になる
```

#### 例 2:

```
入力: nums = [1,2,3], k = 3
出力: 2
説明: [1,2] と [3] が合計3
```

#### 例 3:

```
入力: nums = [1,-1,0], k = 0
出力: 3
```

### 実行結果

```text
2
2
3
```
