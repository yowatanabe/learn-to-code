## 問題: 温度が上がるまで待て（Daily Temperatures）

整数配列 `temperatures` が与えられます。`temperatures[i]` は `i` 日目の気温です。

> 各日 `i` について、`i` 日目より **気温が高い日** が **次に訪れるまで** 何日待つ必要があるかを求めてください。
> もし将来により高い日が存在しない場合は `0` を入れてください。

結果は同じ長さの配列 `ans` として返してください。

### 制約（目安）

* `1 ≤ len(temperatures) ≤ 10^5`
* `30 ≤ temperatures[i] ≤ 100`

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
temperatures = [73,74,75,71,69,72,76,73]
# 出力: [1,1,4,2,1,1,0,0]

# 例2
temperatures = [30,40,50,60]
# 出力: [1,1,1,0]

# 例3
temperatures = [30,60,90]
# 出力: [1,1,0]
```

## 実行結果

```
例1 入力: [73, 74, 75, 71, 69, 72, 76, 73]
例1 出力: [1, 1, 4, 2, 1, 1, 0, 0]
--------------------------------------------------
例2 入力: [30, 40, 50, 60]
例2 出力: [1, 1, 1, 0]
--------------------------------------------------
例3 入力: [30, 60, 90]
例3 出力: [1, 1, 0]
--------------------------------------------------
```
