## 問題文

### 最小コストで障害対応手順を完了する

あなたは障害対応の手順を順番に実行します。

整数配列 `costs` が与えられ、`costs[i]` は `i` 番目の手順を実行するコストを表します。

あなたは index `0` または index `1` から開始できます。

1回の移動で、次のどちらかへ進めます。

* 1つ先
* 2つ先

最後の手順の次に到達すれば、対応完了です。

対応完了までに必要な**最小コスト**を返してください。

## 実行例

### 例1

```python
costs = [10, 15, 20]
```

index `1` から開始し、コスト `15` を支払って完了地点へ進みます。

出力:

```python
15
```

### 例2

```python
costs = [1, 100, 1, 1, 1, 100, 1, 1, 100, 1]
```

最小コストは以下です。

```python
1 + 1 + 1 + 1 + 1 = 5
```

出力:

```python
5
```

### 例3

```python
costs = [5, 5, 10, 1]
```

index `1 -> 3 -> 完了` と進むと、

```python
5 + 1 = 6
```

出力:

```python
6
```

## 実行結果

```
15
6
6
```
