# No.416

## 問題：目的地に到達する車群の数

あなたは SRE として、複数のバッチジョブが同じ処理キューに向かって進む様子を監視しています。

各ジョブは 1 本の直線上を目的地 `target` に向かって進みます。

長さ `n` の配列 `position` と `speed` が与えられます。

* `position[i]` は `i` 番目のジョブの現在位置
* `speed[i]` は `i` 番目のジョブの速度

各ジョブは一定速度で進みます。

ただし、**後ろのジョブが前のジョブに追いついた場合、その後は前のジョブと同じ速度で進み、1つのグループ（車群）として扱います**。

目的地 `target` に到達するまでに、最終的に **何個の車群** になるかを求めてください。

* 追いつくのが目的地ちょうどの位置で起きる場合も、同じ車群とみなします
* `1 <= n <= 2 * 10^5`
* `0 < target <= 10^9`
* `0 <= position[i] < target`
* `1 <= speed[i] <= 10^6`
* `position` の値はすべて異なる

**求めるもの**：目的地に到達する車群の数（整数）

## 実行例

例1:

* 入力:
  `target = 12`
  `position = [10, 8, 0, 5, 3]`
  `speed = [2, 4, 1, 1, 3]`
* 説明:
  * 各ジョブの単独到達時間は
    * 位置10 → `(12-10)/2 = 1`
    * 位置8 → `(12-8)/4 = 1`
    * 位置5 → `(12-5)/1 = 7`
    * 位置3 → `(12-3)/3 = 3`
    * 位置0 → `(12-0)/1 = 12`
  * 位置10と位置8のジョブは目的地で合流して1車群
  * 位置5と位置3のジョブも途中で合流して1車群
  * 位置0のジョブは単独
* 出力:
  `3`

例2:

* 入力:
  `target = 10`
  `position = [3]`
  `speed = [3]`
* 説明:
  * 1台だけなので車群は1つ
* 出力:
  `1`

例3:

* 入力:
  `target = 10`
  `position = [0, 2, 4]`
  `speed = [4, 2, 1]`
* 説明:
  * 単独での到達時間はそれぞれ

    * 位置4 → `6`
    * 位置2 → `4`
    * 位置0 → `2.5`
  * 後ろのジョブほど速いので、前のジョブに追いつく
  * 最終的に **全て1つの車群** になる
* 出力:
  `1`

## 実行結果

```
3
1
1
```
