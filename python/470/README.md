## 問題文

### メトリクス異常値の上位K件を求める

あなたは監視システムから大量のメトリクス値を受け取っています。

整数配列 `metrics` が与えられ、`metrics[i]` はある時点のメトリクス値を表します。

整数 `k` が与えられたとき、**大きい順でk番目の値**を返してください。

重複する値も別々に数えます。

### 制約

* `1 <= k <= len(metrics) <= 10^5`
* `-10^9 <= metrics[i] <= 10^9`

## 実行例

### 例1

```python
metrics = [3, 2, 1, 5, 6, 4]
k = 2
```

大きい順に並べると、

```python
[6, 5, 4, 3, 2, 1]
```

2番目は `5` です。

出力:

```python
5
```

### 例2

```python
metrics = [3, 2, 3, 1, 2, 4, 5, 5, 6]
k = 4
```

大きい順に並べると、

```python
[6, 5, 5, 4, 3, 3, 2, 2, 1]
```

4番目は `4` です。

出力:

```python
4
```

### 例3

```python
metrics = [10]
k = 1
```

出力:

```python
10
```

## 実行結果

```
5
4
10
```
