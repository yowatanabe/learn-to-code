## 問題: より暖かい日まで何日待つか

あなたは毎日の気温を記録しています。

整数配列 `temperatures` が与えられ、`temperatures[i]` は **i 日目の気温** を表します。

各日 `i` について、次のことを求めてください:

> 「**自分よりも気温が高い日** がこの先に初めて現れるまで、**何日待てばよいか**」

* もしそのような日が存在しない場合、その日の答えは `0` とします。
* 戻り値は、各日について上記の値を並べた整数配列とします。

### 制約（目安）

* `1 ≤ len(temperatures) ≤ 10^5`
* `30 ≤ temperatures[i] ≤ 100`

時間計算量 **O(n)** で解くことを意識してください（全探索 O(n^2) は TLE 相当）。

## 実行例

```python
# 例1
temperatures = [73, 74, 75, 71, 69, 72, 76, 73]
# 各日の答え:
# 0日目: 73 -> 1日後に 74 が来る       → 1
# 1日目: 74 -> 1日後に 75 が来る       → 1
# 2日目: 75 -> 4日後に 76 が来る       → 4
# 3日目: 71 -> 2日後に 72 が来る       → 2
# 4日目: 69 -> 1日後に 72 が来る       → 1
# 5日目: 72 -> 1日後に 76 が来る       → 1
# 6日目: 76 -> これ以降、より高い日はない → 0
# 7日目: 73 -> これ以降、より高い日はない → 0
# 出力: [1, 1, 4, 2, 1, 1, 0, 0]

# 例2
temperatures = [30, 40, 50, 60]
# すべて右に行くほど高くなっている
# 出力: [1, 1, 1, 0]

# 例3
temperatures = [30, 60, 90]
# 出力: [1, 1, 0]
```

## 実行結果

```
例1の入力: [73, 74, 75, 71, 69, 72, 76, 73]
例1の結果: [1, 1, 4, 2, 1, 1, 0, 0]
例2の入力: [30, 40, 50, 60]
例2の結果: [1, 1, 1, 0]
例3の入力: [30, 60, 90]
例3の結果: [1, 1, 0]
```
