## 問題：障害ログの連続期間数

あなたは SRE として、日ごとの障害発生有無を記録した配列を分析しています。

整数配列 `logs` が与えられ、`logs[i]` は `i` 日目の状態を表します。

* `logs[i] = 1` のとき、その日は障害が発生した
* `logs[i] = 0` のとき、その日は障害が発生していない

**1 が連続している区間の数**を求めてください。

ここで、連続区間とは「隣り合う要素がすべて 1 の最大区間」のことです。

* `1 <= len(logs) <= 2 * 10^5`
* `logs[i]` は `0` または `1`

**求めるもの**：障害発生期間の個数（整数）

## 実行例

例1:

* 入力: `logs = [1, 1, 0, 1, 0, 1, 1, 1]`
* 説明:

  * `1,1`
  * `1`
  * `1,1,1`
    の3区間
* 出力: `3`

例2:

* 入力: `logs = [0, 0, 0]`
* 説明:

  * 1 の区間は存在しない
* 出力: `0`

例3:

* 入力: `logs = [1, 1, 1, 1]`
* 説明:

  * 全体が1つの連続区間
* 出力: `1`

## 実行結果

```
3
0
1
```
