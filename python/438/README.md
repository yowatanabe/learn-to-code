## 問題文

### エラー率のしきい値を超える区間数

あなたはAPIの1分ごとのエラー数を監視しています。整数配列 `errors` が与えられ、`errors[i]` は `i` 分目のエラー数を表します。

整数 `threshold` が与えられたとき、**合計エラー数が `threshold` 以上になる連続部分配列の個数** を返してください。

### 制約

* `1 <= len(errors) <= 10^5`
* `1 <= errors[i] <= 10^4`
* `1 <= threshold <= 10^9`

## 実行例

### 例1

```python
errors = [2, 1, 3]
threshold = 3
```

合計が `3` 以上になる連続部分配列は、

```python
[2, 1]
[2, 1, 3]
[1, 3]
[3]
```

の4つです。

出力:

```python
4
```

### 例2

```python
errors = [1, 1, 1]
threshold = 2
```

合計が `2` 以上になる連続部分配列は、

```python
[1, 1]
[1, 1, 1]
[1, 1]
```

の3つです。

出力:

```python
3
```

### 例3

```python
errors = [5, 1, 1]
threshold = 5
```

合計が `5` 以上になる連続部分配列は、

```python
[5]
[5, 1]
[5, 1, 1]
```

の3つです。

出力:

```python
3
```

## 実行結果

```
4
3
3
```
