## 問題文

### 重複しない最長イベント列

あなたはシステム上で発生したイベントを時系列順に確認しています。

配列 `events` には、発生したイベントIDを表す文字列が順番に格納されています。

**同じイベントIDを2回以上含まない連続部分配列のうち、長さが最大のもの** を求めてください。

その最大長を返してください。

### 制約

* `1 <= len(events) <= 10^5`
* `events[i]` は長さ1以上20以下の文字列

## 実行例

### 例1

```python
events = ["login", "deploy", "scale", "deploy", "alert"]
```

重複しない連続部分配列の最大の一例は

```python
["scale", "deploy", "alert"]
```

なので、出力は

```python
3
```

### 例2

```python
events = ["a", "b", "c", "d"]
```

すべて異なるため、配列全体を選べます。

出力は

```python
4
```

### 例3

```python
events = ["db", "db", "db"]
```

同じイベントしかないため、長さ1が最大です。

出力は

```python
1
```

## 実行結果

```
3
4
1
```
