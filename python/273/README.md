# No.273

## 問題文

**アナグラム開始位置の全列挙**

小文字英字のみからなる2つの文字列 `s` と `p` が与えられる。

`p` の文字を並べ替えてできる文字列（アナグラム）が、` s` の**どの位置から始まるか**をすべて求めよ。

つまり、すべてのインデックス `i` (`0 ≤ i ≤ len(s) - len(p)`) について、

* `s[i : i + len(p)]` が `p` のアナグラムであるなら、その `i` を答えに含める。

答えはインデックスのリストで返し、**昇順**に並んでいるものとする。

* `1 ≤ len(p) ≤ 10^5`
* `1 ≤ len(s) ≤ 10^5`
* 両方とも `'a'`〜`'z'` のみからなる
* 期待計算量: **O(len(s) + len(p))**
* 追加メモリ: **O(1)**（文字種は26固定なので）

## 実行例

**例1**

```text
入力:
s = "cbaebabacd"
p = "abc"

出力:
[0, 6]

説明:
s[0:3] = "cba" は "abc" のアナグラム
s[6:9] = "bac" も "abc" のアナグラム
他の位置では一致しない。
```

**例2**

```text
入力:
s = "abab"
p = "ab"

出力:
[0, 1, 2]

説明:
s[0:2] = "ab"
s[1:3] = "ba"
s[2:4] = "ab"
いずれも "ab" のアナグラム。
```

**例3**

```text
入力:
s = "aaaaa"
p = "aa"

出力:
[0, 1, 2, 3]
```

**例4**

```text
入力:
s = "abc"
p = "abcd"

出力:
[]

説明:
p の方が長いので一致する部分文字列は存在しない。
```

## 実行結果

```
[0, 6]
[0, 1, 2]
[0, 1, 2, 3]
[]
```
