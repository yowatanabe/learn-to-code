## 問題文

### 直近K件のログから中央値を求める

あなたはログ処理基盤で、直近 `k` 件のレスポンスタイムの中央値を監視しています。

整数配列 `times` と整数 `k` が与えられます。

各位置 `i` について、`times[i-k+1]` から `times[i]` までの **連続する `k` 件** の中央値を求め、配列で返してください。

`k` が偶数の場合、中央値は中央2つの平均値とします。

### 制約

* `1 <= k <= len(times) <= 10^5`
* `0 <= times[i] <= 10^9`

## 実行例

### 例1

```python
times = [1, 3, 2, 6, 4, 5]
k = 3
```

各ウィンドウの中央値は、

```python
[1, 3, 2] -> 2
[3, 2, 6] -> 3
[2, 6, 4] -> 4
[6, 4, 5] -> 5
```

出力:

```python
[2, 3, 4, 5]
```

### 例2

```python
times = [1, 2, 3, 4]
k = 2
```

各ウィンドウの中央値は、

```python
[1, 2] -> 1.5
[2, 3] -> 2.5
[3, 4] -> 3.5
```

出力:

```python
[1.5, 2.5, 3.5]
```

### 例3

```python
times = [5, 5, 5]
k = 1
```

出力:

```python
[5, 5, 5]
```

## 実行結果

```
[2, 3, 4, 5]
[1.5, 2.5, 3.5]
[5, 5, 5]
```
