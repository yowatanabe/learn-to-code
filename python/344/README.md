## 問題: 監視ログの「異常スパイク」を最小化せよ（Split Array Largest Sum）

SRE のメトリクス集計を想定します。

非負整数配列 `nums`（時系列の負荷）と整数 `m` が与えられます。

`nums` を **連続する m 個の部分配列** に分割するとき（各要素はちょうど1つの部分配列に属する）：

> 各部分配列の合計のうち **最大値** をできるだけ小さくしたい。
> その最小可能値を返してください。

### 制約（目安）

* `1 ≤ len(nums) ≤ 10^5`
* `0 ≤ nums[i] ≤ 10^9`（非負）
* `1 ≤ m ≤ len(nums)`

期待解法: **二分探索 + 貪欲チェック**（`O(n log(sum(nums)))`）

## 実行例

```python
# 例1
nums = [7, 2, 5, 10, 8]
m = 2
# 分割例: [7,2,5] と [10,8] -> 合計は 14 と 18, 最大は 18
# これが最小
# 出力: 18

# 例2
nums = [1, 2, 3, 4, 5]
m = 2
# 最小は 9（[1,2,3,4] と [5] では 10 だが、[1,2,3] と [4,5] で 6 と 9）
# 出力: 9

# 例3
nums = [1, 4, 4]
m = 3
# 3分割なら [1],[4],[4] -> 最大は 4
# 出力: 4
```
