## 問題文

### インシデント対応の最大利益

あなたはインシデント対応の改善施策を検討しています。

整数配列 `scores` が与えられ、`scores[i]` は `i` 日目に施策を開始した場合の評価値を表します。

あなたは **1回だけ開始日を選び、その後の別の日に終了する** とします。

開始日より後の終了日を選んだときの利益は、

```python
scores[end] - scores[start]
```

です。

得られる最大利益を返してください。
利益が出ない場合は `0` を返してください。

### 制約

* `2 <= len(scores) <= 10^5`
* `0 <= scores[i] <= 10^9`

## 実行例

### 例1

```python
scores = [7, 1, 5, 3, 6, 4]
```

`1` の日に開始し、`6` の日に終了すると利益は `5` です。

出力:

```python
5
```

### 例2

```python
scores = [7, 6, 4, 3, 1]
```

後の日ほど評価値が下がるため、利益は出ません。

出力:

```python
0
```

### 例3

```python
scores = [2, 4, 1, 8]
```

`1` の日に開始し、`8` の日に終了すると利益は `7` です。

出力:

```python
7
```

## 実行結果

```
5
0
7
```
