# No.388

## 問題文（LeetCode風・日本語）

### 問題：次に高いレイテンシが現れるまでの日数

SREとして、日ごとの最大レイテンシを監視しています。

整数配列 `latencies` が与えられ、`latencies[i]` は `i` 日目の最大レイテンシを表します。

各日について、**その日より後で、より大きいレイテンシが初めて現れるまで何日待つか**を求めてください。

もしそのような日が存在しない場合は `0` を設定してください。

長さ `n` の配列 `answer` を返してください。

* `1 <= len(latencies) <= 2 * 10^5`
* `0 <= latencies[i] <= 10^9`

**求めるもの**：各日についての待ち日数を格納した整数配列

## 実行例

例1:

* 入力: `latencies = [73, 74, 75, 71, 69, 72, 76, 73]`
* 出力: `[1, 1, 4, 2, 1, 1, 0, 0]`
* 説明:

  * 0日目の73より高い値は1日後の74 → `1`
  * 2日目の75より高い値は6日目の76 → `4`
  * 6日目の76より高い値は存在しない → `0`

例2:

* 入力: `latencies = [30, 40, 50, 60]`
* 出力: `[1, 1, 1, 0]`

例3:

* 入力: `latencies = [30, 60, 90]`
* 出力: `[1, 1, 0]`
