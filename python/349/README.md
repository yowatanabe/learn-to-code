## 問題: 最長の「ほぼ同一」区間（Longest Repeating Character Replacement）

文字列 `s`（英大文字のみ）と整数 `k` が与えられます。

あなたは文字列のある連続区間を選び、その区間内の文字を **高々 `k` 回** だけ別の文字に置き換えて、区間内の文字をすべて同じにできます。

> 置き換え後に「すべて同じ文字」になれる連続部分文字列のうち、**最大長**を返してください。

### 制約（目安）

* `1 ≤ len(s) ≤ 10^5`
* `0 ≤ k ≤ len(s)`
* `s` は `'A'`〜`'Z'`

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
s = "ABAB"
k = 2
# "ABAB" を全て A にするには B を2回置換 -> 長さ4
# 出力: 4

# 例2
s = "AABABBA"
k = 1
# 最長は "AABA" や "ABBA" などで長さ4
# 出力: 4

# 例3
s = "AAAA"
k = 0
# すでに全部同じ
# 出力: 4
```

## 実行結果

```

例1 出力: 4 (期待値: 4)
------------------------------------------------------------

------------------------------------------------------------
例3 入力: s='AAAA', k=0
例3 出力: 4 (期待値: 4)
------------------------------------------------------------
```
