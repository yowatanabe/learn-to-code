# No.101

### 問題: 配列の中で最も頻度の高い要素を見つける

整数の配列 `nums` が与えられます。
`nums` において最も多く出現する要素を返す関数 `most_frequent_element` を実装してください。
複数ある場合は、**最初に出現した最頻値** を返してください。

### 実行例

#### 例 1:

```
入力: nums = [1, 3, 1, 3, 2, 1]
出力: 1
```

#### 例 2:

```
入力: nums = [4, 4, 2, 2, 3]
出力: 4
```

#### 例 3:

```
入力: nums = [5]
出力: 5
```

### 実行結果

```text
1
4
5
```
