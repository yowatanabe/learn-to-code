# No.384

## 問題文（LeetCode風・日本語）

### 問題：最長のユニークイベント区間

あなたはログストリームを解析しています。

イベントIDの配列 `events` が与えられます。

**同じイベントIDを2回以上含まない連続部分配列のうち、最長の長さ**を求めてください。

* `1 <= len(events) <= 2 * 10^5`
* `events[i]` は整数（負でも可）

**求めるもの**：条件を満たす最長の長さ（整数）

## 実行例

例1:

* 入力:
  `events = [1, 2, 3, 1, 2, 3, 4]`
* 説明:
  `[1,2,3,4]` が最長で長さ4
* 出力:
  `4`

例2:

* 入力:
  `events = [5, 5, 5, 5]`
* 説明:
  同じ値しかないので最長は1
* 出力:
  `1`

例3:

* 入力:
  `events = [1, 2, 3, 4, 5]`
* 説明:
  すべて異なるので長さ5
* 出力:
  `5`

## 実行結果

```
4
1
5
```
