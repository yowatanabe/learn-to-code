## 問題文

### 障害通知の遅延を最小化する

あなたは `n` 個の監視ノードを持つ通知システムを運用しています。

各ノードは `0` から `n - 1` の番号で表されます。

配列 `edges` が与えられます。

```python
edges[i] = [from_node, to_node, delay]
```

これは `from_node` から `to_node` へ通知を送ると `delay` 秒かかることを表します。

開始ノード `start` から通知を送信したとき、**すべてのノードに通知が届くまでの最短時間**を返してください。

すべてのノードに通知できない場合は `-1` を返してください。

### 制約

* `1 <= n <= 10^5`
* `0 <= len(edges) <= 2 * 10^5`
* `0 <= from_node, to_node < n`
* `1 <= delay <= 10^9`
* `0 <= start < n`

## 実行例

### 例1

```python
n = 4
edges = [
    [0, 1, 1],
    [0, 2, 4],
    [1, 2, 2],
    [2, 3, 1]
]
start = 0
```

最短到達時間は以下です。

```python
0 -> 0 = 0
0 -> 1 = 1
0 -> 2 = 3
0 -> 3 = 4
```

すべてに届くまでの時間は `4` です。

出力:

```python
4
```

### 例2

```python
n = 3
edges = [
    [0, 1, 5]
]
start = 0
```

ノード `2` には届きません。

出力:

```python
-1
```

### 例3

```python
n = 1
edges = []
start = 0
```

開始ノードしか存在しないため、必要時間は `0` です。

出力:

```python
0
```

## 実行結果

```
4
-1
0
```
