## 問題: 0 と 1 の数が等しい最長部分配列

0 と 1 からなる整数配列 `nums` が与えられます。

次の条件を満たす「**連続した部分配列**」のうち、**最も長いものの長さ**を求めてください。

> 部分配列内の **0 の個数** と **1 の個数** が等しい

* 部分配列とは、配列の連続した区間のことです。
* 該当する部分配列が存在しない場合は `0` を返してください。

### 制約（目安）

* `1 ≤ len(nums) ≤ 10^5`
* `nums[i]` は必ず `0` または `1`

時間計算量 O(n)、追加メモリ O(n) 程度の解法を目指してください。

## 実行例

```python
# 例1
nums = [0, 1]
# 部分配列 [0, 1] は 0 と 1 が1つずつ → 長さ 2
# 出力: 2

# 例2
nums = [0, 1, 0]
# 条件を満たす部分配列:
# - [0, 1] (長さ 2)
# - [1, 0] (長さ 2)
# 最長は長さ 2
# 出力: 2

# 例3
nums = [0, 0, 1, 0, 0, 0, 1, 1]
# 条件を満たす最長部分配列の一例: [1, 0, 0, 0, 1, 1] (インデックス 2〜7)
# この部分配列の 0 の数 = 3, 1 の数 = 3 → 長さ 6
# 出力: 6

# 例4
nums = [0, 0, 0]
# どの部分配列も 0 と 1 の個数が等しくならない → 0
# 出力: 0
```

## 実行結果

```
例1の入力: [0, 1]
例1の結果: 2
例2の入力: [0, 1, 0]
例2の結果: 2
例3の入力: [0, 0, 1, 0, 0, 0, 1, 1]
例3の結果: 6
例4の入力: [0, 0, 0]
例4の結果: 0
```
