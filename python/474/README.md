## 問題文

### 障害イベントの最長連鎖

あなたは障害イベントの依存関係を分析しています。

`n` 個のイベントがあり、イベント番号は `0` から `n - 1` です。

配列 `relations` が与えられます。

```python
relations[i] = [before, after]
```

これは、イベント `before` が発生したあとにイベント `after` が発生しうることを表します。

イベントの依存関係は有向グラフで表され、循環は存在しないものとします。

**最も長いイベント連鎖に含まれるイベント数**を返してください。

### 制約

* `1 <= n <= 10^5`
* `0 <= len(relations) <= 2 * 10^5`
* `0 <= before, after < n`
* グラフに循環は存在しない

## 実行例

### 例1

```python
n = 5
relations = [[0, 1], [1, 2], [0, 3], [3, 4]]
```

最長連鎖は

```python
0 -> 1 -> 2
```

または

```python
0 -> 3 -> 4
```

なので、出力は

```python
3
```

### 例2

```python
n = 4
relations = [[0, 1], [1, 2], [2, 3]]
```

最長連鎖は

```python
0 -> 1 -> 2 -> 3
```

なので、出力は

```python
4
```

### 例3

```python
n = 3
relations = []
```

依存関係がないため、どのイベントも単独です。

出力:

```python
1
```

## 実行結果

```
3
4
1
```
