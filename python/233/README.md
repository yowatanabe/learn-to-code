# No.233

## 問題文

整数配列 `nums` が与えられます。

`nums[i] + nums[j] + nums[k] == 0` を満たす 重複のない三つ組 `[nums[i], nums[j], nums[k]]`（`i < j < k`）を すべて 求めてください。

解は任意の順序で構いませんが、同じ値の三つ組を重複して返してはいけません。

* `3 <= len(nums) <= 10^4`
* `-10^5 <= nums[i] <= 10^5`

## 実行例

```
入力: nums = [-1,0,1,2,-1,-4]
出力: [[-1,-1,2],[-1,0,1]]

入力: nums = [0,1,1]
出力: []

入力: nums = [0,0,0,0]
出力: [[0,0,0]]
```

## 実行結果

```
[[-1, -1, 2], [-1, 0, 1]]
[]
[[0, 0, 0]]
```
