# No.280

## 問題文

**最長連続稼働日数（Longest Consecutive Sequence）**

あるシステムの「稼働していた日」を表す整数配列 `days` が与えられる。

`days[i]` は **その日にシステムが稼働していたことを示す日付ID**（例えば通し番号や UNIX 日ベース）とする。

`days` はソートされておらず、重複を含む場合もある。

このとき、

> 値として **連続した日付** がどこまで続いているかを見たとき、
> 最長で何日連続して稼働していたか

を表す整数を返せ。

* ここで「連続」とは、`x, x+1, x+2, ...` のように **値が1ずつ増加する** ことを意味する。
* 日付の順番は入力配列のインデックスとは無関係とする（配列内での並び順はバラバラで良い）。

制約:

* `0 ≤ len(days) ≤ 2 * 10^5`
* `-10^9 ≤ days[i] ≤ 10^9`
* 期待計算量: **O(n)**（n = len(days)）
* 追加メモリ: **O(n)**

## 実行例

**例1**

```text
入力:
days = [100, 4, 200, 1, 3, 2]

出力:
4

説明:
値として連続しているのは [1, 2, 3, 4] で長さ4。
それより長い連続区間は存在しない。
```

**例2**

```text
入力:
days = [10, 5, 12, 3, 55, 11, 4, 13]

出力:
4

説明:
[10, 11, 12, 13] が長さ4の連続区間。
[3,4,5] は長さ3。
```

**例3**

```text
入力:
days = [1, 2, 2, 3]

出力:
3

説明:
重複は無視してよい。
値としては [1,2,3] が連続しているので長さ3。
```

**例4**

```text
入力:
days = []

出力:
0
```

**例5**

```text
入力:
days = [7, 7, 7]

出力:
1

説明:
値としては 7 だけなので、最大の連続区間は長さ1。
```

## 実行結果

```
4
4
3
0
1
```
