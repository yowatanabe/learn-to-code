# No.227

## 問題文

整数配列 `nums` と整数 `k` が与えられます。

`nums` の連続部分配列（空でない）のうち、合計がちょうど `k` になるものの個数を求めてください。

負の数を含んでも構いません。

* `1 <= len(nums) <= 2 * 10^5`
* `-10^9 <= nums[i] <= 10^9`
* `-10^14 <= k <= 10^14`

## 実行例

```
入力: nums = [1, 1, 1], k = 2
出力: 2
説明: [1,1]（前半）, [1,1]（後半）の2つ。

入力: nums = [1, 2, 3], k = 3
出力: 2
説明: [1,2], [3] の2つ。

入力: nums = [3, 4, 7, 2, -3, 1, 4, 2], k = 7
出力: 4
説明: [3,4], [7], [7,2,-3,1], [1,4,2] の4つ。

入力: nums = [-1, -1, 1], k = 0
出力: 1
説明: [-1, -1, 1] のみ。
```

## 実行結果

```
2
2
4
1
```
