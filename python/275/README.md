# No.275

## 問題文

**上位 K 個の頻出要素**

整数配列 `nums` と整数 `k` が与えられる。

`nums` の中で**出現回数が多い順に**見たときの「上位 `k` 個の異なる値」を配列で返せ。

* もし複数の値が同じ出現回数だった場合、それらの**相対的な順序はどれでもよい**（どれを先に返してもよい）。
* `k` は必ず `1 ≤ k ≤ 異なる値の個数` を満たす。

制約:

* `1 ≤ len(nums) ≤ 2 * 10^5`
* `-10^9 ≤ nums[i] ≤ 10^9`
* `1 ≤ k ≤` `nums` に含まれる**異なる値の個数**
* 期待計算量: **O(n log k)** または **O(n)** 程度
  ここでは `Counter` + 最小ヒープで **O(n log k)** を想定

## 実行例

**例1**

```text
入力:
nums = [1, 1, 1, 2, 2, 3]
k = 2

出力の一例:
[1, 2]

説明:
出現回数:
  1 → 3回
  2 → 2回
  3 → 1回
上位2個は 1 と 2。順番は [2,1] でもよい。
```

**例2**

```text
入力:
nums = [4, 4, 4, 5, 6, 6, 7]
k = 3

出力の一例:
[4, 6, 5]

説明:
出現回数:
  4 → 3回
  6 → 2回
  5 → 1回
  7 → 1回
上位3個は 4, 6, (5か7のどちらか)。どの順番でもよい。
```

**例3**

```text
入力:
nums = [1]
k = 1

出力:
[1]
```

**例4**

```text
入力:
nums = [1, 2, 3, 4]
k = 2

出力の一例:
[1, 2]

説明:
全て1回ずつしか出ていないので、どの2つを返してもよい。
```

## 実行結果

```
[2, 1]
[7, 4, 6]
[1]
[3, 4]
```
