# No.382

## 問題文（LeetCode風・日本語）

**問題：ログのスパイク検知（最長の“良い”区間）**

メトリクスの時系列 `a` が与えられます。

ある区間 `[l..r]` が **“良い区間”** であるとは、その区間の **最大値 - 最小値 <= k** を満たすことを指します。

配列 `a` と整数 `k` が与えられたとき、良い区間の **最大長** を返してください。

* `1 <= len(a) <= 2 * 10^5`
* `0 <= a[i] <= 10^9`
* `0 <= k <= 10^9`

**求めるもの**：良い区間の最大長（整数）

## 実行例

例1:

* 入力: `a = [8, 2, 4, 7]`, `k = 4`
* 説明: `[2,4]` は max-min=2 でOK、長さ2。最大は2。
* 出力: `2`

例2:

* 入力: `a = [10, 1, 2, 4, 7, 2]`, `k = 5`
* 説明: `[2,4,7,2]` は max=7, min=2, 差5でOK、長さ4。
* 出力: `4`

例3:

* 入力: `a = [4, 2, 2, 2, 4, 4, 2, 2]`, `k = 0`
* 説明: 同じ値だけの最長は `[2,2,2]` などで長さ3。
* 出力: `3`

## 実行結果

```
2
4
3
```
