## 問題文

### サーバー監視グリッドの孤立領域を数える

あなたはデータセンター内のサーバー配置を2次元グリッドで管理しています。

`grid` が与えられます。

* `"1"`: 稼働中のサーバー
* `"0"`: 空きラック

上下左右に隣接している `"1"` は、同じサーバーグループとみなします。

**稼働中サーバーのグループ数**を返してください。

### 制約

* `1 <= len(grid) <= 300`
* `1 <= len(grid[0]) <= 300`
* `grid[i][j]` は `"0"` または `"1"`

## 実行例

### 例1

```python
grid = [
    ["1", "1", "0", "0"],
    ["1", "0", "0", "1"],
    ["0", "0", "1", "1"]
]
```

左上に1つ、右下に1つのグループがあります。

出力:

```python
2
```

### 例2

```python
grid = [
    ["1", "1", "1"],
    ["0", "1", "0"],
    ["1", "0", "1"]
]
```

中央を通じて上側の `"1"` はつながっています。
左下と右下は別グループです。

出力:

```python
3
```

### 例3

```python
grid = [
    ["0", "0"],
    ["0", "0"]
]
```

稼働中サーバーがないため、出力は

```python
0
```

## 実行結果

```
2
3
0
```
