## 問題: 整数の合計が最小になる部分配列を探せ（Minimum Sum Subarray）

整数配列 `nums` と整数 `k` が与えられます。

あなたは **合計が最小となる** `k` 個の連続した要素からなる部分配列を求め、その合計を返してください。

### 制約（目安）

* `1 ≤ len(nums) ≤ 10^5`
* `-10^4 ≤ nums[i] ≤ 10^4`
* `1 ≤ k ≤ len(nums)`

## 実行例

```python
# 例1
nums = [2, 1, 3, 4, 1, 2]
k = 3
# 部分配列 [2, 1, 3] の合計が最小（2 + 1 + 3 = 6）
# 出力: 6

# 例2
nums = [5, -2, 3, 4, -1]
k = 2
# 部分配列 [-2, 3] の合計が最小（-2 + 3 = 1）
# 出力: 1

# 例3
nums = [1, 2, 3]
k = 3
# 部分配列 [1, 2, 3] の合計が最小（1 + 2 + 3 = 6）
# 出力: 6

# 例4
nums = [-1, -2, -3, -4]
k = 2
# 部分配列 [-3, -4] の合計が最小（-3 + -4 = -7）
# 出力: -7
```
