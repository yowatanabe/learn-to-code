# No.310

## 問題: 文字列の並べ替えをグループ化せよ（Group Anagrams）

文字列配列 `strs` が与えられます。

> **アナグラム（並べ替えで一致する文字列）** 同士を同じグループにまとめて返してください。

* 出力は「グループの配列」で返します。
* 各グループ内の順序、グループ同士の順序は問いません。
* 文字列は英小文字 `'a'`〜`'z'` のみからなるとします。

### 制約（目安）

* `1 ≤ len(strs) ≤ 10^4`
* `0 ≤ len(strs[i]) ≤ 100`

時間計算量はだいたい `O(N * L log L)`（各文字列をソートする場合）または `O(N * L)`（文字頻度をキーにする場合）を目指してください。

## 実行例

```python
# 例1
strs = ["eat","tea","tan","ate","nat","bat"]
# 出力例:
# [["eat","tea","ate"], ["tan","nat"], ["bat"]]

# 例2
strs = [""]
# 出力: [[""]]

# 例3
strs = ["a"]
# 出力: [["a"]]
```

## 実行結果

```
例1 入力: ['eat', 'tea', 'tan', 'ate', 'nat', 'bat']
例1 出力: [['eat', 'tea', 'ate'], ['tan', 'nat'], ['bat']]
----------------------------------------
例2 入力: ['']
例2 出力: [['']]
----------------------------------------
例3 入力: ['a']
例3 出力: [['a']]
----------------------------------------
```
