## 問題文

### 同じ累積変化量になる最長期間

あなたは、あるシステムの1日ごとのリクエスト増減を監視しています。

整数配列 `changes` が与えられ、`changes[i]` は `i` 日目のリクエスト数の増減を表します。

* 正の値: リクエスト数が増えた
* 負の値: リクエスト数が減った
* `0`: 変化なし

**合計値が 0 になる連続部分配列のうち、長さが最大のもの** を求めてください。

その最大長を返してください。

## 実行例

### 例1

```python
changes = [1, -1, 3, 2, -2, -3, 4]
```

`[1, -1, 3, 2, -2, -3]` の合計は `0` です。

長さは

```python
6
```

なので、出力は

```python
6
```

### 例2

```python
changes = [2, -2, 1, -1, 3]
```

`[2, -2, 1, -1]` の合計は `0` です。

長さは

```python
4
```

なので、出力は

```python
4
```

### 例3

```python
changes = [1, 2, 3]
```

合計が `0` になる連続部分配列は存在しません。

出力は

```python
0
```

## 実行結果

```
6
4
0
```
