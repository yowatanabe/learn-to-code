## 問題: ネットワーク遅延時間（Network Delay Time）

`n` 台のノード（`1` から `n`）からなる有向グラフがあります。

辺情報 `times` が与えられ、`times[i] = [u, v, w]` は「ノード `u` から `v` へ重み `w` の有向辺がある」ことを表します。

開始ノード `k` から信号を送るとき、

> すべてのノードが信号を受信するまでに必要な **最小時間** を返してください。
> もし到達できないノードがある場合は `-1` を返してください。

### 制約（目安）

* `1 ≤ n ≤ 10^5`
* `1 ≤ len(times) ≤ 2 * 10^5`
* `1 ≤ u, v ≤ n`
* `0 ≤ w ≤ 10^4`
* `u != v`

**期待解法**: ダイクストラ法（最短経路） `O((E+V) log V)`

## 実行例

```python
# 例1
times = [[2,1,1],[2,3,1],[3,4,1]]
n = 4
k = 2
# 2->1 は1, 2->3 は1, 2->3->4 は2
# 全ノードが受信するまでの時間は max(1,1,2)=2
# 出力: 2

# 例2
times = [[1,2,1]]
n = 2
k = 2
# 2 から 1 に到達できない
# 出力: -1

# 例3
times = [[1,2,1],[2,3,2],[1,3,4]]
n = 3
k = 1
# 1->3 は直接だと4だが 1->2->3 で3が最短
# 全ノード受信まで max(dist)=3
# 出力: 3
```
