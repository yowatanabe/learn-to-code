# No.107

### 問題: リスト内の要素の合計が偶数か奇数かを判定する

整数のリスト `nums` が与えられます。
リスト内の要素の合計が **偶数なら "even"**、**奇数なら "odd"** を返す関数 `sum_even_or_odd` を実装してください。

### 実行例

#### 例 1:

```
入力: nums = [1, 2, 3]
出力: "even"
説明: 合計は 1+2+3=6（偶数）
```

#### 例 2:

```
入力: nums = [1, 2, 4]
出力: "odd"
説明: 合計は 1+2+4=7（奇数）
```

#### 例 3:

```
入力: nums = []
出力: "even"
説明: 合計は 0（偶数）
```

### 実行結果

```text
even
odd
even
```
