## 問題文

### ログ転送経路の最大信頼度

あなたは複数のサーバー間でログを転送しています。

サーバーは `0` から `n - 1` の番号で表されます。

配列 `edges` と `probabilities` が与えられます。

```python
edges[i] = [a, b]
probabilities[i] = p
```

これは、サーバー `a` と `b` の間でログを転送でき、その転送成功確率が `p` であることを表します。
経路全体の成功確率は、通った辺の成功確率の積です。

開始サーバー `start` から終了サーバー `goal` までログを転送するとき、
**成功確率が最大になる経路の確率**を返してください。

到達できない場合は `0.0` を返してください。

### 制約

* `1 <= n <= 10^4`
* `0 <= len(edges) <= 2 * 10^4`
* `len(edges) == len(probabilities)`
* `0 <= a, b < n`
* `0.0 <= probabilities[i] <= 1.0`
* `0 <= start, goal < n`

## 実行例

### 例1

```python
n = 3
edges = [[0, 1], [1, 2], [0, 2]]
probabilities = [0.5, 0.5, 0.2]
start = 0
goal = 2
```

候補経路は以下です。

```python
0 -> 2       成功確率 0.2
0 -> 1 -> 2  成功確率 0.5 * 0.5 = 0.25
```

最大は `0.25` なので、出力は

```python
0.25
```

### 例2

```python
n = 3
edges = [[0, 1]]
probabilities = [0.5]
start = 0
goal = 2
```

`2` には到達できません。

出力:

```python
0.0
```

### 例3

```python
n = 4
edges = [[0, 1], [1, 2], [2, 3], [0, 3]]
probabilities = [0.9, 0.9, 0.9, 0.5]
start = 0
goal = 3
```

```python
0 -> 1 -> 2 -> 3
```

の成功確率は `0.729` で、直接経路 `0 -> 3` の `0.5` より大きいです。

出力:

```python
0.729
```
