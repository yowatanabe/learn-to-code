## 問題文

### サーバー負荷をK個のグループに分割する最小最大値

あなたはログ処理ジョブを `k` 個のワーカーに割り当てます。

整数配列 `loads` が与えられ、`loads[i]` は `i` 番目のジョブの負荷を表します。

ジョブの順番は変えられません。

連続するジョブをまとめて1つのワーカーに割り当てます。

`loads` を **最大で `k` 個の連続グループ** に分割したとき、
各グループの合計負荷の最大値をできるだけ小さくしてください。

その最小値を返してください。

### 制約

* `1 <= len(loads) <= 10^5`
* `1 <= loads[i] <= 10^9`
* `1 <= k <= len(loads)`

## 実行例

### 例1

```python
loads = [7, 2, 5, 10, 8]
k = 2
```

最適な分割は

```python
[7, 2, 5], [10, 8]
```

各合計は `14` と `18` なので、最大値は `18` です。

出力:

```python
18
```

### 例2

```python
loads = [1, 2, 3, 4, 5]
k = 2
```

最適な分割は

```python
[1, 2, 3], [4, 5]
```

出力:

```python
9
```

### 例3

```python
loads = [3, 3, 3]
k = 3
```

各ジョブを1つずつ割り当てられます。

出力:

```python
3
```

## 実行結果

```
18
9
3
```
