## 問題文

### 最小の監視ウィンドウ文字列

あなたはログ文字列 `log` の中から、必要なキーワード文字列 `pattern` をすべて含む最短区間を探しています。

文字列 `log` と `pattern` が与えられます。
`pattern` に含まれる各文字を、**出現回数も含めてすべて含む** `log` の連続部分文字列のうち、最も短いものを返してください。

存在しない場合は空文字 `""` を返してください。

## 実行例

### 例1

```python
log = "ADOBECODEBANC"
pattern = "ABC"
```

`"BANC"` は `A`, `B`, `C` をすべて含む最短部分文字列です。

出力:

```python
"BANC"
```

### 例2

```python
log = "aa"
pattern = "aa"
```

`"aa"` がそのまま条件を満たします。

出力:

```python
"aa"
```

### 例3

```python
log = "a"
pattern = "aa"
```

`"a"` は `a` を1回しか含まないため条件を満たせません。

出力:

```python
""
```

## 実行結果

```
BANC
aa

```
