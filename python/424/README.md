## 問題文

### 次にCPU使用率が上がるまでの日数

あなたは複数日のCPU使用率を監視しています。

整数配列 `usages` が与えられ、`usages[i]` は `i` 日目のCPU使用率を表します。

各日について、**その日より後で、より大きいCPU使用率が初めて現れるまで何日待つ必要があるか** を求めてください。もしそのような日が存在しない場合は `0` としてください。

長さ `n` の整数配列を返してください。

### 制約

* `1 <= len(usages) <= 10^5`
* `0 <= usages[i] <= 100`

## 実行例

### 例1

```python
usages = [30, 40, 35, 50, 45, 60]
```

* 0日目の `30` は、1日後の `40` がより大きいので `1`
* 1日目の `40` は、2日後の `50` がより大きいので `2`
* 2日目の `35` は、1日後の `50` がより大きいので `1`
* 3日目の `50` は、2日後の `60` がより大きいので `2`
* 4日目の `45` は、1日後の `60` がより大きいので `1`
* 5日目の `60` は、その後により大きい値がないので `0`

出力:

```python
[1, 2, 1, 2, 1, 0]
```

### 例2

```python
usages = [70, 60, 50, 40]
```

どの日についても、その後により大きいCPU使用率は現れません。

出力:

```python
[0, 0, 0, 0]
```

### 例3

```python
usages = [20, 20, 30, 10, 40]
```

出力:

```python
[2, 1, 2, 1, 0]
```

## 実行結果

```
[1, 2, 1, 2, 1, 0]
[0, 0, 0, 0]
[2, 1, 2, 1, 0]
```
