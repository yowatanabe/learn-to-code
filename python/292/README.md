## 問題: 区間をマージせよ（Merge Intervals）

半開区間ではなく、**閉区間**としての整数区間配列 `intervals` が与えられます。

各要素 `intervals[i] = [start_i, end_i]` は

> `start_i` 以上 `end_i` 以下の連続した整数区間

を表すものとします（`start_i <= end_i`）。

区間同士が **重なっている場合**、それらを 1 つの区間にマージしてください。

> 「重なる」とは、2つの区間 `[a, b]` と `[c, d]` について
> `max(a, c) <= min(b, d)` が成り立つこととします。

すべての区間をマージした結果を返してください。

戻り値も同じ形式（`[start, end]` のリスト）とします。

### 制約（目安）

* `1 ≤ len(intervals) ≤ 10^4`
* `-10^5 ≤ start_i ≤ end_i ≤ 10^5`

時間計算量 O(n log n)（ソートがボトルネック）を目指してください。

## 実行例

```python
# 例1
intervals = [[1,3],[2,6],[8,10],[15,18]]
# [1,3] と [2,6] は重なるので [1,6] にマージされる
# 他は重ならない
# 出力: [[1,6],[8,10],[15,18]]

# 例2
intervals = [[1,4],[4,5]]
# [1,4] と [4,5] は端点で接しており、max(1,4) = 4, min(4,5) = 4 なので
# max <= min が成り立つ → 重なるとみなす
# 出力: [[1,5]]

# 例3
intervals = [[1,2]]
# 区間が1つだけなので、そのまま
# 出力: [[1,2]]
```

## 実行結果

```
例1の入力: [[1, 3], [2, 6], [8, 10], [15, 18]]
例1の結果: [[1, 6], [8, 10], [15, 18]]
例2の入力: [[1, 4], [4, 5]]
例2の結果: [[1, 5]]
例3の入力: [[1, 2]]
例3の結果: [[1, 2]]
```
