## 問題文

### ログ保存容量を超えない最大連続期間

あなたは日ごとのログ増加量を監視しています。

整数配列 `logs` が与えられ、`logs[i]` は `i` 日目に増えたログ容量を表します。

整数 `limit` が与えられたとき、**合計ログ容量が `limit` 以下になる連続部分配列のうち、長さが最大のもの**を返してください。

### 制約

* `1 <= len(logs) <= 10^5`
* `1 <= logs[i] <= 10^4`
* `1 <= limit <= 10^9`

## 実行例

### 例1

```python
logs = [2, 1, 3, 2, 1]
limit = 5
```

条件を満たす最長区間の一例は、

```python
[1, 3, 1]
```

ではなく、実際には

```python
[2, 1]
```

や

```python
[3, 2]
```

などの長さ `2` です。

出力:

```python
2
```

### 例2

```python
logs = [1, 1, 1, 1]
limit = 3
```

`[1, 1, 1]` の長さ `3` が最大です。

出力:

```python
3
```

### 例3

```python
logs = [6, 1, 2, 3]
limit = 5
```

`[1, 2]` または `[2, 3]` が条件を満たし、長さは `2` です。

出力:

```python
2
```

## 実行結果

```
2
3
2
```
