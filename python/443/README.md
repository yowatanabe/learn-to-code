# No.443

## 問題文

### 最も早く全サービスが復旧する時刻

あなたは複数サービスの復旧イベントを監視しています。整数 `n` 個のサービスがあり、サービス番号は `0` から `n - 1` です。

配列 `events` が与えられます。

```python
events[i] = [time, service]
```

これは時刻 `time` に `service` が復旧したことを表します。

同じサービスが複数回登場することもあります。すべてのサービスが少なくとも1回復旧済みになった**最も早い時刻**を返してください。

すべてのサービスが復旧しない場合は `-1` を返してください。

### 制約

* `1 <= n <= 10^5`
* `1 <= len(events) <= 10^5`
* `0 <= time <= 10^9`
* `0 <= service < n`

## 実行例

### 例1

```python
n = 3
events = [[5, 0], [2, 1], [8, 2], [3, 0]]
```

時刻順に見ると、

```python
[2, 1]
[3, 0]
[5, 0]
[8, 2]
```

時刻 `8` でサービス `0`, `1`, `2` がすべて復旧済みになります。

出力は

```python
8
```

### 例2

```python
n = 4
events = [[1, 0], [2, 1], [3, 2]]
```

サービス `3` が一度も復旧しないため、出力は

```python
-1
```

### 例3

```python
n = 2
events = [[10, 1], [5, 1], [7, 0]]
```

時刻順に見ると、

```python
[5, 1]
[7, 0]
[10, 1]
```

時刻 `7` ですべてのサービスが復旧済みになります。

出力は

```python
7
```

## 実行結果

```
8
-1
7
```
