## 問題: 最長「連続稼働」区間（Longest Subarray with At Most Two Distinct）

SRE のログを想定します。配列 `events` は時系列で観測されたイベント種別（整数ID）を表します。

あなたは「同じ2種類までのイベントだけが出ている連続区間」を **安定稼働区間** と定義します。

整数配列 `events` が与えられたとき、

> 含まれる異なる値の種類数が **高々2** である連続部分配列のうち、**最大長**を返してください。

### 制約（目安）

* `1 ≤ len(events) ≤ 10^5`
* `-10^9 ≤ events[i] ≤ 10^9`

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
events = [1, 2, 1]
# 全体が {1,2} の2種類なので長さ3
# 出力: 3

# 例2
events = [0, 1, 2, 2]
# 最長は [1,2,2]（種類 {1,2}）で長さ3
# 出力: 3

# 例3
events = [1, 2, 3, 2, 2]
# 最長は [2,3,2,2]（種類 {2,3}）で長さ4
# 出力: 4
```

## 実行結果

```
例1 入力: events=[1, 2, 1]
例1 出力: 3 (期待値: 3)
------------------------------------------------------------
例2 入力: events=[0, 1, 2, 2]
例2 出力: 3 (期待値: 3)
------------------------------------------------------------
例3 入力: events=[1, 2, 3, 2, 2]
例3 出力: 4 (期待値: 4)
------------------------------------------------------------
```
