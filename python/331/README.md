## 問題: ランタイム監視の「スパイク区間」を数えよ（Number of Subarrays with Sum = k）

SRE のメトリクス監視を想定します。

1分ごとのリクエスト増減を `+1 / -1 / 0` のような整数で表した配列 `nums` が与えられます。

整数配列 `nums` と整数 `k` が与えられたとき、

> 合計が **ちょうど `k`** になる **連続部分配列**の個数を返してください。

### 制約（目安）

* `1 ≤ len(nums) ≤ 2 * 10^5`
* `-10^4 ≤ nums[i] ≤ 10^4`（負の数を含む）
* `-10^9 ≤ k ≤ 10^9`

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
nums = [1, 1, 1]
k = 2
# [1,1] が2回（インデックス (0..1), (1..2)）
# 出力: 2

# 例2
nums = [1, 2, 3]
k = 3
# [1,2], [3]
# 出力: 2

# 例3
nums = [1, -1, 0]
k = 0
# 部分配列の合計0: [1,-1], [0], [1,-1,0]
# 出力: 3
```

## 実行結果

```
例1 入力: nums=[1, 1, 1], k=2
例1 出力: 2
--------------------------------------------------
例2 入力: nums=[1, 2, 3], k=3
例2 出力: 2
--------------------------------------------------
例3 入力: nums=[1, -1, 0], k=0
例3 出力: 3
--------------------------------------------------
```
