## 問題：文字列の並び替えによる等価判定

文字列 `s` と `t` が与えられます。`t` が `s` の文字を並び替えることで得られる文字列（アナグラム）であるかを判定してください。大文字と小文字は区別し、英字のみが含まれていると仮定します。

アナグラムとは、単語やフレーズを構成する文字をすべて使って別の単語やフレーズに並び替えたものです。

### 実行例

#### 例 1:

```
入力: s = "listen", t = "silent"
出力: True
```

#### 例 2:

```
入力: s = "hello", t = "bello"
出力: False
```

#### 例 3:

```
入力: s = "anagram", t = "nagaram"
出力: True
```

### 実行結果

```text
True
False
True
```
