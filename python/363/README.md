## 問題: K個までのログを使って目標スコアに到達せよ（Shortest Subarray with Sum at Least K）

整数配列 `nums`（負の数を含む可能性あり）と整数 `k` が与えられます。

> 合計が **k以上** となる **連続部分配列** のうち、長さの最小値を返してください。
> 条件を満たす部分配列が存在しない場合は `-1` を返してください。

### 注意

- `nums` には **負の数** が含まれる可能性があります。
- そのため、単純なスライディングウィンドウでは解けません。

### 制約（目安）

- `1 ≤ len(nums) ≤ 10^5`
- `-10^9 ≤ nums[i] ≤ 10^9`
- `1 ≤ k ≤ 10^9`

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
nums = [2, -1, 2]
k = 3
# 全体 [2,-1,2] の合計は 3 なので長さ 3
# 出力: 3

# 例2
nums = [1, 2]
k = 4
# どの部分配列でも合計 4 以上にならない
# 出力: -1

# 例3
nums = [1, -1, 5, -2, 3]
k = 3
# [5] の合計は 5 で長さ 1 が最小
# 出力: 1

# 例4
nums = [-2, -1, 2, 1]
k = 1
# [1] で長さ 1
# 出力: 1
````

