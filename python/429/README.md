## 問題文

### 連続したメンテナンス時間帯の統合

あなたは複数チームが登録したメンテナンス時間帯をまとめて管理しています。

各メンテナンス時間帯は `[start, end]` という2要素の配列で表され、`start` 以上 `end` 以下の時間を意味します。

整数の2次元配列 `intervals` が与えられたとき、**重なっている、または端で接している時間帯を統合した結果** を返してください。

返り値は、統合後の区間を `start` の昇順に並べた2次元配列としてください。

### 制約

* `1 <= len(intervals) <= 10^5`
* `0 <= start <= end <= 10^9`

## 実行例

### 例1

```python
intervals = [[1, 3], [2, 6], [8, 10], [15, 18]]
```

`[1, 3]` と `[2, 6]` は重なっているので統合できます。

出力は

```python
[[1, 6], [8, 10], [15, 18]]
```

### 例2

```python
intervals = [[1, 4], [4, 5]]
```

`[1, 4]` と `[4, 5]` は端で接しているので統合します。

出力は

```python
[[1, 5]]
```

### 例3

```python
intervals = [[5, 7], [1, 2], [2, 4]]
```

まず開始時刻順に並べると

```python
[[1, 2], [2, 4], [5, 7]]
```

`[1, 2]` と `[2, 4]` は端で接しているので統合できます。

出力は

```python
[[1, 4], [5, 7]]
```

## 実行結果

```
[[1, 6], [8, 10], [15, 18]]
[[1, 5]]
[[1, 4], [5, 7]]
```
