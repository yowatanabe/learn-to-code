# No.127

## 問題：2つのリストに共通する要素を見つける

2つの整数のリスト `list1` と `list2` が与えられます。
この2つのリストに共通して存在する要素を、**昇順で重複なく** リストで返してください。

## 実行例

### 例 1:

```
入力: list1 = [1, 2, 2, 3], list2 = [2, 3, 4]
出力: [2, 3]
```

### 例 2:

```
入力: list1 = [5, 6, 7], list2 = [1, 2, 3]
出力: []
```

### 例 3:

```
入力: list1 = [1, 2, 3], list2 = [3, 2, 1]
出力: [1, 2, 3]
```

### 実行結果

```text
[2, 3]
[]
[1, 2, 3]
```
