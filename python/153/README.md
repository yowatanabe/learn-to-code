# No.153

## 問題文

**文字列の圧縮**

英小文字のみからなる文字列 `s` が与えられます。同じ文字が連続している部分を、**文字 + 連続回数** の形式で圧縮した文字列を返してください。ただし、圧縮後の長さが**元の文字列より短くない場合**は、**元の文字列をそのまま返してください**。

### 条件

* 1 <= s.length <= 1000
* `s` は英小文字のみで構成される

## 実行例

### 例 1

```
入力: s = "aabcccccaaa"
出力: "a2b1c5a3"
```

### 例 2

```
入力: s = "abc"
出力: "abc"
```

### 例 3

```
入力: s = "aabb"
出力: "aabb"
```

### 実行結果

```text
a2b1c5a3
abc
aabb
```
