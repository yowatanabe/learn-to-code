# No.391

## 問題文

### 問題：最も頻出するイベントID 上位 K 個

あなたはSREとして、大量のログからよく出現するイベントIDを調べています。

整数配列 `events` と整数 `k` が与えられます。

`events` の中で、**出現回数が多い順に上位 `k` 個の要素**を返してください。

* 返す順序は問いません
* `1 <= k <=` 異なる要素数
* `1 <= len(events) <= 2 * 10^5`
* `-10^9 <= events[i] <= 10^9`

**求めるもの**：出現回数上位 `k` 個の整数を格納した配列

## 実行例

例1:

* 入力: `events = [1,1,1,2,2,3]`, `k = 2`
* 説明:

  * `1` は3回
  * `2` は2回
  * `3` は1回
  * 上位2個は `1` と `2`
* 出力: `[1, 2]`

例2:

* 入力: `events = [4]`, `k = 1`
* 説明:

  * 要素は `4` だけ
* 出力: `[4]`

例3:

* 入力: `events = [5,5,6,6,6,7,7,7,7]`, `k = 2`
* 説明:

  * `7` は4回
  * `6` は3回
  * `5` は2回
  * 上位2個は `7` と `6`
* 出力: `[7, 6]`

## 実行結果

```
[1, 2]
[4]
[7, 6]
```
