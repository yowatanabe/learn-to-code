## 問題: 期限付きジョブを最大数こなせ（Schedule Maximum Jobs）

`n` 個のジョブがあり、ジョブ `i` は `jobs[i] = [duration, deadline]` で表されます。

* `duration` はそのジョブに必要な時間
* `deadline` はそのジョブを **deadline までに完了**しなければならない期限

ジョブは **同時に実行できず**、1つずつ順番に実行します。開始時刻は `0` とします。

> 期限を守って完了できるジョブ数の最大値を返してください。

### 制約（目安）

* `1 ≤ n ≤ 10^5`
* `1 ≤ duration, deadline ≤ 10^9`

時間計算量 **O(n log n)** を目指してください。

## 実行例

```python
# 例1
jobs = [[3,5],[2,6],[1,3]]
# 3つ全て入れても合計6で、最後のdeadline=6に収まる
# 出力: 3

# 例2
jobs = [[5,5],[4,6],[2,6]]
# (2,6)->(4,6) は合計6でOK、(5,5)は入れられない
# 出力: 2

# 例3
jobs = [[1,2],[2,2],[3,3]]
# 期限を守れるのは (1,2) のみ
# 出力: 1
```

## 実行結果

```
例1 入力: jobs=[[1, 3], [3, 5], [2, 6]]
例1 出力: 3 (期待値: 2)
------------------------------------------------------------
例2 入力: jobs=[[5, 5], [4, 6], [2, 6]]
例2 出力: 2 (期待値: 2)
------------------------------------------------------------
例3 入力: jobs=[[1, 2], [2, 2], [3, 3]]
例3 出力: 1 (期待値: 1)
------------------------------------------------------------
```
