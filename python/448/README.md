## 問題文

### 最小コストで監視エージェントを配置する

あなたは `n` 台のサーバーに監視エージェントを配置します。

各サーバー `i` にエージェントを配置するコストは `costs[i]` です。

ただし、監視エージェントを配置しないサーバーがあっても、**隣接するサーバーのどちらかにエージェントがあれば監視できる** とします。

つまり、各サーバー `i` は以下のいずれかを満たす必要があります。

* サーバー `i` 自身にエージェントがある
* サーバー `i - 1` にエージェントがある
* サーバー `i + 1` にエージェントがある

すべてのサーバーを監視するための最小コストを返してください。

### 制約

* `1 <= len(costs) <= 10^5`
* `1 <= costs[i] <= 10^4`

## 実行例

### 例1

```python
costs = [5, 1, 4]
```

サーバー `1` にだけ配置すれば、サーバー `0`, `1`, `2` をすべて監視できます。

出力:

```python
1
```

### 例2

```python
costs = [3, 2, 10, 1, 4]
```

サーバー `1` と `3` に配置すると、すべて監視できます。

合計コストは `2 + 1 = 3` です。

出力:

```python
3
```

### 例3

```python
costs = [8, 6, 7, 5]
```

サーバー `1` と `3` に配置すると、すべて監視できます。

合計コストは `6 + 5 = 11` です。

出力:

```python
11
```

## 実行結果

```
1
3
11
```
