## 問題: ログ集計のピーク値を求めよ（Maximum Sum of Two Non-Overlapping Subarrays）

整数配列 `nums` と、2つの整数 `firstLen`, `secondLen` が与えられます。

あなたは `nums` から、

- 長さが `firstLen` の連続部分配列 1つ
- 長さが `secondLen` の連続部分配列 1つ

を選びたいです。
ただし、2つの部分配列は **重ならない**（インデックスが交差しない）必要があります。

> 選んだ2つの部分配列の合計和を最大化した値を返してください。

### 制約（目安）

- `1 ≤ len(nums) ≤ 10^5`
- `-10^9 ≤ nums[i] ≤ 10^9`
- `1 ≤ firstLen, secondLen ≤ len(nums)`
- `firstLen + secondLen ≤ len(nums)`

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
nums = [0,6,5,2,2,5,1,9,4]
firstLen = 1
secondLen = 2
# 最適: 長さ1の [9] と 長さ2の [6,5] を選ぶ => 9 + 11 = 20
# 出力: 20

# 例2
nums = [3,8,1,3,2,1,8,9,0]
firstLen = 3
secondLen = 2
# 最適: [3,8,1] (12) と [8,9] (17) => 合計 29
# 出力: 29

# 例3
nums = [2,1,5,6,0,9,5,0,3,8]
firstLen = 4
secondLen = 3
# 出力: 31
```
