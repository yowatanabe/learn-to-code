## 問題文

### 最短で全サービスのヘルスチェックを完了する

あなたは `n` 個のサービスのヘルスチェックを実行します。

サービスは `0` から `n - 1` の番号で表されます。

配列 `checks` が与えられます。

```python
checks[i] = [service, duration]
```

これは、`service` のヘルスチェックに `duration` 秒かかることを表します。

同じサービスが複数回登場する場合があります。

その場合、そのサービスについては **最も短い duration のチェックだけを採用** できます。

すべてのサービスを1回ずつチェックするとき、必要な最小合計時間を返してください。

すべてのサービスをチェックできない場合は `-1` を返してください。

### 制約

* `1 <= n <= 10^5`
* `1 <= len(checks) <= 2 * 10^5`
* `0 <= service < n`
* `1 <= duration <= 10^9`

## 実行例

### 例1

```python
n = 3
checks = [[0, 5], [1, 3], [2, 4], [0, 2]]
```

サービス `0` は `5` 秒と `2` 秒の候補があるため、`2` 秒を選びます。

合計は

```python
2 + 3 + 4 = 9
```

出力:

```python
9
```

### 例2

```python
n = 4
checks = [[0, 1], [1, 2], [2, 3]]
```

サービス `3` のチェック候補がありません。

出力:

```python
-1
```

### 例3

```python
n = 2
checks = [[0, 10], [0, 5], [1, 7]]
```

サービス `0` は `5` 秒、サービス `1` は `7` 秒を選びます。

出力:

```python
12
```

## 実行結果

```
9
-1
12
```
