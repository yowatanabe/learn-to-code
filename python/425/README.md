## 問題文

### ログ保持期間内の最大連続日数

あなたはあるシステムのログ保存容量を監視しています。

配列 `sizes` には、1日ごとのログ増加量（GB）が記録されています。

整数配列 `sizes` と整数 `capacity` が与えられたとき、**連続する日数の合計ログ増加量が `capacity` 以下となる期間のうち、最も長い日数** を求めてください。

その最大日数を返してください。

### 制約

* `1 <= len(sizes) <= 10^5`
* `1 <= sizes[i] <= 10^4`
* `1 <= capacity <= 10^9`

## 実行例

### 例1

```python
sizes = [4, 2, 1, 3, 2]
capacity = 6
```

* `[4, 2]` の合計は `6`
* `[2, 1, 3]` の合計も `6`

どちらも条件を満たし、長さは `2` と `3` です。
最長は `[2, 1, 3]` なので、出力は

```python
3
```

### 例2

```python
sizes = [1, 1, 1, 1, 1]
capacity = 3
```

連続する3日分までなら合計は `3` 以下です。
出力は

```python
3
```

### 例3

```python
sizes = [7, 2, 3]
capacity = 5
```

* `7` は単体でも `capacity` を超える
* `[2, 3]` は合計 `5`

したがって出力は

```python
2
```

## 実行結果

```
3
3
2
```
