## 問題文

### 設定ファイルの重複ブロックを取り除く

あなたは設定ファイルを行単位で管理しています。

文字列配列 `lines` が与えられ、`lines[i]` は `i` 行目の設定を表します。

同じ設定行が複数回出てくる場合、**最初に出てきた行だけを残し、それ以降の重複行は削除**したいです。

ただし、残す行の順番は元の順番を保つ必要があります。

重複を取り除いた後の配列を返してください。

### 制約

* `1 <= len(lines) <= 10^5`
* `1 <= len(lines[i]) <= 100`
* `lines[i]` は英数字と記号を含む文字列

## 実行例

### 例1

```python
lines = ["port=80", "host=web", "port=80", "timeout=30", "host=web"]
```

`"port=80"` と `"host=web"` は2回出ていますが、最初の1回だけ残します。

出力:

```python
["port=80", "host=web", "timeout=30"]
```

### 例2

```python
lines = ["a", "b", "c"]
```

重複がないため、そのまま返します。

出力:

```python
["a", "b", "c"]
```

### 例3

```python
lines = ["x", "x", "x"]
```

最初の `"x"` だけ残します。

出力:

```python
["x"]
```

## 実行結果

```
['port=80', 'host=web', 'timeout=30']
['a', 'b', 'c']
['x']
```
