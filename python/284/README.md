# No.284

## 問題: 繰り返し文字を含まない最長部分文字列の長さ

文字列 `s` が与えられます。

「**同じ文字が二度以上出てこない連続部分文字列**」のうち、**最も長いものの長さ**を求めてください。

* 部分文字列とは、文字列中の連続した区間のことです。
* 英小文字・英大文字・数字・記号など、どんな文字が含まれていてもよいものとします。

### 制約（目安）

* `0 ≤ len(s) ≤ 10^5`
* 文字は ASCII 文字と仮定してよい（面接では「Unicode まで考えるか？」と聞かれることもあります）

時間計算量 O(n) の解法を意識してください（全探索 O(n^2) は TLE 想定）。

## 実行例

```python
# 例1
s = "abcabcbb"
# "abc" が条件を満たす最長部分文字列の1つ → 長さ 3
# 出力: 3

# 例2
s = "bbbbb"
# "b" だけが条件を満たす最長部分文字列 → 長さ 1
# 出力: 1

# 例3
s = "pwwkew"
# 条件を満たす最長部分文字列の1つは "wke" → 長さ 3
# ※ "pwke" は 'w' が2回出るのでNG（連続ではない）
# 出力: 3

# 例4
s = ""
# 空文字列 → 最長部分文字列の長さは 0
# 出力: 0
```

## 実行結果

```
3
1
3
0
```
