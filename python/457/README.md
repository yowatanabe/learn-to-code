## 問題文

### 最小削除で括弧ログを正しくする

あなたはログフォーマット中の括弧構造を検証しています。文字列 `s` が与えられます。

`s` には以下の文字が含まれます。

* `"("`
* `")"`
* 英小文字

いくつかの括弧を削除して、括弧の対応が正しい文字列にしたいです。英小文字は削除してはいけません。

**削除する括弧の最小数**を返してください。

### 制約

* `1 <= len(s) <= 10^5`
* `s[i]` は `"("`, `")"`, 英小文字のいずれか

## 実行例

### 例1

```python
s = "a)b(c)d"
```

最初の `")"` を削除すると、

```python
"ab(c)d"
```

となり、括弧が正しくなります。

出力:

```python
1
```

### 例2

```python
s = "))(("
```

すべての括弧を削除する必要があります。

出力:

```python
4
```

### 例3

```python
s = "(log(error))"
```

すでに括弧が正しいです。

出力:

```python
0
```

## 実行結果

```
1
4
0
```
