## 問題文（LeetCode風・日本語）

### 問題：デプロイログのグループ化（アナグラム分類）

あなたはSREとして、複数サービスから送られてきた短いログタグを整理しています。

文字の並び順だけが異なるタグは、同じ種類のイベントとしてまとめたいです。

文字列の配列 `tags` が与えられます。

**互いにアナグラムである文字列同士を同じグループにまとめて**返してください。

アナグラムとは、文字の出現回数がすべて同じで、並び順だけが異なる文字列のことです。

* 返すグループの順序は問いません
* 各グループ内の文字列の順序も問いません
* `1 <= len(tags) <= 10^4`
* `0 <= len(tags[i]) <= 100`
* `tags[i]` は英小文字のみで構成される

**求めるもの**：アナグラムごとに分けた二次元配列

## 実行例

例1:

* 入力: `tags = ["eat", "tea", "tan", "ate", "nat", "bat"]`
* 出力: `[['eat', 'tea', 'ate'], ['tan', 'nat'], ['bat']]`
* 説明:

  * `"eat"`, `"tea"`, `"ate"` は同じ文字の組み合わせ
  * `"tan"`, `"nat"` も同じ文字の組み合わせ
  * `"bat"` は単独グループ

例2:

* 入力: `tags = [""]`
* 出力: `[['']]`

例3:

* 入力: `tags = ["a"]`
* 出力: `[['a']]`

## 実行結果

```
[['eat', 'tea', 'ate'], ['tan', 'nat'], ['bat']]
[['']]
[['a']]
```
