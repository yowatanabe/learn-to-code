# No.276

## 問題文

**重複文字のない最長部分文字列**

文字列 `s` が与えられる。

` s` の部分文字列（連続した区間）のうち、**同じ文字が二度以上出現しない（＝全ての文字がユニーク）** 部分文字列の「最大の長さ」を求めよ。

* 部分文字列とは、`s[l:r]`（`0 ≤ l ≤ r ≤ len(s)`）のような**連続区間**を指す。
* 文字列には ASCII 文字が含まれているものとする（小文字限定ではない）。

制約:

* `0 ≤ len(s) ≤ 2 * 10^5`
* 期待計算量: **O(n)**（n = len(s)）
* 追加メモリ: **O(min(n, 文字種類数))**

## 実行例

**例1**

```text
入力:
s = "abcabcbb"

出力:
3

説明:
"abc" が条件を満たす最長部分文字列の一つ（長さ3）。
他にも "bca", "cab" などがあるが、いずれも長さ3。
```

**例2**

```text
入力:
s = "bbbbb"

出力:
1

説明:
どの部分文字列も同じ文字しか含まないので、
"b"（長さ1）が最長。
```

**例3**

```text
入力:
s = "pwwkew"

出力:
3

説明:
"wke"（長さ3）が条件を満たす最長部分文字列の一つ。
"pwke" は 'w' が重複するのでNG。
```

**例4**

```text
入力:
s = ""

出力:
0
```

## 実行結果

```
3
1
3
0
```
