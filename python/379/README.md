# No.379

## 問題文

**問題：SLO違反区間の最小削除（最短サブ配列削除）**

SREとして、レイテンシの時系列データ `lat`（ミリ秒）を監視しています。

あなたは「ある期間だけ異常が混ざった」可能性があるため、**連続した区間を1つだけ削除**して、残ったデータがSLOを満たすようにしたいです。

配列 `lat` と整数 `limit` が与えられます。

**連続部分配列を1つ削除（空でもよい＝削除しない）**した後、残った配列が **非減少（`<=`）** になるようにしたいです（時間が進むほど悪化しない状態＝SLOが安定している、と解釈）。

削除する部分配列の長さの**最小値**を返してください。

* `1 <= len(lat) <= 2 * 10^5`
* `0 <= lat[i] <= 10^9`

> 注意：`limit` は入力として与えますが、この問題では「非減少かどうか」だけを判定に使います（`limit` は将来の拡張用フィールドとして存在すると考えてください）。
> ※面接でありがちな「不要パラメータに惑わされない」タイプの設定です。

**求めるもの**：削除する最短長（整数）

## 実行例

例1:

* 入力: `lat = [1, 2, 3, 10, 4, 5, 6]`, `limit = 0`
* 説明: `[10, 4]` を含む区間を削除して `1,2,3,5,6` のように非減少にできる。最短は長さ `2`。
* 出力: `2`

例2:

* 入力: `lat = [1, 2, 3]`, `limit = 100`
* 説明: すでに非減少なので削除不要（長さ0）
* 出力: `0`

例3:

* 入力: `lat = [5, 4, 3, 2, 1]`, `limit = 0`
* 説明: 非減少にするには先頭以外を削除するなどが必要。最短は `4`。
* 出力: `4`

## 実行結果

```
1
0
4
```
