# No.336

## 問題: リクエストを間引け（最短削除で単調増加にせよ / Minimum Removals to Make Increasing）

整数配列 `nums` が与えられます。

> `nums` からいくつかの要素を削除して、残った配列が **厳密に単調増加**（`a[i] < a[i+1]`）になるようにしたい。
> 削除する要素数の **最小値** を返してください。

* 要素の **順序は変えられません**（削除のみ）
* `nums` がすでに単調増加なら答えは `0`

### 制約（目安）

* `1 ≤ len(nums) ≤ 10^5`
* `-10^9 ≤ nums[i] ≤ 10^9`

時間計算量 **O(n log n)** を目指してください。

## 実行例

```python
# 例1
nums = [3, 1, 2, 1, 4]
# 最長の単調増加部分列（LIS）の長さは 3（例: [1,2,4]）
# 削除最小 = 5 - 3 = 2
# 出力: 2

# 例2
nums = [1, 2, 3, 4]
# すでに単調増加
# 出力: 0

# 例3
nums = [5, 4, 3, 2, 1]
# LIS 長は 1
# 削除最小 = 5 - 1 = 4
# 出力: 4
```

## 実行結果

```
例1 入力: nums=[3, 1, 2, 1, 4]
例1 出力: 2 (期待値: 2)
--------------------------------------------------
例2 入力: nums=[1, 2, 3, 4]
例2 出力: 0 (期待値: 0)
--------------------------------------------------
例3 入力: nums=[5, 4, 3, 2, 1]
例3 出力: 4 (期待値: 4)
--------------------------------------------------
```
