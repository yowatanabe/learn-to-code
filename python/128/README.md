# No.128

## 問題：文字列が回文かどうかを判定する

文字列 `s` が与えられます。
この文字列が**前から読んでも後ろから読んでも同じになる（回文である）かどうか**を判定してください。
ただし、大文字小文字の違いは無視し、英数字以外の文字は無視してください。

## 実行例

### 例 1:

```
入力: s = "A man, a plan, a canal: Panama"
出力: True
```

### 例 2:

```
入力: s = "race a car"
出力: False
```

### 例 3:

```
入力: s = "0P"
出力: False
```

### 実行結果

```text
True
False
False
```
