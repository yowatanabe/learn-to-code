## 問題: Top K Frequent Elements（上位K個の頻出要素）

整数配列 `nums` と整数 `k` が与えられます。

`nums` の中で **出現回数が多い順に k 個** の要素を返してください。

* 返す順序は問いません（どの順でもOK）
* `k` は常に有効（`1 ≤ k ≤` 異なる要素数）とします
* できるだけ **O(n log k)**（または O(n)）で解いてください

### 制約（目安）

* `1 ≤ len(nums) ≤ 10^5`
* `-10^4 ≤ nums[i] ≤ 10^4`
* `1 ≤ k ≤` 異なる要素数

## 実行例

```python
# 例1
nums = [1,1,1,2,2,3]
k = 2
# 出現回数: 1->3回, 2->2回, 3->1回
# 上位2つは [1,2]
# 出力例: [1, 2]

# 例2
nums = [1]
k = 1
# 出力: [1]

# 例3
nums = [4,4,4,6,6,7,7,7,7]
k = 2
# 出現回数: 7->4回, 4->3回, 6->2回
# 出力例: [7, 4]
```

## 実行結果

```
例1 入力: nums=[1, 1, 1, 2, 2, 3], k=2
例1 出力: [2, 1]
----------------------------------------
例2 入力: nums=[1], k=1
例2 出力: [1]
----------------------------------------
例3 入力: nums=[4, 4, 4, 6, 6, 7, 7, 7, 7], k=2
例3 出力: [4, 7]
----------------------------------------
```
