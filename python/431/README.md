## 問題文

### ログ文字列を並べ替えて異常コードを作れるか

あなたは監視システムのログを解析しており、文字列 `s` に含まれる文字を使って、異常コード `t` を作れるか確認したいとします。

文字列 `s` と `t` が与えられたとき、**`s` の文字を並べ替えて `t` を作れるなら `True`、作れないなら `False`** を返してください。

各文字は **`s` に出現した回数まで** しか使えません。

### 制約

* `1 <= len(s), len(t) <= 10^5`
* `s` と `t` は英小文字のみからなる

## 実行例

### 例1

```python
s = "silent"
t = "listen"
```

`s` の文字を並べ替えると `t` を作れるので、出力は

```python
True
```

### 例2

```python
s = "aabbcc"
t = "abccba"
```

文字数も各文字の出現回数も一致しているため、出力は

```python
True
```

### 例3

```python
s = "aabbc"
t = "aabbd"
```

`t` には `"d"` が必要ですが、`s` には含まれていません。
したがって出力は

```python
False
```

## 実行結果

```
True
True
False
```
