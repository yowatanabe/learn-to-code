## 問題: “壊れた監視タグ”を最小修正で直せ（Validate Parentheses With Wildcards）

文字列 `s` は `'('`, `')'`, `'*'` から構成されます。`'*'` は以下のいずれかとして扱えます。

* `'('` として扱う
* `')'` として扱う
* 空文字（削除）として扱う

> `s` を有効な括弧列にできるなら `True`、できないなら `False` を返してください。

有効な括弧列とは、左から見て常に `')'` が `'('` を超えず、最後に `'('` と `')'` の数が一致する文字列です。

### 制約（目安）

* `1 ≤ len(s) ≤ 10^5`

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
s = "()"
# 出力: True

# 例2
s = "(*)"
# '*' を空にすれば "()"
# 出力: True

# 例3
s = "(*))"
# '*' を '(' にすれば "(())"
# 出力: True

# 例4
s = ")*("
# どんな置換でも無理
# 出力: False
```

## 実行結果

```
------------------------------------------------------------
例3 入力: s='(*))'
例3 出力: True (期待値: True)
------------------------------------------------------------

------------------------------------------------------------
```
