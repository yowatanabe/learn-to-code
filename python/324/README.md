## 問題: 3つのソートされた配列をマージせよ（Merge Three Sorted Arrays）

3つの整数配列 `nums1`、`nums2`、`nums3` が与えられます。

これらを昇順にマージし、1つのソートされた配列として返してください。

### 制約（目安）

* `1 ≤ len(nums1), len(nums2), len(nums3) ≤ 10^5`
* `1 ≤ nums[i] ≤ 10^6`

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
nums1 = [1, 3, 5]
nums2 = [2, 4, 6]
nums3 = [0, 7, 8]
# マージした結果: [0, 1, 2, 3, 4, 5, 6, 7, 8]
# 出力: [0, 1, 2, 3, 4, 5, 6, 7, 8]

# 例2
nums1 = [1, 2]
nums2 = [3, 4]
nums3 = [5, 6]
# マージした結果: [1, 2, 3, 4, 5, 6]
# 出力: [1, 2, 3, 4, 5, 6]

# 例3
nums1 = [1]
nums2 = []
nums3 = [2]
# マージした結果: [1, 2]
# 出力: [1, 2]
```

## 実行結果

```
例1 入力: nums1=[1, 3, 5], nums2=[2, 4, 6], nums3=[0, 7, 8]
例1 出力: [0, 1, 2, 3, 4, 5, 6, 7, 8]
----------------------------------------
例2 入力: nums1=[1, 2], nums2=[3, 4], nums3=[5, 6]
例2 出力: [1, 2, 3, 4, 5, 6]
----------------------------------------
例3 入力: nums1=[1], nums2=[], nums3=[2]
例3 出力: [1, 2]
----------------------------------------
```
