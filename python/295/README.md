# No.295

## 問題: 最長回文部分文字列を求めよ

文字列 `s` が与えられます。

> `s` の中で、**左右対称な（回文になっている）連続部分文字列** のうち、
> **最も長いもの** を 1 つ返してください。

* 部分文字列とは、文字列中の連続した区間のことです。
* 答えが複数ある場合は、そのうち 1 つを返せばよいものとします。
* 文字は英数字や記号など何でも含まれてよいものとします。

### 制約（目安）

* `1 ≤ len(s) ≤ 1000`
* 文字は ASCII と仮定してよい

時間計算量は **O(n^2)** 程度、追加メモリ O(1) で解ける実装を意識してください。
（中心を決めて左右に伸ばす “expand around center” アプローチ）

## 実行例

```python
# 例1
s = "babad"
# "bab" も "aba" も最長回文部分文字列の候補
# どちらを返してもよい
# 出力例: "bab"  または  "aba"

# 例2
s = "cbbd"
# 最長回文部分文字列は "bb"
# 出力: "bb"

# 例3
s = "a"
# 部分文字列は "a" だけ
# 出力: "a"

# 例4
s = "ac"
# "a" と "c" のどちらも長さ1の回文
# 出力例: "a"  または  "c"
```

## 実行結果

```
例1の入力: babad
例1の結果: bab
例2の入力: cbbd
例2の結果: bb
例3の入力: a
例3の結果: a
例4の入力: ac
例4の結果: a
```
