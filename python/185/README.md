# No.185

## 問題文

整数配列 `nums` が与えられます。

`nums` の任意の非空連続部分配列の「累積積（全要素の積）」の中で、最大値を返してください。

* 要素は正・負・ゼロを含む可能性があります。
* `1 <= len(nums) <= 2 * 10^4`
* `-10 <= nums[i] <= 10`

## 実行例

```
入力: nums = [2,3,-2,4]
出力: 6
説明: 部分配列 [2,3] の積が 6 で最大。

入力: nums = [-2,0,-1]
出力: 0
説明: 最大は単一要素 [0]。

入力: nums = [-2,3,-4]
出力: 24
説明: 部分配列 [-2,3,-4] の積が 24。
```

## 実行結果

```
6
0
24
```
