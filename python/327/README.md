# No.327

## 問題文

あなたは整数のリスト `nums` と整数 `target` を与えられます。リストの中から 2 つの数を選び、その合計が `target` と一致する場合、その 2 つの数のインデックスを返す関数を作成してください。

リストの中に解が存在しない場合は、空のリストを返してください。

## 制約

* 1 <= `nums.length` <= 10⁴
* -10⁹ <= `nums[i]` <= 10⁹
* -10⁹ <= `target` <= 10⁹
* 各入力に対して、解が一意であることが保証されます。

## 実行例

**入力 1:**

```python
nums = [2, 7, 11, 15]
target = 9
```

**出力 1:**

```python
[0, 1]
```

**入力 2:**

```python
nums = [3, 2, 4]
target = 6
```

**出力 2:**

```python
[1, 2]
```

**入力 3:**

```python
nums = [3, 3]
target = 6
```

**出力 3:**

```python
[0, 1]
```

## 実行結果

```
[0, 1]
[1, 2]
[0, 1]
```
