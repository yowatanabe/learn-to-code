## 問題: 文字列の最小ウィンドウ（Minimum Window Substring）

文字列 `s` と `t` が与えられます。

> `t` に含まれるすべての文字（重複回数も含む）を含む
> `s` の **最短の連続部分文字列** を返してください。
> そのような部分文字列が存在しない場合は空文字列 `""` を返してください。

### 例

* `s="ADOBECODEBANC"`, `t="ABC"` のとき最短は `"BANC"`。

### 制約（目安）

* `1 ≤ len(s), len(t) ≤ 10^5`
* `s` と `t` は英大小文字を含む

時間計算量 **O(n)** を目指してください。

## 実行例

```python
# 例1
s = "ADOBECODEBANC"
t = "ABC"
# 出力: "BANC"

# 例2
s = "a"
t = "a"
# 出力: "a"

# 例3
s = "a"
t = "aa"
# 出力: ""
```

## 実行結果

```
例1 入力: s='ADOBECODEBANC', t='ABC'
例1 出力: 'BANC' (期待値: 'BANC')
------------------------------------------------------------
例2 入力: s='a', t='a'
例2 出力: 'a' (期待値: 'a')
------------------------------------------------------------
例3 入力: s='a', t='aa'
例3 出力: '' (期待値: '')
------------------------------------------------------------
```
