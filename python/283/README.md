## 問題: 最も多くの水を貯められる容器

長さ `n` の整数配列 `height` が与えられます。

`height[i]` は、位置 `i` に立っている縦の線の高さを表します。

各線は、x座標軸に沿って、インデックス `i` の位置に立っています。

2 本の線を選んで、その間に「容器」を作ることを考えます。

容器に貯められる水の量は、次のように決まります。

* 容器の幅 = 2 本の線の x 座標の差
* 容器の高さ = 2 本の線のうち **低い方の高さ**

水の量 = `幅 * 高さ`

あなたのタスクは、「**2 本の線を選んだときに貯められる水の量の最大値**」を求めることです。

### 制約（目安）

* `2 ≤ len(height) ≤ 10^5`
* `0 ≤ height[i] ≤ 10^4`

時間計算量は O(n) で解ける実装を意識してください（愚直な全探索 O(n^2) は TLE 相当）。

## 実行例

```python
# 例1
height = [1, 8, 6, 2, 5, 4, 8, 3, 7]
# 最適なのはインデックス 1 (高さ8) と 8 (高さ7) を選ぶ場合
# 幅 = 8 - 1 = 7
# 高さ = min(8, 7) = 7
# 面積 = 7 * 7 = 49
# 出力: 49

# 例2
height = [1, 1]
# 幅 = 1, 高さ = 1 → 面積 = 1
# 出力: 1

# 例3
height = [4, 3, 2, 1, 4]
# 0番目(4)と4番目(4)を選ぶと
# 幅 = 4, 高さ = 4 → 面積 = 16 (これが最大)
# 出力: 16
```

## 実行結果

```
49
1
16
```
