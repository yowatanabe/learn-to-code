# No.400

## 問題：最大 `k` 回の置換で作る最長同一ログ区間

あなたは SRE として、ログレベル列を分析しています。

文字列 `s` が与えられ、各文字は 1 件のログの種類を表します。

あなたは **高々 `k` 回まで文字を別の大文字英字に置き換える** ことができます。

置き換え後、**同じ文字だけで構成される連続部分文字列** の長さをできるだけ長くしたいです。

置き換えによって作ることができる、**最長の連続部分文字列の長さ**を返してください。

* `1 <= len(s) <= 2 * 10^5`
* `0 <= k <= len(s)`
* `s` は大文字英字 `A-Z` のみからなる

**求めるもの**：条件を満たす最長長さ（整数）

## 実行例

例1:

* 入力: `s = "ABAB"`, `k = 2`
* 説明:

  * たとえば 2 文字を `A` に置き換えると `"AAAA"` にできる
  * 最長長さは `4`
* 出力: `4`

例2:

* 入力: `s = "AABABBA"`, `k = 1`
* 説明:

  * 部分文字列 `"AABA"` では `B` を `A` に 1 回置き換えることで `"AAAA"` にできる
  * 最長長さは `4`
* 出力: `4`

例3:

* 入力: `s = "ABCDE"`, `k = 1`
* 説明:

  * 1 文字しか置き換えられないので、同じ文字だけにできる最長長さは `2`
* 出力: `2`

## 実行結果

```
4
4
2
```
