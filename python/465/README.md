## 問題文

### 隣接しないサーバーを選ぶ最大削減額

あなたはサーバーの停止メンテナンス計画を立てています。

整数配列 `savings` が与えられ、`savings[i]` は `i` 番目のサーバーを停止したときに削減できるコストを表します。

ただし、隣接するサーバーを同時に停止すると可用性に影響するため、**隣り合うサーバーを同時に選ぶことはできません**。

削減額の合計が最大になるようにサーバーを選んだときの、最大削減額を返してください。

### 制約

* `1 <= len(savings) <= 10^5`
* `0 <= savings[i] <= 10^9`

## 実行例

### 例1

```python
savings = [2, 7, 9, 3, 1]
```

index `0`, `2`, `4` を選ぶと、

```python
2 + 9 + 1 = 12
```

となり最大です。

出力:

```python
12
```

### 例2

```python
savings = [5, 1, 1, 5]
```

index `0` と `3` を選ぶと、

```python
5 + 5 = 10
```

です。

出力:

```python
10
```

### 例3

```python
savings = [4]
```

1台だけなので、そのサーバーを選びます。

出力:

```python
4
```

## 実行結果

```
12
10
4
```
