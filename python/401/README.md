# No.401

## 問題：次に大きいイベントID

あなたは SRE として、時系列に並んだイベントIDを分析しています。

整数配列 `nums` が与えられます。

各要素 `nums[i]` について、**`i` より右側にある要素のうち、最初に現れる `nums[i]` より大きい値**を求めてください。

そのような値が存在しない場合は `-1` としてください。

長さ `n` の配列 `answer` を返してください。

* `1 <= len(nums) <= 2 * 10^5`
* `-10^9 <= nums[i] <= 10^9`

**求めるもの**：各位置について「右側で最初に現れる、より大きい値」を格納した配列

## 実行例

例1:

* 入力: `nums = [2, 1, 2, 4, 3]`
* 説明:

  * 2 の右側で最初に大きい値は 4
  * 1 の右側で最初に大きい値は 2
  * 2 の右側で最初に大きい値は 4
  * 4 の右側には自分より大きい値がない
  * 3 の右側にもない
* 出力: `[4, 2, 4, -1, -1]`

例2:

* 入力: `nums = [1, 2, 3, 4]`
* 説明:

  * 1 の次に大きい値は 2
  * 2 の次に大きい値は 3
  * 3 の次に大きい値は 4
  * 4 の右側にはない
* 出力: `[2, 3, 4, -1]`

例3:

* 入力: `nums = [4, 3, 2, 1]`
* 説明:

  * どの要素も右側に自分より大きい値を持たない
* 出力: `[-1, -1, -1, -1]`

## 実行結果

```
[4, 2, 4, -1, -1]
[2, 3, 4, -1]
[-1, -1, -1, -1]
```
