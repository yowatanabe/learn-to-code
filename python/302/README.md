## 問題: 最長連続要素列（Longest Consecutive Sequence）

整数配列 `nums` が **順不同** で与えられます。

> 配列の要素の中から、
> **値が連続する（差がちょうど1ずつ）要素だけで作れる列** のうち、
> **最も長いものの長さ** を求めてください。

* 要素を並べ替えてよいものとします（「部分配列」ではなく「部分集合」的なイメージ）。
* 同じ値が複数回出てきても、**連続長には 1 度だけカウント**されます。
* 連続とは「`x, x+1, x+2, ...`」のように 1 ずつ増えていくことです。

### 制約（目安）

* `0 ≤ len(nums) ≤ 10^5`
* `-10^9 ≤ nums[i] ≤ 10^9`

時間計算量 **O(n)** の解法（ハッシュセット利用）を意識してください。

ソートして O(n log n) でも動きますが、ここでは O(n) を狙います。

## 実行例

```python
# 例1
nums = [100, 4, 200, 1, 3, 2]
# 並べ替えると {1,2,3,4} が連続している → 長さ 4
# 他に 100, 200 は単独なので長さ1
# 出力: 4

# 例2
nums = [0, 3, 7, 2, 5, 8, 4, 6, 0, 1]
# 集合にすると {0,1,2,3,4,5,6,7,8}
# 0〜8 まで 9 個連続している → 最長長さ 9
# 出力: 9

# 例3
nums = []
# 要素がないので連続列もない → 長さ 0
# 出力: 0

# 例4
nums = [9, 9, 9]
# 連続列としては「9」だけ → 長さ 1
# 出力: 1
```

## 実行結果

```
例1の入力: [100, 4, 200, 1, 3, 2]
例1の結果: 4
例2の入力: [0, 3, 7, 2, 5, 8, 4, 6, 0, 1]
例2の結果: 9
例3の入力: []
例3の結果: 0
例4の入力: [9, 9, 9]
例4の結果: 1
```
