# No.139

## 問題：回文かどうかを判定する

文字列 `s` が与えられます。
`英数字のみ` を対象に、大文字と小文字を区別せずに比較したとき、
`前から読んでも後ろから読んでも同じ（回文）` であれば `True` を、そうでなければ `False` を返してください。

### 実行例

#### 例 1:

```
入力: s = "A man, a plan, a canal: Panama"
出力: True
```

#### 例 2:

```
入力: s = "race a car"
出力: False
```

#### 例 3:

```
入力: s = " "
出力: True
```

### 実行結果

```text
True
False
True
```
