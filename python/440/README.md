## 問題文

### 最短で到達できるアラート経路

あなたは監視システムの通知経路を管理しています。`n` 個の通知ノードがあり、`0` から `n - 1` の番号が付いています。

有向グラフ `routes` が与えられます。

```python
routes[i] = [from_node, to_node]
```

これは `from_node` から `to_node` へ通知を送れることを表します。

開始ノード `start` から終了ノード `goal` まで、
**最小で何回の通知転送が必要か** を返してください。

到達できない場合は `-1` を返してください。

### 制約

* `1 <= n <= 10^5`
* `0 <= len(routes) <= 2 * 10^5`
* `0 <= from_node, to_node < n`
* `0 <= start, goal < n`

## 実行例

### 例1

```python
n = 5
routes = [[0, 1], [0, 2], [1, 3], [2, 3], [3, 4]]
start = 0
goal = 4
```

最短経路の一例は

```python
0 -> 1 -> 3 -> 4
```

転送回数は `3` なので、出力は

```python
3
```

### 例2

```python
n = 4
routes = [[0, 1], [1, 2]]
start = 0
goal = 3
```

`3` には到達できないため、出力は

```python
-1
```

### 例3

```python
n = 3
routes = [[0, 1], [1, 2]]
start = 2
goal = 2
```

開始地点と終了地点が同じなので、出力は

```python
0
```

## 実行結果

```
3
-1
0
```
