## 問題: 最小の会議室数（Meeting Rooms II）

会議の開始・終了時刻を表す区間配列 `intervals` が与えられます
。
各会議は `intervals[i] = [start, end]`（`start < end`）です。

> すべての会議を重ならないように実施するために必要な
> **会議室の最小数** を返してください。

* ある会議が時刻 `t` に終了し、別の会議が時刻 `t` に開始する場合、**同じ会議室を使える** とします。

  * つまり `[1, 4]` と `[4, 5]` は重ならない扱いです。

### 制約（目安）

* `1 ≤ len(intervals) ≤ 10^5`
* `0 ≤ start < end ≤ 10^9`

時間計算量 **O(n log n)** を目指してください。

## 実行例

```python
# 例1
intervals = [[0,30],[5,10],[15,20]]
# 0-30 と 5-10 が重なる → 2部屋必要
# 出力: 2

# 例2
intervals = [[7,10],[2,4]]
# 互いに重ならない → 1部屋でOK
# 出力: 1

# 例3
intervals = [[1,5],[2,6],[4,8],[9,10]]
# 同時に最大3つ重なるタイミングがある → 3部屋
# 出力: 3
```

## 実行結果

```
例1 入力: [[0, 30], [5, 10], [15, 20]]
例1 出力: 2
----------------------------------------
例2 入力: [[7, 10], [2, 4]]
例2 出力: 1
----------------------------------------
例3 入力: [[1, 5], [2, 6], [4, 8], [9, 10]]
例3 出力: 3
----------------------------------------
```
