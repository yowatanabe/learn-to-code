## 問題文

### しきい値を超える最短期間

あなたはあるシステムの1日ごとのエラー件数を配列 `errors` で管理しています。

整数配列 `errors` と整数 `target` が与えられたとき、**合計が `target` 以上になる連続部分配列のうち、長さが最小のもの**を求めてください。

そのような部分配列が存在しない場合は `0` を返してください。

### 制約

* `1 <= len(errors) <= 10^5`
* `1 <= errors[i] <= 10^4`
* `1 <= target <= 10^9`

## 実行例

### 例1

```python
errors = [2, 3, 1, 2, 4, 3]
target = 7
```

合計が `7` 以上になる連続部分配列はいくつかありますが、
最短のものの1つは

```python
[4, 3]
```

です。

したがって出力は

```python
2
```

### 例2

```python
errors = [1, 4, 4]
target = 4
```

`[4]` だけで条件を満たせるため、出力は

```python
1
```

### 例3

```python
errors = [1, 1, 1, 1]
target = 10
```

合計が `10` 以上になる連続部分配列は存在しないため、出力は

```python
0
```

## 実行結果

```
2
1
0
```
