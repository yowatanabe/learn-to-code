# No.263

## 問題文

**滑動ウィンドウ中央値**

整数配列 `nums` とウィンドウサイズ `k (1 ≤ k ≤ len(nums))` が与えられる。

各位置での**長さ `k` の連続部分配列**（`nums[i..i+k-1]`）の**中央値**を配列として返せ。

* `len(nums) = n (1 ≤ n ≤ 2 * 10^5)`
* `-10^9 ≤ nums[i] ≤ 10^9`
* 中央値の定義:

  * `k` が奇数: 中央の1要素
  * `k` が偶数: **中間2要素の平均（浮動小数）**
* 期待計算量: **O(n log k)**、追加メモリ: **O(k)**
* 小数は Python の `float` で返してよい

## 実行例

**例1**

```
入力:
nums = [1, 3, -1, -3, 5, 3, 6, 7]
k = 3
出力:
[1.0, -1.0, -1.0, 3.0, 5.0, 6.0]
説明:
各ウィンドウの中央値:
[1,3,-1]→1, [3,-1,-3]→-1, [-1,-3,5]→-1, [-3,5,3]→3, [5,3,6]→5, [3,6,7]→6
```

**例2**

```
入力:
nums = [1, 2, 3, 4]
k = 2
出力:
[1.5, 2.5, 3.5]
説明:
[1,2]→(1+2)/2=1.5, [2,3]→2.5, [3,4]→3.5
```

**例3**

```
入力:
nums = [2]
k = 1
出力:
[2.0]
```

## 実行結果

```
[1.0, -1.0, -1.0, 3.0, 5.0, 6.0]
[1.5, 2.5, 3.5]
[2.0]
```
