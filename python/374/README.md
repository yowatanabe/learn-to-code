## 問題文（LeetCode風・日本語）

**問題：最長連続ログ区間（閾値以内の最大長）**

監視ログのイベント発生時刻（秒）が昇順で与えられます。

あなたは「ある連続した区間の最初と最後の時刻の差が `k` 秒以内」であるような**最長の連続区間**を探したいです。

配列 `timestamps`（昇順）と整数 `k` が与えられたとき、条件を満たす連続区間の**最大長**を返してください。

* `1 <= len(timestamps) <= 2 * 10^5`
* `timestamps` は昇順（同値を含む可能性あり）
* `0 <= k <= 10^9`

**求めるもの**：最大長（整数）

## 実行例

例1:

* 入力: `timestamps = [1, 2, 4, 7, 8, 10]`, `k = 3`
* 説明: 区間 `[7, 8, 10]` は `10 - 7 = 3` なのでOK、長さ3。最大は3。
* 出力: `3`

例2:

* 入力: `timestamps = [5, 5, 5, 9, 10, 10, 14]`, `k = 0`
* 説明: 同時刻のみ許可。`[5,5,5]` や `[10,10]` が候補で最大は3。
* 出力: `3`

例3:

* 入力: `timestamps = [1, 3, 6, 10]`, `k = 2`
* 説明: `[1,3]` はOK（差2）で長さ2。ほかは1。
* 出力: `2`

## 実行結果

```
3
3
2
```
