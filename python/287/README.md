# No.287

## 問題: アナグラムをグループ化せよ

文字列の配列 `strs` が与えられます。

それぞれの文字列は英小文字のみからなります。

あなたのタスクは、**アナグラム同士をグループ化**することです。

* 2つの文字列がアナグラムであるとは、
  「文字の並び順は違っていても、**使われている文字とその個数が同じ**」であることを意味します。

  * 例: `"eat"` と `"tea"` と `"ate"` はアナグラム
  * `"eat"` と `"tan"` はアナグラムではない

戻り値は、「アナグラムのグループのリスト」とします。

各グループは、アナグラムな文字列が並んだリスト（順番は問わない）とします。

### 制約（目安）

* `1 ≤ len(strs) ≤ 10^4`
* `1 ≤ len(strs[i]) ≤ 100`
* `strs[i]` は `'a'`〜`'z'` のみを含む

時間計算量 O(n * L log L)（n: 文字列数, L: 最大長）程度の解法を目指してください。

## 実行結果

```
例1の入力: ['eat', 'tea', 'tan', 'ate', 'nat', 'bat']
例1の結果: [['eat', 'tea', 'ate'], ['tan', 'nat'], ['bat']]
例2の入力: ['a']
例2の結果: [['a']]
例3の入力: ['']
例3の結果: [['']]
```
