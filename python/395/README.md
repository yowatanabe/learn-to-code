# No.395

## 問題：最初のサービスから到達できるサービス数

あなたは SRE として、サービス間の通信到達性を調べています。

`n` 個のサービスがあり、`0` から `n-1` までの番号が付いています。

サービス間の一方向通信を表す配列 `edges` が与えられます。

`edges[i] = [u, v]` は、**サービス `u` からサービス `v` に通信できる**ことを表します。

サービス `0` から開始して、0回以上の通信を繰り返すことで到達できるサービスの数を求めてください。

* `1 <= n <= 2 * 10^5`
* `0 <= len(edges) <= 2 * 10^5`
* `0 <= u, v < n`

**求めるもの**：サービス `0` から到達可能なサービス数（整数）

## 実行例

例1:

* 入力:

  * `n = 6`
  * `edges = [[0,1],[0,2],[1,3],[3,4],[2,4]]`
* 説明:

  * 0 から到達できるのは `0,1,2,3,4`
  * `5` には到達できない
* 出力:

  * `5`

例2:

* 入力:

  * `n = 4`
  * `edges = [[1,2],[2,3]]`
* 説明:

  * 0 から出る辺がないので、到達できるのは `0` だけ
* 出力:

  * `1`

例3:

* 入力:

  * `n = 5`
  * `edges = [[0,1],[1,2],[2,0],[2,3],[3,4]]`
* 説明:

  * 0 から `1,2,3,4` に到達できる
* 出力:

  * `5`

## 実行結果

```
5
1
5
```
