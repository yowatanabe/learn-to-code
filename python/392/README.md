# No.392

## 問題文（LeetCode風・日本語）

### 問題：各ノードの負荷を除いた積

あなたは SRE として、複数ノードの負荷倍率を分析しています。

整数配列 `nums` が与えられます。`answer[i]` は、**`nums[i]` 以外のすべての要素の積**になるような配列 `answer` を返してください。

ただし、**割り算は使ってはいけません**。

* `1 <= len(nums) <= 2 * 10^5`
* `-30 <= nums[i] <= 30`
* 返す配列の各要素は 32bit 整数に収まるものとします

**求めるもの**：配列 `answer`

## 実行例

例1:

* 入力: `nums = [1, 2, 3, 4]`
* 説明:

  * `answer[0] = 2 * 3 * 4 = 24`
  * `answer[1] = 1 * 3 * 4 = 12`
  * `answer[2] = 1 * 2 * 4 = 8`
  * `answer[3] = 1 * 2 * 3 = 6`
* 出力: `[24, 12, 8, 6]`

例2:

* 入力: `nums = [-1, 1, 0, -3, 3]`
* 説明:

  * `answer[0] = 1 * 0 * -3 * 3 = 0`
  * `answer[1] = -1 * 0 * -3 * 3 = 0`
  * `answer[2] = -1 * 1 * -3 * 3 = 9`
  * `answer[3] = -1 * 1 * 0 * 3 = 0`
  * `answer[4] = -1 * 1 * 0 * -3 = 0`
* 出力: `[0, 0, 9, 0, 0]`

例3:

* 入力: `nums = [5, 2]`
* 説明:

  * `answer[0] = 2`
  * `answer[1] = 5`
* 出力: `[2, 5]`

## 実行結果

```
[24, 12, 8, 6]
[0, 0, 9, 0, 0]
[2, 5]
```
