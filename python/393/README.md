## 問題文

### 問題：到達可能な最小ジャンプ回数

あなたは障害調査のため、ログ上のチェックポイントを順にたどっています。

整数配列 `nums` が与えられ、`nums[i]` は位置 `i` から **最大で何個先まで進めるか** を表します。

最初は位置 `0` にいます。

配列の最後の位置 `n - 1` に到達するために必要な **最小ジャンプ回数** を求めてください。

到達できることは保証されます。

* `1 <= len(nums) <= 2 * 10^5`
* `0 <= nums[i] <= 10^5`

**求めるもの**：最後の位置に到達するための最小ジャンプ回数（整数）

## 実行例

例1:

* 入力: `nums = [2, 3, 1, 1, 4]`
* 説明:

  * 位置0から1へジャンプ
  * 位置1から4へジャンプ
  * 合計2回
* 出力: `2`

例2:

* 入力: `nums = [2, 3, 0, 1, 4]`
* 説明:

  * 位置0から1へジャンプ
  * 位置1から4へジャンプ
  * 合計2回
* 出力: `2`

例3:

* 入力: `nums = [1, 2, 1, 1, 1]`
* 説明:

  * 位置0から1へジャンプ
  * 位置1から3へジャンプ
  * 位置3から4へジャンプ
  * 合計3回
* 出力: `3`

## 実行結果

```
2
2
3
```
