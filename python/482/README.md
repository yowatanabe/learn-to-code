## 問題文

### サービス依存グループのサイズを求める

あなたはマイクロサービス間の依存関係を分析しています。

`n` 個のサービスがあり、サービス番号は `0` から `n - 1` です。

配列 `connections` が与えられます。

```python
connections[i] = [a, b]
```

これはサービス `a` とサービス `b` が同じ依存グループに属することを表します。

各サービスについて、**そのサービスが属する依存グループのサイズ**を配列で返してください。

## 実行例

### 例1

```python
n = 5
connections = [[0, 1], [1, 2], [3, 4]]
```

グループは以下です。

```python
{0, 1, 2}
{3, 4}
```

出力:

```python
[3, 3, 3, 2, 2]
```

### 例2

```python
n = 4
connections = []
```

どのサービスも単独グループです。

出力:

```python
[1, 1, 1, 1]
```

### 例3

```python
n = 6
connections = [[0, 5], [1, 2], [2, 3], [3, 4]]
```

出力:

```python
[2, 4, 4, 4, 4, 2]
```

## 実行結果

```
[3, 3, 3, 2, 2]
[1, 1, 1, 1]
[2, 4, 4, 4, 4, 2]
```
