# No.396

## 問題：設定変更の集約（重なり区間のマージ）

あなたは SRE として、設定変更が有効だった時間帯を集約しています。

各変更は区間 `[start, end]` で表され、`start <= end` を満たします。

複数の区間が与えられたとき、**重なっている区間をまとめて**、重なりのない区間の配列を返してください。

2つの区間 `[a, b]` と `[c, d]` が `c <= b` を満たすとき、これらは重なっているものとみなし、`[min(a, c), max(b, d)]` にマージできます。

* 返す区間は `start` の昇順でよい
* `1 <= len(intervals) <= 2 * 10^5`
* `0 <= start <= end <= 10^9`

**求めるもの**：マージ後の区間配列

## 実行例

例1:

* 入力: `intervals = [[1,3],[2,6],[8,10],[15,18]]`
* 説明:

  * `[1,3]` と `[2,6]` は重なるので `[1,6]` にまとめる
* 出力: `[[1,6],[8,10],[15,18]]`

例2:

* 入力: `intervals = [[1,4],[4,5]]`
* 説明:

  * `4 <= 4` なので重なるとみなし、`[1,5]` にまとめる
* 出力: `[[1,5]]`

例3:

* 入力: `intervals = [[5,7],[1,2],[2,4]]`
* 説明:

  * まず開始時刻でソートすると `[[1,2],[2,4],[5,7]]`
  * `[1,2]` と `[2,4]` は重なるので `[1,4]`
* 出力: `[[1,4],[5,7]]`

## 実行結果

```
[[1, 6], [8, 10], [15, 18]]
[[1, 5]]
[[1, 4], [5, 7]]
```
