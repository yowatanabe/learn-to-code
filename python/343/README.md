## 問題: 最小の矢の本数（Minimum Number of Arrows to Burst Balloons）

一直線上に風船があり、各風船は区間 `points[i] = [start, end]`（`start <= end`）で表されます。

`x` 座標 `x` に矢を放つと、`start <= x <= end` を満たすすべての風船が割れます。

> すべての風船を割るために必要な **最小の矢の本数** を返してください。

### 制約（目安）

* `1 ≤ len(points) ≤ 10^5`
* `-10^9 ≤ start, end ≤ 10^9`

時間計算量 **O(n log n)** を目指してください。

## 実行例

```python
# 例1
points = [[10,16],[2,8],[1,6],[7,12]]
# 例: x=6 で [2,8],[1,6] が割れる。x=12 で [7,12],[10,16] が割れる。
# 出力: 2

# 例2
points = [[1,2],[3,4],[5,6],[7,8]]
# 全部重ならないのでそれぞれ1本ずつ
# 出力: 4

# 例3
points = [[1,2],[2,3],[3,4],[4,5]]
# 端点で重なり得る。例: x=2 で最初の2つ、x=4 で最後の2つ
# 出力: 2
```
