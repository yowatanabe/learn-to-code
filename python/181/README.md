# No.181

## 問題文

整数配列 `nums` が与えられます。

「厳密に増加する（`nums[i1] < nums[i2] < ...`）」**部分列**（連続でなくてよい）のうち、**最長の長さ**を求めてください。

* 部分列は要素の順序を保ちながら、一部の要素を削除して得られる列です。
* `1 <= len(nums) <= 10^5`

## 実行例

```
入力: nums = [10,9,2,5,3,7,101,18]
出力: 4
説明: 最長は [2,3,7,101] などで長さ4

入力: nums = [0,1,0,3,2,3]
出力: 4
説明: [0,1,2,3] など

入力: nums = [7,7,7,7,7]
出力: 1
説明: すべて同値なのでどの単一要素も長さ1
```

### 実行結果

```
4
4
1
```
